## 원가 계산 핵심 공식

### 중국 사입
```
COGS = 매입가(KRW) + 매입가(KRW)×10% + 국제운송비 + (매입가+운송비)×관세율
```
- 수입 부가세는 매입세액 공제로 실질 부담 0

### 국내 직제조
```
COGS = 직접재료비 + 직접노무비 + 제조간접비(월고정비/월생산량)
```

### 국내 사입
```
COGS = 도매매입가 + 국내배송비
```
