## 핵심 마진 계산 공식

```
실질 매출 = 판매가 ÷ 1.1
순마진 = 실질 매출 - COGS - 채널 수수료 - 배송비
순마진율 = 순마진 ÷ 실질 매출 × 100
```

## 마진율 임계값

- 25%+ : 우수 (녹색)
- 15~25% : 보통 (노란색)
- 15% 미만 : 위험 (빨간색, 즉시 검토)

## 광고비 여력 공식

```
최대 광고비/건 = 순마진 × 50%
목표 ROAS = 판매가 ÷ 최대 광고비/건
```

## 특이 구조: 쿠팡 로켓배송

- 수수료 차감 방식 아님
- 납품가(매입가) 협의 방식
- 마진 = 납품가 - COGS
- 판매가는 쿠팡이 결정
