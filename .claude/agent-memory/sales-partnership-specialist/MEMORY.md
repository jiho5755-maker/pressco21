## 등급 체계 (2026-03-05 확정)

| 등급 | 자격 조건 | 할인율 |
|------|----------|-------|
| 파트너스 | 협회 소속 또는 인정 자격증 보유 | 10~20% |
| VIP | PRESSCO21 직접 지정 (주로 협회장) | 25% |
| 엠버서더 | 바이럴 능력 검증 후 초청 | 30% |

- 파트너클래스 플랫폼 기존 등급: SILVER(20%) / GOLD(25%) / PLATINUM(30%) — 수수료율 기준, 별개 체계

## VIP 인센티브 구조

분기 협회원 합산 구매액 기준 협회장에게 적립금 지급:
- 30만원 미만: 없음
- 30만원: 1.5만원 (5%)
- 100만원: 6만원 (6%)
- 300만원: 21만원 (7%)
- 500만원: 40만원 (8%)
- 1,000만원: 90만원 (9%)

## 엠버서더 최소 기준

- 인스타 팔로워 3,000 이상 또는 유튜브 구독자 1,000 이상 또는 블로그 월 3,000
- 최근 3개월 월 4건 이상 꽃공예 포스팅
- 인게이지먼트율 1% 이상 (팔로워 대비 댓글/저장/공유)

## 자격증 인정 기준

공인 협회 발급 + 꽃공예 관련 분야 + 이론+실기 포함 자격

등급: 기본(10%) / 우대(연5건+ 클래스, 15%) / 최우대(협회 검정위원, 20%)

## 협회 제휴 우선순위

1순위: 압화/프레스드플라워, 보존화/플로리스트 협회 (재료 수요 직결)
2순위: 레진공예, 캔들/디퓨저 협회
3순위: 석고/드라이플라워/리스

## 2026-03-10 협회 제휴 방향 업데이트

- 파트너클래스의 1순위 고객은 수강생이고, 협회는 공급자이자 제휴 레이어다.
- 협회 제휴의 목적은 단순 노출이 아니라 `파트너 제휴 서비스`를 미끼로 파트너십을 맺고, 협회원 혜택과 전용 상품으로 자사몰 락인을 강화하는 것이다.
- 협회 레이어는 `협회 일정/세미나`, `협회원 전용 제품`, `협회 시그니처 제품`, `협회원 전용 혜택`을 같은 허브 안에서 운영하는 방향으로 본다.
- 협회 세일즈 메시지는 `홍보해드립니다`보다 `회원 혜택과 구매 락인을 같이 설계해드립니다`에 가깝게 잡는다.
- 현재 기준 문서는 `docs/파트너클래스/README.md`, `shared-service-identity.md`, `enterprise-elevation-strategy-2026-03-10.md` 이다.

## 2026-03-11 S2-1 파트너 세일즈 랜딩 기준

- 파트너 신청(2609)은 제휴 영업 페이지로 보고, 첫 화면에서 `강의만 잘하시면 됩니다` 메시지를 명확히 전달해야 한다.
- 비교 축은 `직접 운영 vs PRESSCO21` 이고, 항목은 `일정관리 / 재료준비 / 결제·정산 / 홍보 / 확장성` 이 가장 설득력이 높다.
- 성장 경로는 `BLOOM -> GARDEN -> ATELIER -> AMBASSADOR` 로 보여주되, 공개 화면에서는 금전 조건보다 운영/브랜딩 혜택 중심으로 설명한다.
- 다음 협회 B2B 영업 도구(S2-2)는 이 파트너 세일즈 구조를 협회용 제안서와 ROI 시뮬레이터로 확장하는 단계다.

## 2026-03-11 S2-2 협회 영업 도구 기준

- 협회 제안서의 핵심 문장은 `협회원 혜택과 협회 홍보를 자사몰 안에서 같이 설계합니다.` 에 가깝다.
- 협회 제안 구조는 `협회원 혜택 -> 일정/세미나 홍보 -> 협회원 운영 확장 -> 구매 락인` 4가치 축으로 정리한다.
- 영업 현장에서는 협회명과 로고, 예상 회원 수와 예상 수강 인원만 바꿔 바로 URL을 보여줄 수 있어야 한다.
- 인센티브 설명은 추상적 제안보다 ROI 계산기와 실제 단계 구간으로 보여줄 때 설득력이 높다.
