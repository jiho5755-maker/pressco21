## 핵심 역량

### 1. 선순환 수수료 모델 설계

**현재 수수료 구조 (PRD v1.2 확정):**

| 매출 구간 | 수수료율 | 적립금 전환 | 파트너 순수익 | PRESSCO21 수익 |
|----------|---------|-----------|------------|--------------|
| 월 50만 미만 | 10% | 100% | 90% (현금) + 10%(적립금) | 적립금 환류 |
| 월 50~200만 | 12% | 80% | 88% (현금) + 9.6%(적립금) | 2.4% 순수익 |
| 월 200만+ | 15% | 60% | 85% (현금) + 9%(적립금) | 6% 순수익 |

**수익 시뮬레이션 (Phase 2 목표: 월 50건):**
```
시나리오A: 50건 x 평균 4만원 = 매출 200만원
  - 20명 파트너 (각 월 10만원 매출) -> 수수료 10% -> 적립금 100% 환류
  - PRESSCO21 직접 수익: 0원 (적립금으로 재료 구매 유도)
  - 간접 수익: 적립금 20만원 중 50% 재료 구매 = 10만원+ 추가 매출

시나리오B: 50건 x 평균 5.5만원 = 매출 275만원
  - 10명 파트너 (각 월 27.5만원 매출) -> 수수료 10% -> 적립금 100% 환류
  - 직접 수익: 0원 / 간접 수익: 적립금 27.5만원 x 50% = 13.75만원

Phase 3 목표: 150건 x 5만원 = 750만원
  - 혼합: 10% 구간 60% + 12% 구간 30% + 15% 구간 10%
  - 평균 수수료율: ~11.1%
  - 수수료 총액: ~83만원 / 적립금 전환: ~72만원
  - 적립금 재료 구매 전환: ~43만원
```

### 2. 정산 로직 설계

**정산 플로우:**
```
1. 주문 감지 (GAS 폴링)
2. 주문 금액 확인 (수강료 x 인원)
3. 해당 파트너의 월간 누적 매출 계산
4. 매출 구간별 수수료율 적용
5. 수수료 금액 산출
6. 적립금 전환 비율 적용
7. 파트너 적립금 잔액 업데이트
8. 파트너에게 정산 내역 이메일
```

**정산 정확성 보장 규칙:**
- 월 시작일: 매월 1일 00:00 (KST)
- 매출 구간 변동: 월 중간에 구간 변경 시 초과분에만 새 요율 적용
- 취소/환불: 정산 완료 후 환불 시 다음 달 정산에서 차감
- 소수점: 원 단위 반올림

### 3. 전환율 최적화 (CRO)

**클래스 플랫폼 전환 퍼널:**
```
메인/검색 유입 (100%)
  -> 클래스 목록 조회 (70%)
    -> 상세 페이지 진입 (40%)
      -> 일정 선택 (25%)
        -> 결제 완료 (15%)
          -> 수강 완료 (13%)
            -> 후기 작성 (5%)
              -> 재수강 (3%)
```

**퍼널 최적화 전략:**

| 단계 | 핵심 지표 | 최적화 방법 |
|------|----------|-----------|
| 목록 -> 상세 | CTR | 매력적 썸네일, 별점/잔여석 표시, 가격 투명성 |
| 상세 -> 일정선택 | 상세페이지 체류시간 | 후기 노출, 커리큘럼 상세, 강사 신뢰도 |
| 일정선택 -> 결제 | 장바구니 이탈률 | 간편 결제, 잔여석 긴급감, 포함 내역 명확 |
| 수강 -> 후기 | 후기 작성률 | D+7 이메일, 인센티브 (500~2000원) |
| 후기 -> 재수강 | 재수강률 | 추천 알고리즘, 시리즈 클래스, 적립금 |

**클래스 가격 가이드라인 (파트너 참고용):**

| 유형 | 시간 | 권장 가격대 | 재료비 포함 |

**적립금 사용처 우선순위 설계:**

3. 신상품 샘플 -> 새 클래스 개발 촉진
4. 계절 프로모션 참여 -> 시즌 매출 증가
