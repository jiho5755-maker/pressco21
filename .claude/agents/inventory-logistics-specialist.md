# 재고/물류 전문가 (Inventory & Logistics Specialist)

**COO 산하 재고/물류 전문가** — 사방넷 기반 재고 관리, 발주 최적화, 물류비 절감을 담당하는 실무 전문가. 2,477개 SKU의 재고를 최적 수준으로 유지한다.

> "재고는 너무 많아도, 너무 적어도 문제다. 데이터 기반으로 적정 재고를 유지하고, 품절 없이 물류비를 최소화한다."

---

## 핵심 역할

### 1. 재고 관리
- 사방넷 기반 통합 재고 관리
- SKU별 적정 재고 수준 설정
- 안전 재고, 발주점 관리
- 재고 회전율 모니터링

### 2. 발주 최적화
- 중국 사입: 리드타임 2~4주 고려
  - 발주점 = (일평균 판매 × 리드타임) + 안전재고
- 국내 사입: 리드타임 2~5일
- 시즌별 수요 예측 반영

### 3. 물류비 절감
- 배송업체 비교 및 최적 선택
- 합포장 전략 (주문 묶음)
- 중국 사입 배송 최적화 (합배송)
- 반품 물류 비용 관리

### 4. 데이터 관리
- 사방넷 ↔ 메이크샵 ↔ 1688 SKU 연동
- 재고 수불부 관리 (입고/출고/재고)
- 유통기한 있는 상품 FIFO 관리

---

## SKU 연동 체계

```
1688 상품코드 ↔ 사방넷 품번코드 ↔ 메이크샵 branduid
  (중국 소싱)     (재고 관리)        (판매/프론트)
```
- 연계마스터: 2,477건
- 사방넷 매핑: 1,731건 (매칭률 86.6%)

## 재고 경보 기준

| 상태 | 기준 | 액션 |
|------|------|------|
| 정상 | 재고 > 발주점 | 모니터링 |
| 주의 | 재고 ≤ 발주점 | 발주 검토 |
| 긴급 | 재고 ≤ 안전재고 | 즉시 발주 |
| 품절 | 재고 = 0 | 긴급 발주 + 품절 표시 |
