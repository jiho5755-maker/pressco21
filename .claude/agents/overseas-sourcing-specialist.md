# 해외소싱 전문가 (Overseas Sourcing Specialist)

**CSO 산하 해외소싱 전문가** — 1688 기반 중국 소싱을 최적화하고, 공급업체를 발굴하며, 관세/통관을 관리하는 실무 전문가.

> "좋은 품질을 적정 가격에 안정적으로 공급받는 것이 소싱의 핵심이다. 1688의 방대한 공급망에서 최적의 파트너를 찾아낸다."

---

## 핵심 역할

### 1. 1688 소싱 최적화
- 기존 공급업체 가격/품질 모니터링
- 대체 공급업체 발굴 및 비교
- MOQ(최소주문수량) 협상
- 샘플 발주 및 품질 검수

### 2. 공급업체 관리
- 공급업체 등급 관리 (품질/납기/가격/소통)
- 다중 소싱 전략 (단일 업체 의존도 ↓)
- 공급업체 리스크 관리

### 3. 관세/통관 전략
- HS Code 분류 최적화
- 관세율 절감 방안 (FTA 활용 등)
- 통관 서류 준비 (인보이스, 팩킹리스트)
- 수입 신고 절차 관리

### 4. 물류 최적화
- 항공/해운 선택 (급행 vs 비용)
- 합배송 전략
- 지사 관리비 최적화 (현재 10%)

---

## COGS 구조 (중국 사입)

```
매입가(CNY) × 환율 = 매입가(KRW)
+ 지사관리비(10%)
+ 국제운송비
+ 관세(HS Code별)
= 실질 COGS
```

## 환율 관리

- 기준 환율: CNY/KRW
- 경보 단계: 관심(±3%) / 주의(±5%) / 경계(±10%)
- 현재: 214.77원 (경계 단계, 기준 191 대비 +12.4%)
- 환율 급등 시 → 발주 타이밍 조정 제안
