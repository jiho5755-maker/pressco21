# 디자인 트렌드 & 영감

## 2025~2026 e-Commerce 트렌드

| 트렌드 | 설명 | PRESSCO21 적용 방향 |
|--------|------|---------------------|
| **Organic Shapes & Soft Geometry** | 딱딱한 직사각형을 벗어난 유기적 곡선, blob 형태 | 상품 카드, 섹션 구분선에 부드러운 곡선 도입 |
| **Subtle Glassmorphism** | 반투명 배경 + backdrop-filter blur | 헤더 스크롤 시, 플로팅 CTA에 활용 |
| **Micro-interactions** | hover, scroll, click 시 섬세한 피드백 | 상품 카드 hover, 장바구니 추가 피드백 |
| **Editorial Layout** | 매거진 스타일의 비대칭 그리드, 대담한 타이포 | 브랜드 스토리, 기획전 페이지에 적용 |
| **Scroll-driven Storytelling** | 스크롤에 따른 순차적 콘텐츠 공개 | 메인 페이지 섹션 진입 애니메이션 |
| **Dark Mode Accent** | 전체가 아닌 특정 섹션만 다크 모드 | 브랜드 철학 섹션(이미 다크 그린 배경) 강화 |
| **Variable Font Play** | 폰트 weight/width 애니메이션 | 히어로 타이틀, 섹션 헤딩에 활용 |
| **Sensory Richness** | 질감, 깊이감, 촉각적 느낌의 디지털 표현 | 꽃잎 질감, 레진 광택 효과 CSS로 구현 |

## 벤치마크 브랜드 (참고용)
- **Aesop (aesop.com)**: 미니멀하면서 고급스러운 내추럴 브랜드 e-commerce
- **Byredo (byredo.com)**: 에디토리얼 레이아웃 + 감각적 타이포그래피
- **Le Labo (lelabofragrances.com)**: 장인 정신 브랜딩 + 깔끔한 상품 진열
- **Diptyque (diptyqueparis.com)**: 자연/식물 모티프 + 프리미엄 감성
- **Etsy 우수 셀러 샵**: 핸드메이드/공예 카테고리의 UX 베스트 프랙티스

## 트렌드 적용 원칙
1. **브랜드 적합성 우선**: 트렌드가 PRESSCO21의 "우아하고 따뜻한 내추럴 프리미엄" 톤에 맞는지 검증 후 적용
2. **점진적 도입**: 한 번에 모든 트렌드를 적용하지 않고, 섹션 단위로 테스트하며 확장
3. **성능 균형**: 시각 효과가 로딩 속도나 사용성을 해치지 않도록 주의
4. **메이크샵 호환**: 모든 트렌드를 Vanilla CSS/JS로 구현 가능한 형태로 해석
