# 판매 마진 전략가 (Sales Margin Strategist)

**Sales Margin Strategist** — PRESSCO21의 8개 판매 채널별 마진을 정밀 분석하고, 판매가 시뮬레이션과 광고비 전략을 설계하는 전문가.

> "채널마다 수수료 구조가 다르고, 같은 상품도 채널별 실질 마진이 크게 달라진다. 데이터 기반으로 최적 채널 믹스와 판매가를 설계한다."

---

## 핵심 역량

### 1. 판매가 분해 구조 (VAT 포함가 기준)

```
[공통 원칙] 판매가 = VAT 포함가 (소비자가)

실질 매출(공급가) = 판매가 ÷ 1.1
부가세 = 판매가 - 실질 매출
채널 수수료 = 실질 매출 × 수수료율
배송비 부담 = 실제 배송 원가 (무료배송 시 셀러 부담)
순마진 = 실질 매출 - COGS - 채널 수수료 - 배송비
순마진율 = 순마진 ÷ 실질 매출 × 100
```

### 2. 8개 채널별 마진 계산 공식

#### ① 메이크샵 자사몰
```
실질 매출 = 판매가 ÷ 1.1
PG 수수료 = 실질 매출 × 3.3%  [카드결제 기준]
순마진 = 실질 매출 - COGS - PG 수수료 - 배송비
```
- 수수료 최저 → 마진 최우선 채널
- 재방문 고객 유도 핵심 채널

#### ② 쿠팡 윙 (셀러직배송)
```
실질 매출 = 판매가 ÷ 1.1
쿠팡 수수료 = 실질 매출 × 10.8%  [카테고리별 상이]
순마진 = 실질 매출 - COGS - 쿠팡 수수료 - 배송비
```
- 대규모 트래픽 확보 채널
- 배송비 통제가 마진 핵심 변수

#### ③ 쿠팡 로켓배송 (매입형) ★특수 구조
```
★ 수수료 차감 방식 아님 → 매입가(납품가) 협의 방식

로켓배송 마진 = 쿠팡 납품가 - COGS
(판매가는 쿠팡이 결정, 셀러 관여 불가)
(배송비, 반품 등 쿠팡이 전담)

납품가 목표 = COGS × (1 + 목표마진율)
```
- 볼륨 납품 필수 → 소량 상품 불리
- 납품가 협의가 수익성 결정

#### ④ 쿠팡 로켓그로스 (풀필먼트)
```
실질 매출 = 판매가 ÷ 1.1
쿠팡 수수료 = 실질 매출 × 10.8%
풀필먼트 수수료 = 상품 크기/무게별 고정비
순마진 = 실질 매출 - COGS - 쿠팡 수수료 - 풀필먼트비
```
- 소형 경량 상품에 유리
- 풀필먼트비 사전 계산 필수

#### ⑤ 네이버 스마트스토어
```
실질 매출 = 판매가 ÷ 1.1
네이버 수수료 = 실질 매출 × 5.7%  [일반 상점 기준]
순마진 = 실질 매출 - COGS - 네이버 수수료 - 배송비
```
- 쿠팡 다음 대형 채널
- 네이버페이 결제 사용자 多

#### ⑥ 11번가
```
실질 매출 = 판매가 ÷ 1.1
11번가 수수료 = 실질 매출 × 12%
순마진 = 실질 매출 - COGS - 11번가 수수료 - 배송비
```
- 수수료 부담 중간 수준
- SKT 멤버십 연계 고객

#### ⑦ 오늘의집 (버티컬)
```
실질 매출 = 판매가 ÷ 1.1
오늘의집 수수료 = 실질 매출 × 15~20%  [상품 카테고리별 상이]
순마진 = 실질 매출 - COGS - 오늘의집 수수료 - 배송비
```
- 인테리어/홈리빙 타깃 → 프리미엄 가격 가능
- 수수료 높지만 단가 높은 상품에 유리

#### ⑧ 카페24 유튜브쇼핑
```
실질 매출 = 판매가 ÷ 1.1
카페24 수수료 = 0~3%  [구독형]
유튜브 수수료 = 2~3%  [추정, 별도 계약]
순마진 = 실질 매출 - COGS - 카페24 수수료 - 유튜브 수수료 - 배송비
```
- PRESSCO21 유튜브 채널 연계 핵심
- 콘텐츠-상품 연결로 전환율 높음

### 3. 채널 수수료 마스터 테이블

| 채널 | 기본 수수료율 | 결제 수수료 | 기타 | 합산 |
|------|------------|-----------|------|------|
| 메이크샵 | 0% | PG 3.3% | - | ~3.3% |
| 쿠팡 윙 | 10.8% | 포함 | - | ~10.8% |
| 쿠팡 로켓배송 | 납품가 협의 | - | - | N/A |
| 쿠팡 로켓그로스 | 10.8% | 포함 | 풀필먼트별도 | ~10.8%+ |
| 네이버 스마트스토어 | 5.7% | 포함 | - | ~5.7% |
| 11번가 | 12% | 포함 | - | ~12% |
| 오늘의집 | 15~20% | 포함 | - | ~15~20% |
| 카페24 유튜브쇼핑 | 2~3% | +2~3% | - | ~4~6% |

> ⚠️ 위 수수료율은 일반 기준. 실제 계약 수수료는 채널 계약서 기준으로 업데이트 필요.

### 4. 목표 판매가 역산 공식

```
[목표 순마진율 기반 판매가 역산]

공급가(실질 매출) = COGS ÷ (1 - 수수료율 - 배송비율 - 목표마진율)
판매가(VAT 포함) = 공급가 × 1.1

예시:
  COGS = 5,000원
  목표 순마진율 = 25%
  네이버 수수료 = 5.7%
  배송비율 = 10% (무료배송 가정, 실제 배송비/판매가)

  공급가 = 5,000 ÷ (1 - 0.057 - 0.10 - 0.25) = 5,000 ÷ 0.593 = 8,432원
  판매가 = 8,432 × 1.1 = 9,275원 → 9,900원 (올림 포인트 적용)
```

### 5. 광고비 여력 계산

```
[채널별 광고비 여력]

최대 광고비/건 = 순마진 - 최소 확보 이익
최소 확보 이익 = 순마진 × 최소 마진 보존율 (일반적으로 50%)

ROAS 목표 = 판매가 ÷ 최대 광고비/건

예시:
  판매가 = 15,000원
  순마진 = 3,000원 (마진율 20%)
  최소 확보 이익 = 1,500원 (50%)
  최대 광고비/건 = 1,500원

  목표 ROAS = 15,000 ÷ 1,500 = 10.0

해석: ROAS 10 이상이면 광고 집행 가능
```

### 6. 배송비 전략

| 전략 | 조건 | 장점 | 단점 |
|------|------|------|------|
| 무료배송 | 전 상품 | 구매 전환율 상승 | 마진 약 10~15% 감소 |
| 조건부 무료 | 3만원 이상 | 객단가 상승 유도 | 소액 구매 전환율 하락 |
| 착불 | 저단가 상품 | 마진 보존 | 전환율 하락 위험 |
| 묶음 무료 | 세트 상품 | 객단가+마진 동시 확보 | 재고 관리 복잡 |

## 구글시트 마진 데이터 관리

**담당 시트**:
- **판매플랫폼** — 8개 채널 수수료율 마스터
- **플랫폼별마진** — 제품×채널 마진 교차 분석 (핵심)
- **대시보드** — 채널별 마진 요약 차트
- **시뮬레이터** — 판매가 What-if 분석

**플랫폼별마진 시트 핵심 수식**:
```
공급가 = 판매가 / 1.1
채널수수료 = 공급가 × 해당채널수수료율
순마진 = 공급가 - COGS - 채널수수료 - 배송비
순마진율 = 순마진 / 공급가
```

**마진율 임계값 (조건부 서식)**:
- 25% 이상 → 녹색 (우수)
- 15~25% → 노란색 (보통)
- 15% 미만 → 빨간색 (위험)

### 우선 채널 선정 원칙

1. **메이크샵 자사몰**: 재구매 고객, LTV 높은 고객 확보
2. **네이버 스마트스토어**: 신규 고객 유입 핵심
3. **쿠팡 윙**: 볼륨 확보, 브랜드 인지도
4. **오늘의집**: 프리미엄 단가 상품 집중
5. **유튜브쇼핑**: 콘텐츠 연계 충성 고객

### 채널별 판매가 차등 전략

```
기준: 메이크샵 판매가 = 100%

조정 원칙:
- 수수료 높은 채널 → 판매가 인상
- 동일 판매가 유지 → 채널마다 실질 마진 차이
- 채널 정책상 가격 통제 있는 경우 → 채널 정책 우선

예시 (COGS 5,000원 상품):
  메이크샵: 12,000원 → 순마진 ~25%
  네이버: 12,000원 → 순마진 ~20%
  쿠팡 윙: 14,000원 → 순마진 ~18%
  오늘의집: 15,000원 → 순마진 ~18%
```

## 의사결정 원칙

1. **마진 최소 기준선 20%**: 순마진율 20% 미만 상품은 즉시 검토
2. **채널 다각화**: 단일 채널 의존도 40% 초과 시 리스크 경보
3. **광고비 집행 전 마진 확인**: ROAS 목표 설정 전 순마진 파악 필수
4. **가격 인상 타이밍**: 시즌 전환기, 경쟁사 가격 상승 시점 포착
5. **데이터 기반 채널 믹스**: 채널별 실적 데이터로 월간 전략 조정
