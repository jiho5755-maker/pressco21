## 3. 섹션별 디자인 규격

### 01. 헤더 (상단바)
- 기본 배경: `#637561` / 텍스트: 흰색
- 스크롤 시: 배경 `#FFFFFF` / 텍스트: 검정색
- 메뉴 글자: 16px

### 02. 카테고리 섹션
**아이콘 메뉴 영역**
- 배치: 한 줄 4개 (모바일 기준)
- 아이콘: 가로 최대 50px, 중앙 정렬
- 텍스트: 16px, `#000000`, 중앙 정렬

**이미지 카드 슬라이드 (Swiper)**
- 카드 모서리: `border-radius: 15px`
- 메인 문구: 흰색 강조, 바로 아래 1px 흰색 밑줄
- 보조 설명: 흰색
- 그림자: `text-shadow` 미세 적용
- 슬라이드 dot: 기본 `--color-gray`, 활성 `#000000`

### 03. New Arrival 탭
- 탭 배경: `#e2e8dd`
- 활성 탭: `#b7c9ad` / 글자 흰색 / `border-radius: 25px`
- 비활성 탭: 배경 없음 / 글자 `#444444`
- 섹션 타이틀: 모바일 20px / PC 26px, `font-weight: 800`

### 04. Learn & Shop (YouTube)
- 섹션 배경: `#F4F7FF`
- 레이아웃: 메인 영상(좌) + 관련 상품 그리드(우)
- 상품 카드: `#fafafa` / `border-radius: 8px`

### 05. EVENT
- 배경: 전체 이미지 배너 (사업자 혜택 / 파트너사 안내)
- 텍스트: 제목 26px (Bold 700) / 본문 14px (Regular)
- 레이아웃: 모바일 1열 세로 / PC 2열 가로

### 06. 브랜드 스토리
- 배경: `#637561`
- 버튼: `.btn-outline` (흰색 테두리, 배경 투명, hover 시 흰색 배경)

### 07. 푸터
- 배경: `#425B51`
- 텍스트: 흰색 기본
- 카피라이트: `#BBBBBB`
- 로고: 흰색 전용 (`logo_w.png`)
