- **플랫폼**: 메이크샵 D4 카멜레온

- **외부 API**: Naver Map API, Google Apps Script (YouTube/Sheets 프록시)

## 메이크샵 필수 규칙 (CRITICAL)

1. **`\${var}` 이스케이프**: JS 템플릿 리터럴 `${변수}` → 반드시 `\${변수}`로 작성. 안 하면 저장 실패
2. **가상태그 보존**: `{$치환코드}`, `<!--/코드/-->` 주석, `<makeshop:>` 태그 절대 수정/삭제 금지
3. **IIFE 패턴**: 모든 JS는 `(function() { 'use strict'; ... })();`로 감쌈
4. **CSS 스코핑**: 컨테이너 ID/클래스로 선택자 범위 제한
5. **CDN만 사용**: npm, 빌드 도구 금지

## 파일 네이밍 규칙

각 폴더의 파일은 메이크샵 편집기 탭에 대응합니다:
- `Index.html` → HTML 탭에 붙여넣기
- `css.css` → CSS 탭에 붙여넣기
- `js.js` → JS 탭에 붙여넣기
- `*.fixed.*` → 개선/수정 버전 (원본과 비교 후 선택 사용)
