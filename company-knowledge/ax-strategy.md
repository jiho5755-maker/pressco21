# AX(AI 전환) 전략

> 소스: 2026-03-20 통합회의, AX-사전질문지 4명, 통합회의 워크시트 4명
> 상태: 회의 확정

## AI 도입 우선순위 (확정)

| 순위 | 영역 | 담당 | 핵심 내용 |
|------|------|------|----------|
| **1** | 상세페이지 기획 자동화 | 조승해(디자인), 이진선(대표) | 상품 정보 → 섹션 구조안 + 카피 초안 자동 생성 |
| **2** | 영상 콘텐츠 기획 | 장다경(영상) | 2개월치 촬영 계획안, 시즌별 기획, 영상 아카이빙 |
| **3** | 재고/출고 자동화 | 이재혁(물류), 이진선(대표) | 재고 자동 파악, 품절/해제 자동화, 부족 알림 |
| **4** | 고객 응대 자동화 | 이재혁(물류) | 전화주문 대응, 택배조회, 재고문의, 상품사용법 |
| **5** | 신제품 탐색/키트 구성 | 이진선(대표) | 신제품 발굴, 키트상품 기획 및 등록 |

> 논리: B(상세페이지) 빨라져야 → E(키트 등록) → D(영상 소재) → 매출 선순환

## 직원별 AI 니즈 상세

### 이재혁 (물류운영팀 과장)

**현재 업무 흐름:**
1. 전날 주문리스트 체크 (지하창고 피킹 점검)
2. 오전 여유 시 서류 업무 (세금계산서, 고객 요청 서류) + 부재료 준비
3. 출고 업무
4. 마감 정리 + 부재료 준비

**반복 업무:** 포장, 주문 확인, 출고 처리, 택배 이슈 대응, 시즌 서류 작성

**병목:**
- 특수포장 (액자, 대형 상품, 파손위험 상품)
- 고객전화상담 (상품사용법, 후불결제, 전화주문) ← **가장 시간 소모**
- 방문손님 응대 (무작정 방문, 시간 낭비)

**AI 희망:**
- 지하창고 피킹 리스트 자동 생성
- 재고현황 자동 파악 + 예상 입고량 제시 + 부족 시 알림
- 설정 수량 도달 시 자동 품절/품절해제
- 고객응대 대신: 택배조회, 재고문의, 사이트 이용문의, 상품사용설명, 상품사이즈, 강사가문의

**AI 금지:** (미작성)

**3개월 후 성공 기준:**
- 주문서 체크 시간 절약
- 재고파악 시간 절약
- 직접적인 전화 응대 감소
- 출고 외 업무가 시스템 상에서 해결

### 이진선 (대표)

**우선 강화 부분:** 손익증대, 매출상승, 키트상품 사이트 올리기

**반복 설명 내용:** 식물표본 안내, 강의내용, 제품안내 및 사용설명

**직원에게 강조:** 고객응대

**좋은 디자인 기준:** 고급스러움

**디자이너/영상팀 아쉬운 점:**
- 썸네일 서체 통일 안 됨
- 실물컬러에 적합하게 촬영 안 됨
- 실제 작업 경험 없이 디자인하면 연결이 매끄럽지 못함

**AI 시간절약:**
- 신제품 찾아주기 (실현불가능이겠지만)
- 강의안 정리

**AI 금지:** 모든 최종결정은 직원이 해야함

**AI 활용 태도:** 많이 시도해보기

**1년 후 성공 기준:**
- 영상/디자인 작업시간 단축 → 여분시간에 신제품 탐색
- 월 3~4개 신제품 토의 → 시판 → 반응 없으면 폐기, 있으면 추가주문
- 제2, 제3의 레터링(히트상품) 탄생

### 장다경 (영상기획팀장)

**현재:** 유튜브 쇼츠 제작

**필요 정보:** 어떤 제품 영상을 찍을 것인지

**병목:** 어떤 제품을 찍을지 계획이 없는 경우

**AI 희망:**
- 2개월치(총 8개) 촬영 제품 계획안 선행
- 시기별 잘 먹힐 제품 영상 기획
- 기존 영상속 언어를 습득해서 톤 유지

**AI 금지:** 최종 게시

**아쉬운 점:**
- 과거 영상들이 아카이빙 되고 있지 않음
- 과거 영상들이 인스타/틱톡 등 다른 플랫폼에 응용 안 됨

**현재 AI 사용:** Canva, CapCut (이미 AI 기반 제작 템플릿 사용 중)

**3개월 후 성공 기준:** 기존 영상들을 잘 관리해서 응용

### 조승해 (디자인기획팀 사원)

**현재:** 상세페이지 제작

**필요 정보:** 상품 특징과 상품명

**병목:** 기획

**AI 희망:**
- 상세페이지 구조안과 상품기획
- 상세페이지 순서대로 정리
- 섹션 구성안

**AI 금지:** 최종 레이아웃

**정리 안 된 자료:** 성원애드피아 출력물, 사진자료물, 폰트와 톤

**필요 템플릿:** 기획안 템플릿 + 상세페이지 섹션 구성안 템플릿

**3개월 후 성공 기준:** 작업 시간 단축

**상세페이지 제작 프로세스 (웹디자이너 질문지 기반):**
- 소요시간: 간단한 이미지 수정 1~2일 / 상세페이지 1~7섹션 1차 시안까지 약 일주일
- 도구: 포토샵(이미지보정+상세), 피그마(레이아웃), 일러스트(상세+패키지), 미리캔버스(공지배너), Gemini/Adobe Firefly/Freepik(이미지 생성)
- 작업 순서: 기획 → 사진 받기 → 배치 → 사진 보정+텍스트
- 사진/카피 수신: 카톡 + 폴더공유 + 직접 핸드폰 촬영
- 수정 빈도: 많지 않으나 초기 기획 방향성 미공유로 보완 작업 발생
- 디자인 통일성: 페이지마다 다름 (기존 송대리 작업은 통일됨)
- 사진 품질: 보정 필요한 경우 많음 + 사진 부족
- 모바일 최적화: 하고 있음
- 오픈마켓 복사등록: 본인이 담당 (가격+상품+이미지 수정, 11번가 부가이미지 규격 다름)
- 시간 많이 걸리는 TOP 3: 사방넷, 상세페이지 기획, 이미지 생성
- 템플릿 필요: 카테고리별보다 모음전/단일상품 구분이 더 유용
- 개선 요청: 섹션별 멘트+상품 리스트가 기초 기획안으로 미리 정리되면 작업 수월

## AI 금지 영역 종합

| 영역 | 근거 |
|------|------|
| 모든 최종 결정 | 직원이 해야 함 (대표 원칙) |
| 최종 영상 게시 | 장다경 답변 |
| 최종 레이아웃 | 조승해 답변 |
| 이미지 제작 (최종) | 조승해 답변 |
| 환불/가격/발주 판단 | 공통 인식 |
