## 1. 등급 체계 개요

### 1-1. 5등급 구조

PRESSCO21 회원 등급은 **4단계 성장 경로 + 1개 특별 칭호**로 구성됩니다.

```
[ 성장 경로 (순차 진행) ]

  회원(씨앗) ──> 인스트럭터 ──> 파트너스 ──> VIP
     |               |              |          |
   일반회원       꽃공예 강사     제휴업체     협회장/
                                /협회 소속    최고 선정

[ 독립 칭호 (별도 부여) ]

  엠버서더  ── 바이럴 특별 칭호 (어떤 등급이든 병행 가능)
```

### 1-2. 등급 간 관계 원칙

| 원칙 | 설명 |
|------|------|
| **순차 성장** | 회원 -> 인스트럭터 -> 파트너스 -> VIP는 순서를 건너뛸 수 없음 |
| **엠버서더 독립** | 엠버서더는 등급이 아닌 '칭호'로, 어떤 등급의 회원이든 병행 부여 가능 |
| **하향 보호** | 일정 기간 활동이 줄어도 즉시 강등하지 않음 (6개월 유예 기간) |
| **복수 불가** | 성장 경로 등급은 동시에 1개만 보유 (엠버서더 칭호만 중복 가능) |

## 2. 등급별 브랜드 스토리

### 2-1. 성장 메타포: "한 송이 꽃이 정원이 되기까지"

PRESSCO21의 등급 체계는 **꽃의 성장 과정**을 메타포로 삼습니다. 씨앗에서 시작하여 뿌리를 내리고, 꽃을 피우고, 정원을 가꾸며, 마침내 꽃밭을 이끄는 사람이 되는 여정입니다.

```
씨앗(회원) -> 뿌리(인스트럭터) -> 꽃밭(파트너스) -> 정원사(VIP)

"하나의 씨앗이 뿌리를 내리고,
 꽃을 피워 밭을 이루며,
 아름다운 정원을 가꾸는 사람이 됩니다."
```

### 2-2. 등급별 아이덴티티

---

#### [회원] 씨앗 -- Seed

**슬로건**: "꽃의 세계에 첫 발을 내딛다"

**브랜드 내러티브**:
모든 아름다운 정원은 하나의 작은 씨앗에서 시작합니다. 회원님은 지금 PRESSCO21이라는 토양 위에 첫 씨앗을 심으신 거예요. 아직은 무엇이 피어날지 모르지만, 그 설렘이야말로 가장 소중한 순간입니다.

**핵심 감정**: 설렘, 기대, 환영
**아이콘 메타포**: 씨앗 (잠재력과 가능성의 상징)

**포지셔닝**:
- 꽃 공예에 관심을 갖고 첫 걸음을 내딛은 분
- 아직 전문적인 활동은 없지만, PRESSCO21의 상품과 콘텐츠를 탐색하는 단계
- "처음이어도 괜찮아요. 우리가 함께할게요."

---

#### [인스트럭터] 뿌리 -- Root

**슬로건**: "깊은 뿌리가 아름다운 꽃을 피웁니다"

**브랜드 내러티브**:
씨앗이 싹을 틔우려면 먼저 깊은 뿌리를 내려야 합니다. 인스트럭터는 꽃 공예의 기술과 지식으로 단단한 뿌리를 내린 분들입니다. 수업을 통해 자신만의 기법을 나누고, 학생들에게 꽃의 아름다움을 전하는 사람. 뿌리가 깊을수록 더 아름다운 꽃이 피어납니다.

**핵심 감정**: 자부심, 전문성, 성장
**아이콘 메타포**: 뿌리/새싹 (전문성의 기반, 성장의 시작)

**포지셔닝**:
- 꽃 공예 강사로 활동하며 PRESSCO21 재료로 수업을 진행하는 분
- 교육 역량을 갖추고 수강생에게 기술을 전수하는 단계
- "당신의 수업이 누군가의 첫 번째 꽃이 됩니다."

---

#### [파트너스] 꽃밭 -- Bloom Field

**슬로건**: "함께 피우는 꽃밭, 함께 성장하는 파트너"

**브랜드 내러티브**:
한 송이 꽃은 아름답지만, 여러 송이가 모이면 꽃밭이 됩니다. 파트너스는 개인의 역량을 넘어 협회와 함께, 동료와 함께 더 넓은 꽃밭을 만들어가는 분들입니다. 제휴 업체로서, 협회의 일원으로서, PRESSCO21과 손잡고 꽃 공예 문화를 함께 넓혀갑니다.

**핵심 감정**: 연대, 신뢰, 확장
**아이콘 메타포**: 꽃밭 (협력과 확장의 상징)

**포지셔닝**:
- 협회 소속 또는 제휴 업체로 PRESSCO21과 공식 파트너십을 맺은 분
- 개인 활동을 넘어 조직적 협력으로 더 큰 가치를 만드는 단계
- "함께여서 더 아름답습니다."

---

#### [VIP] 정원사 -- Gardener

**슬로건**: "아름다운 정원을 가꾸는 사람"

**브랜드 내러티브**:
정원사는 씨앗을 뿌리고, 물을 주고, 햇빛을 나누며 정원 전체를 돌보는 사람입니다. VIP는 협회장이나 분야 최고 전문가로 선정된 분들로, 단순히 자신의 꽃만 키우는 것이 아니라 다른 이들의 꽃도 함께 피울 수 있도록 이끄는 분입니다. PRESSCO21이 가장 존경하고 감사하는 분들입니다.

**핵심 감정**: 존경, 감사, 리더십
**아이콘 메타포**: 정원/나무 (총괄적 돌봄, 성숙한 리더십)

**포지셔닝**:
- 협회장 또는 분야 최고 전문가로 PRESSCO21이 직접 선정한 분
- 커뮤니티 전체에 영향력을 미치고, 후배 양성과 문화 확산에 기여하는 단계
- "당신이 가꾼 정원에서 수많은 꽃이 피어납니다."

---

#### [엠버서더] 별빛 -- Starlight

**슬로건**: "꽃밭 위의 별빛, 세상에 아름다움을 비추다"

**브랜드 내러티브**:
꽃은 낮에 피어나지만, 별빛은 밤에도 빛납니다. 엠버서더는 꽃의 아름다움을 세상에 널리 알리는 특별한 존재입니다. SNS, 영상, 작품 활동 등을 통해 PRESSCO21의 가치를 바이럴하고, 새로운 사람들이 꽃 공예의 세계로 들어올 수 있도록 길을 밝히는 별빛 같은 분들입니다.

> **왜 "별빛"인가**: 엠버서더는 성장 경로의 일부가 아닌 독립된 특별 칭호입니다. 꽃(식물)의 메타포 체계에서 벗어나, "별(Star)"과 "빛(Light)"이라는 메타포를 사용하여 차별화합니다. 꽃밭 위에 빛나는 별, 즉 꽃 공예 세계를 비추는 영향력이라는 의미입니다.

**핵심 감정**: 영광, 특별함, 영향력
**아이콘 메타포**: 별/빛 (바이럴 영향력, 빛나는 존재감)

**포지셔닝**:
- PRESSCO21이 직접 위촉하는 바이럴 특별 칭호
- 등급과 무관하게 부여 가능 (회원이면서 엠버서더, VIP이면서 엠버서더 등)
- SNS/콘텐츠/작품 활동을 통해 브랜드 가치를 확산하는 분
- "당신의 빛이 더 많은 사람에게 꽃의 아름다움을 전합니다."

### 2-3. 성장 스토리라인 요약

```
"한 알의 씨앗이 뿌리를 내리고,
 꽃밭을 이루어 정원을 가꾸기까지.
 그 여정의 모든 순간을, PRESSCO21이 함께합니다."

 씨앗   ->   뿌리   ->   꽃밭   ->   정원사
 (Seed)     (Root)    (Bloom     (Gardener)
                       Field)
                                      + 별빛 (Starlight)
                                        어디서든 빛나는 특별한 존재
```

## 3. 등급별 UI 배지 디자인 가이드

### 3-1. 배지 색상 체계

브랜드 프라이머리 컬러 `#7d9675`(세이지 그린)을 기축으로, 등급이 올라갈수록 따뜻하고 깊은 톤으로 진행합니다. 엠버서더는 독립 계열(보라-골드 그라데이션)로 시각적 차별화합니다.

| 등급 | 주 색상(HEX) | 보조 색상(HEX) | 텍스트 색상 | 설계 의도 |
|------|-------------|---------------|-----------|---------|
| **회원** (씨앗) | `#a8b5a0` (연한 세이지) | `#e8efe5` (아이보리 그린) | `#5a6b55` (진한 세이지) | 편안하고 부담 없는 입문 톤 |
| **인스트럭터** (뿌리) | `#7d9675` (프라이머리 그린) | `#d4e8d0` (라이트 그린) | `#ffffff` (흰색) | 브랜드 대표색으로 전문성 표현 |
| **파트너스** (꽃밭) | `#5e8a6e` (딥 그린) | `#b7d4bf` (소프트 민트) | `#ffffff` (흰색) | 깊어진 녹색으로 신뢰와 연대 표현 |
| **VIP** (정원사) | `#b89b5e` (브랜드 골드) | `#f5ecd7` (아이보리 골드) | `#ffffff` (흰색) | 포인트 컬러 골드로 최고 등급 강조 |
| **엠버서더** (별빛) | `#8b6fae` (소프트 퍼플) | `#e8dff5` (라벤더) | `#ffffff` (흰색) | 별빛의 신비로움, 독립적 특별함 |

**색상 선정 근거**:

1. **회원 `#a8b5a0`**: 프라이머리 그린(`#7d9675`)을 채도 낮추고 밝기 높인 변형. 아직 성장 전이지만 브랜드 패밀리에 속한다는 소속감.
2. **인스트럭터 `#7d9675`**: 정확히 브랜드 프라이머리 컬러. "PRESSCO21의 핵심 구성원"임을 표현.
3. **파트너스 `#5e8a6e`**: 프라이머리에서 명도를 한 단계 낮춘 딥 그린. 깊이와 신뢰를 더함.
4. **VIP `#b89b5e`**: 기존 브랜드 시스템의 포인트 컬러(골드). CTA 버튼, 이메일 강조색과 일관성 유지.
5. **엠버서더 `#8b6fae`**: 그린 계열에서 완전히 벗어난 퍼플 계열. "별빛"이라는 독립 메타포와 시각적으로 일치하며, 한눈에 "다른 종류의 특별함"이 느껴짐.

### 3-2. 배지 형태 컨셉

#### A안 (채택 권고): 원형 아이콘 배지

```
형태: 원형 (circle)
크기 (CRM 고객 카드): 24px x 24px
크기 (고객 목록 행): 20px x 20px
크기 (고객 상세 헤더): 32px x 32px

구조:
+------------------+
|   [원형 배경색]    |
|    [중앙 아이콘]   |
+------------------+
아래: 등급명 텍스트 (선택적)

아이콘 예시:
  회원:       씨앗 아이콘 (작은 씨앗 실루엣)
  인스트럭터:  새싹 아이콘 (잎 달린 새싹)
  파트너스:    꽃 아이콘 (활짝 핀 꽃)
  VIP:        왕관 아이콘 (작은 왕관/화관)
  엠버서더:    별 아이콘 (빛나는 별)
```

**채택 근거**:
- 원형은 CRM 고객 카드, 목록 행, 상세 페이지 등 다양한 크기에서 가독성 유지
- 기존 CRM의 customer_type 아이콘(학교/센터/협회 이모지)과 병렬 배치에 적합
- 축소해도 식별 가능한 단순한 실루엣 아이콘 사용

#### B안 (대안): 리본형 텍스트 배지

```
형태: 모서리 둥근 직사각형 (pill shape)
크기: 높이 20px, 너비 가변 (텍스트 길이에 따라)

구조:
+---[등급명]---+
| [배경색 + 흰색 텍스트] |
+------------------+

예시:
  [회원]  [인스트럭터]  [파트너스]  [VIP]  [엠버서더]
  연회색     녹색         딥그린     골드      퍼플
```

**B안 장점**: 텍스트가 직접 보여 별도 범례 불필요
**B안 단점**: 한글 텍스트 길이 가변으로 레이아웃 일관성 유지 어려움

#### C안 (하이브리드 -- 실무 구현 추천)

**목록/카드**: A안(원형 아이콘) 사용
**상세 페이지**: A안 + 등급명 텍스트 병행

```
고객 목록 행:
  [원형배지] 홍길동  [학교아이콘] 서울초등학교  ACTIVE

고객 상세 헤더:
  [원형배지 32px]
  인스트럭터
  홍길동 선생님
```

### 3-3. CRM 고객 카드 배치 가이드

```
+---------------------------------------------------------+
|  [VIP등급배지 24px]  [고객유형 아이콘]                     |
|                                                         |
|  거래처명 / 이름                          [상태 배지]     |
|  사업자번호: XXX-XX-XXXXX                ACTIVE          |
|                                                         |
|  총 주문 금액: 5,200,000원    최종 주문: 2026-02-28      |
|  총 주문 횟수: 47건            미수금: 0원               |
|                                                         |
|  [엠버서더 별 칭호]  (해당 시에만 표시)                     |
+---------------------------------------------------------+

배치 규칙:
- 등급 배지: 카드 좌상단 (가장 먼저 눈에 들어오는 위치)
- 고객 유형 아이콘: 등급 배지 오른쪽 (학교/센터/협회 구분)
- 상태 배지: 카드 우상단 (ACTIVE=초록, DORMANT=노랑, CHURNED=빨강)
- 엠버서더 칭호: 카드 하단 (별 아이콘 + "엠버서더" 텍스트, 보라색)
```

### 6-2. VIP 인센티브 발표 커뮤니케이션 방식

VIP는 PRESSCO21이 **직접 선정**하는 최고 등급이므로, 일반적인 "조건 달성 -> 자동 승급" 방식이 아닙니다. 따라서 발표 커뮤니케이션도 특별해야 합니다.

---

**발표 프로세스 (3단계):**

```
1단계: 내부 선정
   관리자가 CRM에서 후보 리스트 작성
   -> 대표 최종 승인
   -> 선정 사유 구체적 기록

2단계: 개인 통보 (비공개)
   대표 명의 이메일 발송 (섹션 5-3 템플릿)
   -> 전화 통화로 직접 축하 (가능한 경우)
   -> 수락 여부 확인

3단계: 공개 발표 (선택적)
   자사몰 공지 또는 SNS 게시
   -> 해당 VIP의 동의를 반드시 사전 획득
   -> 과도한 홍보 금지, 감사 중심 톤
```

**공개 발표 카피 (자사몰 공지/SNS용):**

```
A안 (채택 권고) -- 간결 + 품격:
----------------------------------------------------
  PRESSCO21이 존경하는 분을 소개합니다.

  {partnerName} 님
  가드너 클럽 VIP 선정

  "{selectionQuote}"
  -- {partnerName}님이 보여주신 꽃 공예에 대한 열정에
     깊은 감사를 전합니다.

  PRESSCO21
----------------------------------------------------

B안 (대안) -- 스토리텔링형:
----------------------------------------------------
  한 분의 정원사가 가꿔온 아름다운 이야기.

  [partnerName]님은 [활동 내용 1~2문장].
  그 헌신에 감사드리며,
  PRESSCO21 가드너 클럽 VIP로 모십니다.
----------------------------------------------------
```

**커뮤니케이션 원칙**:

| 원칙 | 설명 | 예시 |
|------|------|------|
| **개인 통보 우선** | 공개 전 반드시 본인에게 먼저 알림 | 이메일 + 전화 |
| **동의 기반 공개** | SNS/공지 게시 전 본인 동의 필수 | "축하 게시물을 올려도 될까요?" |
| **과장 금지** | "최고", "유일한", "업계 1위" 사용 금지 | "존경하는", "감사하는" 사용 |
| **실적 비공개** | 매출액/주문 횟수 등 수치 공개 금지 | 활동 내용/기여 중심으로 표현 |
| **형식적 표현 배제** | "선정되었습니다"보다 "모십니다" 사용 | 초대/존경의 톤 |

### 6-3. 등급 유지/하향 커뮤니케이션

등급 유지 조건 미충족 시, 즉시 강등하지 않고 유예 기간을 두되 자연스럽게 활동을 독려합니다.

**유예 안내 (활동 감소 감지 시):**

```
{partnerName}님, 안녕하세요.

최근 주문이 뜸하신 것 같아 안부를 전해요.
바쁘신 시간 보내고 계신 건 아닌지요.

혹시 필요하신 재료가 있으시거나
궁금하신 점이 있으시면 언제든 연락주세요.

다시 함께할 수 있는 날을 기다리고 있을게요.

PRESSCO21 드림
```

**설계 의도**:
- "등급이 하향될 수 있습니다" 같은 경고 톤 완전 배제
- 안부 인사 형식으로 자연스럽게 관계 유지
- 다시 돌아올 수 있다는 문을 열어둠

## 7. 브랜드 일관성 체크리스트

이 문서의 모든 카피와 디자인 가이드가 PRESSCO21 브랜드 톤에 부합하는지 확인하는 체크리스트입니다.

### 7-1. 카피 톤 체크

- [ ] 5대 톤 준수: 우아 + 따뜻 + 신뢰 + 자연 + 포용
- [ ] 초대/권유형 표현 사용 ("~해볼까요?", "~해보세요")
- [ ] 과도한 판촉/긴급성 표현 미사용
- [ ] 경쟁사 비교/폄하 표현 미사용
- [ ] 초보자 친화적 언어 사용
- [ ] "거절/부적합/미달/강등/탈락" 등 부정적 단어 미사용
- [ ] 슬로건 연결성: "Forever and ever and Blooming"

### 7-2. 비주얼 일관성 체크

- [ ] 배지 색상이 브랜드 컬러 시스템과 조화
- [ ] 프라이머리 그린 `#7d9675` 기축 유지
- [ ] 포인트 컬러 `#b89b5e` (골드) VIP 전용
- [ ] 엠버서더 퍼플 `#8b6fae`은 엠버서더 칭호에만 사용
- [ ] Pretendard 폰트 사용
- [ ] 원형 배지 형태 일관성

### 7-3. 메타포 일관성 체크

- [ ] 회원 = 씨앗 (Seed) 메타포 일관 사용
- [ ] 인스트럭터 = 뿌리 (Root) 메타포 일관 사용
- [ ] 파트너스 = 꽃밭 (Bloom Field) 메타포 일관 사용
- [ ] VIP = 정원사 (Gardener) 메타포 일관 사용
- [ ] 엠버서더 = 별빛 (Starlight) 메타포 일관 사용 (꽃 메타포와 분리)
- [ ] 성장 경로 순서: 씨앗 -> 뿌리 -> 꽃밭 -> 정원사
