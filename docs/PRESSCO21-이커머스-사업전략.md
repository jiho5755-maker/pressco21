## 1. 경영 요약

### 1.1 사업 개요

PRESSCO21은 30년 이상의 역사를 가진 압화(프레스드플라워) 및 보존화 전문 브랜드다.
꽃 공예 재료(압화, 레진, 캔들, 석고, 비즈, 인조꽃 등)를 국내 직제조, 중국 사입, 국내 사입의
3가지 조달 경로로 확보하여 8개 온라인 채널에서 판매하고 있다.

### 1.2 핵심 자산

| 자산 | 설명 | 전략적 활용 |
|------|------|-----------|
| **30년 브랜드 헤리티지** | 압화/보존화 분야 국내 최장 역사 | 프리미엄 포지셔닝, B2B 신뢰도 |
| **자체 제조 역량** | 일부 제품 국내 직제조 | 원가 통제, 차별화 상품 |
| **중국 조달 네트워크** | 지사 운영 기반 중국 사입 | 가격 경쟁력 확보 |
| **유튜브 채널** | 카페24 유튜브쇼핑 연동 | 콘텐츠 커머스, 자사몰 유입 |
| **파트너클래스 플랫폼** | n8n + NocoDB 기반 클래스 마켓플레이스 | 수수료 수익 + 재료 판매 선순환 |
| **메이크샵 자사몰** | foreverlove.co.kr (D4/카멜레온) | 최고 마진 채널, 충성 고객 허브 |
| **파트너맵** | 전국 파트너 공방 지도 | O2O 연계, 파트너 생태계 |

### 1.3 전략적 과제

1. **원가-마진 관리 체계 부재**: 8개 채널 각각의 실질 순마진을 정확히 파악하지 못하고 있음
2. **쿠팡 의존도 리스크**: 쿠팡 채널(윙+로켓배송+그로스) 합산 매출 비중이 높을 경우 정책 변경 리스크
3. **자사몰 비중 강화 필요**: 가장 마진이 높은 채널이지만 트래픽 확보가 과제
4. **파트너클래스 수익화**: 플랫폼은 구축되었으나 초기 파트너 모집과 수익 모델 검증이 미완

### 1.4 핵심 전략 방향

```
[가치 극대화 삼각형]

         자사몰 마진 극대화
              /\
             /  \
            /    \
           /  핵심  \
          / 전략 방향  \
         /____________\
   채널 포트폴리오    파트너클래스
     리스크 분산      선순환 생태계
```

- **단기** (Q1-Q2): 원가-마진 데이터 인프라 구축, 채널별 수익성 가시화
- **중기** (Q3-Q4): 데이터 기반 가격/채널 최적화, 파트너클래스 수익화
- **장기** (2027+): 자사몰 비중 40% 이상, 유튜브-자사몰 직접 전환 파이프라인 확립

## 2. 채널 포트폴리오 전략

### 2.1 8개 채널 역할 분류

각 판매 채널에 전략적 역할을 부여하여 채널 간 자원 배분을 최적화한다.

| 채널 | 수수료(합산) | 역할 분류 | 전략적 포지션 | 중점 상품 |
|------|-----------|----------|-------------|----------|
| **메이크샵 자사몰** | ~3.3% (PG) | **수익형** | 마진 최우선, 충성 고객 허브 | 전 상품 (특히 고마진, 세트) |
| **네이버 스마트스토어** | ~5.7% | **트래픽형** | 신규 고객 유입 핵심 | 입문 키트, 인기 재료 |
| **카페24 유튜브쇼핑** | ~4~6% | **브랜딩형** | 콘텐츠 커머스, 팬 전환 | 영상 연관 재료, 키트 |
| **쿠팡 윙** | ~10.8% | **볼륨형** | 대량 판매, 시장 점유 | 소모재, 기본 재료 |
| **쿠팡 로켓배송** | 납품가 협의 | **볼륨형** | 안정적 볼륨, 풀필먼트 위임 | 고회전 상품, 표준 사이즈 |
| **쿠팡 로켓그로스** | ~10.8%+풀필먼트 | **볼륨형** | 로켓배송 느낌+가격 통제권 | 소형 경량 상품 |
| **11번가** | ~12% | **보조 트래픽형** | SKT 고객 접점 | 시즌 상품, 프로모션 |
| **오늘의집** | ~15~20% | **프리미엄형** | 인테리어/홈리빙 타깃 | 프리미엄 키트, 완성품, 인테리어 소품 |

### 2.2 채널별 역할 상세 분석

#### [수익형] 메이크샵 자사몰 -- 핵심 마진 엔진

```
수수료: PG ~3.3%만 발생 (8개 채널 중 최저)
순마진율: 타 채널 대비 7~17%p 높음

동일 상품(COGS 5,000원, 판매가 15,000원) 기준:
  메이크샵: 순마진 ~30.0% (공급가 13,636 - COGS 5,000 - PG 450 - 배송 2,500 = 5,686원)
  네이버:   순마진 ~24.3% (13,636 - 5,000 - 777 - 2,500 = 5,359원) ... -5.7%p
  쿠팡 윙:  순마진 ~19.2% (13,636 - 5,000 - 1,473 - 2,500 = 4,663원) ... -10.8%p
  오늘의집:  순마진 ~12.2% (13,636 - 5,000 - 2,727 - 2,500 = 3,409원) ... -17.8%p
```

**자사몰 강화 전략 5가지:**

| # | 전략 | 구체적 실행 방안 | 기대 효과 |
|---|------|----------------|----------|
| 1 | **자사몰 전용 상품** | 세트 번들, 한정판, 시즌 키트를 자사몰에서만 판매 | 채널 이동 동기 부여 |
| 2 | **유튜브 -> 자사몰 직결** | 영상 설명/카드에 자사몰 링크, 유튜브쇼핑 -> 자사몰 전환 유도 | 콘텐츠 유입 극대화 |
| 3 | **적립금/쿠폰 차등** | 자사몰 구매 시 적립금 5%, 타채널 연계 없음 | 재구매 유도 |
| 4 | **파트너클래스 재료 연동** | 수강 완료 -> "이 재료 자사몰에서 구매" CTA | 수강생 -> 고객 전환 |
| 5 | **CRM 이메일 마케팅** | 구매 후 D+7 관련 상품 추천, 시즌 뉴스레터 | LTV 극대화 |

#### [트래픽형] 네이버 스마트스토어 -- 신규 유입 게이트웨이

```
역할: "네이버 검색 -> 발견 -> 첫 구매" 퍼널의 진입점
핵심 지표: 유입 수, 첫 구매 전환율, 자사몰 재구매 전환율

전략:
1. 네이버 검색광고(파워링크) + 쇼핑검색 광고 집중
2. 입문 키트/체험 세트를 3만원 무료배송 조건으로 셋업
3. 첫 구매자에게 자사몰 전용 쿠폰 동봉 (패키지 삽지)
4. 네이버 블로그/카페 체험단 운영 -> 리뷰 확보
```

#### [브랜딩형] 카페24 유튜브쇼핑 -- 콘텐츠 커머스

```
역할: "영상 시청 -> 재료 발견 -> 즉시 구매" 임펄스 전환
핵심 지표: 영상 -> 상품 클릭률, 장바구니 전환율

전략:
1. 유튜브 영상에 카테고리 자동 매칭된 상품 표시 (v4 시스템 활용)
2. 시리즈 콘텐츠: "이 재료로 할 수 있는 3가지 작품" -> 재료 구매 유도
3. 숏폼(Shorts) 활용: 30초 만들기 영상 + 하단 쇼핑 태그
4. 유튜브 -> 자사몰 연계: 영상 상세설명에 foreverlove.co.kr 링크
```

#### [볼륨형] 쿠팡 3개 채널 -- 시장 점유 + 볼륨

```
역할: 대규모 시장 접근, 볼륨으로 조달 협상력 확보
핵심 지표: 판매량, 납품가 대비 마진, 재고 회전율

채널 내 세분화:
  쿠팡 윙: 자체 배송으로 가격 통제권 유지, 마진 감시 필수
  로켓배송: 납품가 협상이 곧 수익 -> 고회전 표준 상품에 집중
  로켓그로스: 소형 경량 상품 (비즈, 작은 재료, 도구류)에 적합
```

#### [프리미엄형] 오늘의집 -- 고단가 프리미엄 라인

```
역할: 인테리어/홈리빙 타깃 고객에게 프리미엄 가격으로 판매
핵심 지표: 객단가, 순마진액(율이 아닌 '액'에 집중)

전략:
1. 수수료 15~20%를 감당할 수 있는 고단가 상품만 투입
   - 프리미엄 하바리움 키트 (5~8만원대)
   - 인테리어 완성품 (압화 액자, 보존화 꽃다발)
   - DIY 풀키트 (재료+도구+설명서 세트, 4~6만원대)
2. 오늘의집 콘텐츠(집들이/노하우) 연계 네이티브 광고
3. "오늘의집 전용 패키지" 프리미엄 포장
```

### 2.4 쿠팡 의존도 리스크 헤징 방안

쿠팡 3개 채널 합산 매출 비중이 40%를 초과하면 다음 리스크가 발생한다:

| 리스크 유형 | 상세 | 영향도 |
|-----------|------|--------|
| **수수료 인상** | 쿠팡이 수수료율을 일방적으로 인상할 가능성 | 순마진 직격타 |
| **납품가 인하 압력** | 로켓배송 갱신 시 납품가 하향 협의 | 마진 축소 |
| **정책 변경** | 카테고리 규정, 검색 알고리즘 변경 | 매출 급감 가능 |
| **정산 주기 변경** | 정산일 변경 시 자금 유동성 압박 | 현금흐름 영향 |
| **경쟁 입점** | 쿠팡 PB 상품 등장, 유사 셀러 대량 입점 | 가격 경쟁 심화 |

**헤징 전략 4단계:**

```
[1단계: 모니터링] 쿠팡 합산 비중 매월 확인
  -> 40% 이상: 주의
  -> 50% 이상: 경보
  -> 60% 이상: 긴급

[2단계: 분산 실행]
  - 네이버 광고비 증액 (쿠팡 축소분의 50%를 네이버에 배분)
  - 자사몰 전용 상품/프로모션 강화
  - 유튜브쇼핑 콘텐츠 제작 빈도 증가

[3단계: 자사몰 직접 전환 강화]
  - 쿠팡 구매자 대상 자사몰 전환 인센티브 (패키지 삽지)
  - "foreverlove.co.kr에서 재구매 시 10% 할인" 쿠폰
  - 유튜브 영상에서 자사몰 직접 링크만 사용

[4단계: 채널 정책 변경 대비]
  - 쿠팡과의 납품 계약 갱신 3개월 전 협상 준비
  - 대안 물류(CJ대한통운 등) 계약 유지
  - 로켓배송 탈퇴 시나리오 시뮬레이션 (매출/마진 영향 분석)
```

## 3. 상품 포트폴리오 수익성 분석 프레임워크

### 3.1 상품 수익성 매트릭스 (BCG 응용)

BCG 매트릭스를 꽃 공예 재료 비즈니스에 맞게 변형한다.
가로축: **매출 기여도(판매량)**, 세로축: **순마진율**

```
                 순마진율 높음 (25%+)
                      |
     [Cash Cow]       |       [Star]
     안정 수익형        |       성장 주도형
     "흑자 기둥"        |       "핵심 성장 엔진"
                      |
 ─────────────────────┼─────────────────────
                      |
     [Question Mark]  |       [Loss Leader]
     전환/정리 대상     |       트래픽 유도형
     "마진 검토 필요"   |       "미끼 상품"
                      |
                 순마진율 낮음 (15% 미만)

       매출 기여도 낮음 <-----> 매출 기여도 높음
```

#### [Star] 성장 주도형 -- 고마진 + 고매출

```
특징: 마진율 25% 이상 AND 월 매출 상위 20%
예시 상품군:
  - 압화 DIY 풀키트 (재료+도구+설명서, 3.5~5만원)
  - 레진 아트 프리미엄 세트 (몰드+레진+데코, 4~6만원)
  - 캔들 원데이 홈키트 (왁스+향료+용기, 3~5만원)
  - 시즌 한정 세트 (어버이날/크리스마스 등)

전략: 재투자 -> 광고비 집중, 콘텐츠 제작, 자사몰 메인 노출
목표: 전체 매출의 30~40%를 Star 상품이 차지
```

#### [Cash Cow] 안정 수익형 -- 고마진 + 안정 매출

```
특징: 마진율 25% 이상 AND 매출 안정적 (중간 수준)
예시 상품군:
  - 국내 직제조 소재 (프레스 시트, 특수 유리 등)
  - 전문가용 도구 (압화 프레스기, 레진 UV 조사기)
  - B2B 대량 재료 (교육기관용)

전략: 유지 -> 원가 절감, 가격 인상 기회 포착
목표: 안정적 현금흐름 확보
```

#### [Loss Leader] 트래픽 유도형 -- 저마진 + 고매출

```
특징: 마진율 15% 미만 AND 월 매출 상위
예시 상품군:
  - 기본 소모재 (글루건 스틱, 기본 비즈, 표준 리본)
  - 초입문자 체험 키트 (1만원 이하)
  - 쿠팡 로켓배송 납품 상품

전략: 통제 -> 판매량 유지하되 마진 침식 방지
  - 번들/세트화로 객단가 상승 유도
  - "기본 재료 + 프리미엄 재료" 조합 추천
  - 자사몰에서는 기본가, 쿠팡에서는 10% 인상
목표: 순마진율 최소 10% 유지, 객단가 향상을 통한 실질 수익 확보
```

#### [Question Mark] 전환/정리 대상 -- 저마진 + 저매출

```
특징: 마진율 15% 미만 AND 매출 하위 30%
예시 상품군:
  - 경쟁 심화된 범용 재료 (가격 경쟁으로 마진 하락)
  - 장기 체화 재고 상품
  - 트렌드 이탈 상품

전략: 분석 후 결정
  - 가격 인상 가능? -> Star/Cash Cow로 전환
  - 원가 절감 가능? -> 조달 재검토 (국내 -> 중국 사입 등)
  - 둘 다 불가? -> 할인 소진 후 단종 검토
목표: 분기당 1회 리뷰하여 포트폴리오 최적화
```

### 3.2 상품 분류 실행 방법

**구글시트 "플랫폼별마진" 시트에서 자동 분류:**

```
마진등급 수식:
  =IF(AND(K>=0.25, 매출>상위20기준), "Star",
   IF(AND(K>=0.25, 매출<=상위20기준), "CashCow",
   IF(AND(K<0.15, 매출>하위30기준), "LossLeader",
   IF(AND(K<0.15, 매출<=하위30기준), "QuestionMark",
   "Moderate"))))

조건부 서식:
  Star = 파란색 배경
  CashCow = 녹색 배경
  LossLeader = 주황색 배경
  QuestionMark = 빨간색 배경
  Moderate = 기본 (별도 관리 불필요)
```

### 3.3 세트 상품/번들 전략 -- 객단가 + 마진 동시 향상

#### 번들 유형 3가지

| 번들 유형 | 구성 방식 | 마진 효과 | 예시 |
|----------|----------|----------|------|
| **테마 번들** | 하나의 프로젝트에 필요한 모든 재료 | 객단가 2~3배, 마진율 5~10%p 상승 | "압화 에코백 만들기 세트" (압화 10종+에코백+접착+가위) |
| **기본+프리미엄 묶음** | Loss Leader + Star 조합 | Loss Leader 마진 보전, Star 판매 촉진 | "레진 기본 키트" (기본 레진+프리미엄 몰드+데코) |
| **수량 할인 번들** | 동일 상품 대량 패킹 | 단위 원가 절감, 마진율 유지 | "캔들 왁스 1kg -> 5kg 세트" (단가 20% 할인, 마진율 유지) |

#### 번들 가격 설정 공식

```
[테마 번들 가격 설정]

개별 구매 합산가: A + B + C + D = 35,000원
번들 할인율: 15% (체감 할인)
번들 판매가: 35,000 × 0.85 = 29,750원 -> 29,900원 (포인트 프라이싱)
번들 COGS: 개별 COGS 합계 - 패키징 절감 (10~15%)
번들 순마진: 일반적으로 개별 판매보다 3~8%p 높음

이유:
- 패키징 1회로 포장비 절감
- 배송 1회로 배송비 절감
- 고객은 "할인 받았다"고 느낌 (실제로는 마진 상승)
```

#### 번들 상품 운영 캘린더

| 시기 | 테마 번들 | 대상 채널 | 비고 |
|------|---------|----------|------|
| 3~4월 | 봄꽃 압화 키트 | 자사몰, 오늘의집 | 벚꽃 시즌 |
| 5월 | 어버이날 보존화 세트 | 전 채널 | 선물 수요 |
| 6~7월 | 여름 캔들 키트 | 자사몰, 네이버 | 캔들 성수기 |
| 9~10월 | 가을 하바리움 세트 | 자사몰, 오늘의집 | 인테리어 수요 |
| 11~12월 | 크리스마스 DIY 세트 | 전 채널 | 연말 선물 |
| 1~2월 | 신년 취미 입문 키트 | 네이버, 쿠팡 | 새해 취미 도전 |

### 3.4 조달 유형별 수익성 전략

| 조달 유형 | 마진 특성 | 최적 채널 | 전략 |
|----------|----------|----------|------|

- **발생 가능성**: 낮음(1) / 중간(2) / 높음(3)
- **영향도**: 낮음(1) / 중간(2) / 높음(3) / 치명적(4)

### 6.3 리스크별 상세 대응 방안

#### R1: CNY/KRW 환율 급등 (점수 6)

```
[예방]
- 분기별 환율 전망 모니터링 (한국은행, 증권사 리포트)
- 구글시트 환율 시트에서 주간 자동 트래킹
- 사입 물량 분기 계획 수립 시 환율 시나리오 3가지 (기준/상승/급등) 반영

[감지]
- GOOGLEFINANCE 수식으로 실시간 환율 모니터링
- CNY 5% 이상 변동 시 자동 경고 (구글시트 조건부 서식)

[대응]
- 5% 이내 변동: 기존 가격 유지, 마진 흡수
- 5~10% 변동: 중국 사입 비중 높은 상품 가격 3~5% 인상
- 10% 초과 변동:
  (1) 전 상품 원가 재계산
  (2) 판매가 5~10% 인상 또는 번들 구성 변경
  (3) 국내 사입 대체 가능한 품목 전환 검토
  (4) 대량 사입 시점 조정 (환율 안정 시까지 소량 유지)

[장기 대비]
- 국내 직제조 비중 확대 (환율 영향 없음)
- 중국 외 조달처 개발 (베트남, 인도 등)
- 환율 방향성 판단 시 선도 매입(Forward Buy) 전략
```

```
[예방]

- 업계 뉴스 모니터링

[대응]
- 수수료 1~2%p 인상: 쿠팡 판매가 인상으로 전가 (마진 유지)
- 수수료 3%p 이상 인상:
  (1) 저마진 상품 쿠팡 철수 -> 자사몰/네이버로 이동
  (2) 쿠팡 전용 상품을 고마진 상품 위주로 재구성
  (3) 로켓배송 -> 쿠팡 윙 전환 (가격 통제권 확보)
```

### 7.1 협상 가능 조건

채널과 수수료 협상이 가능한 상황:

| 협상 조건 | 설명 | 레버리지 강도 |
|----------|------|-------------|
| **월 매출 일정 규모 이상** | 채널별로 다르나 보통 월 500만원~3,000만원 이상 | 강 |
| **카테고리 내 상위 셀러** | 해당 카테고리 매출 상위 10~20% | 강 |
| **독점 상품/브랜드** | 타 채널에서 구할 수 없는 상품 보유 | 매우 강 |
| **프로모션 참여도** | 채널 기획전/행사에 적극 참여한 이력 | 중 |
| **입점 기간** | 장기 입점 셀러 (2년 이상) | 중 |
| **반품율/CS 우수** | 반품율 낮고 고객 만족도 높음 | 중 |
| **대량 입점** | 다수 상품을 한 번에 입점 | 중 |

### 7.2 채널별 협상 전략

#### 쿠팡 (윙/로켓배송/그로스)

```
[협상 가능 영역]
- 쿠팡 윙: 카테고리 수수료율 협상 (10.8% -> 9~10%)
- 로켓배송: 납품가(매입가) 협의 -- 이것이 사실상 수수료 협상
- 로켓그로스: 풀필먼트비 구간 협상

[협상 레버리지]
- "꽃 공예 재료" 니치 카테고리에서의 매출 점유율
- 로켓배송 상품의 판매량/회전율 데이터
- 쿠팡에서만 판매하는 전용 상품 수

[협상 팁]
- 쿠팡 담당 MD와 직접 소통 채널 확보 (이메일/전화)
- 분기별 판매 실적 리포트 제출 -> "다음 분기 확대 계획" 제시
- "다른 채널(네이버, 자사몰)로 물량 이동 가능"을 암시적으로 언급
- 신상품 출시 시 쿠팡 독점/우선 제공 카드 활용
```

#### 네이버 스마트스토어

```
[협상 가능 영역]
- 파워셀러/빅파워 등급 달성 시 수수료 할인 (자동 적용)
- 네이버 쇼핑 프리미엄 입점 시 별도 수수료 협의
- 네이버 광고 대량 집행 시 광고 수수료 할인

[등급별 수수료]
- 일반: 5.63%
- 파워: 5.63% (+ 추가 혜택: 로그, 쿠폰 등)
- 빅파워: 별도 협의 가능

[전략]
- 파워셀러 기준 달성을 우선 목표로 설정
- 네이버 쇼핑 라이브 참여 -> 추가 혜택 확보
- 네이버 블로그/카페 브랜드 채널 운영 -> 유입 다각화
```

#### 11번가

```
[협상 가능 영역]
- 카테고리 수수료 협상 (12% -> 10~11%)
- 셀러 등급(파워, 프리미엄) 달성 시 수수료 할인
- 기획전/프로모션 참여 시 일시적 수수료 감면

[전략]
- 11번가는 SKT 멤버십 연계 고객이 핵심
- 프로모션 참여도를 높여 담당 MD와 관계 구축
- 월 매출 일정 규모 이상 시 계약 수수료 협의 요청
```

#### 오늘의집

```
[협상 가능 영역]
- 카테고리 수수료 협상 (15~20% -> 13~17%)
- 전용 상품/기획전 참여 시 수수료 감면
- 오늘의집 콘텐츠 (집들이/노하우) 참여 시 마케팅 비용 지원

[전략]
- 오늘의집 수수료가 가장 높으므로 고단가 상품만 투입
- "인테리어 콘텐츠 + 상품" 패키지 제안 -> 콘텐츠 마케팅 비용 절감
- 오늘의집 PD와 기획전 공동 기획 -> 수수료 감면 협상
```

### 7.4 계약 수수료 vs 표준 수수료 -- 실제 영향 시뮬레이션

```
[시뮬레이션: 월 매출 1,000만원 상품군, 쿠팡 윙 기준]

표준 수수료 (10.8%):
  공급가 = 10,000,000 / 1.1 = 9,090,909원
  수수료 = 9,090,909 × 10.8% = 981,818원

계약 수수료 (9.0%, 협상 성공 시):
  수수료 = 9,090,909 × 9.0% = 818,182원

차이 = 981,818 - 818,182 = 163,636원/월

연간 절감액 = 163,636 × 12 = 1,963,632원 (약 200만원/년)

[시뮬레이션: 오늘의집 고단가 상품, 월 매출 500만원]

표준 수수료 (17.5%):
  공급가 = 5,000,000 / 1.1 = 4,545,455원
  수수료 = 4,545,455 × 17.5% = 795,455원

계약 수수료 (14.0%, 협상 성공 시):
  수수료 = 4,545,455 × 14.0% = 636,364원

차이 = 795,455 - 636,364 = 159,091원/월

연간 절감액 = 159,091 × 12 = 1,909,092원 (약 190만원/년)
```

## 8. 부록: 핵심 계산 공식 모음

### 8.1 마진 계산 공식

```
[기본 마진 계산 (VAT 포함가 기준)]
실질 매출(공급가) = 판매가(VAT 포함) / 1.1
채널 수수료 = 실질 매출 x 채널 수수료율
순마진 = 실질 매출 - COGS - 채널 수수료 - 배송비
순마진율 = 순마진 / 실질 매출 x 100

[쿠팡 로켓배송 마진 (매입형)]
순마진 = 납품가(매입가) - COGS
순마진율 = 순마진 / 납품가 x 100
```

### 8.2 판매가 역산 공식

```
[목표 마진율 기반 판매가 계산]
필요 공급가 = COGS / (1 - 수수료율 - 배송비율 - 목표마진율)
필요 판매가(VAT 포함) = 필요 공급가 x 1.1

예시: COGS 5,000원, 네이버(5.7%), 배송비율 10%, 목표마진 25%
  필요 공급가 = 5,000 / (1 - 0.057 - 0.10 - 0.25) = 5,000 / 0.593 = 8,432원
  필요 판매가 = 8,432 x 1.1 = 9,275원 -> 9,900원 (올림)
```

### 8.3 광고 ROAS 공식

```
[광고비 여력 계산]
최대 광고비/건 = 순마진 x (1 - 최소 마진 보존율)
목표 ROAS = 판매가 / 최대 광고비/건

예시: 판매가 15,000원, 순마진 3,000원, 최소 마진 보존 50%
  최대 광고비/건 = 3,000 x 0.5 = 1,500원
  목표 ROAS = 15,000 / 1,500 = 10.0
```

### 8.4 중국 사입 COGS 공식

```
COGS = 매입가(KRW) + 지사관리비(10%) + 국제운송비 + 관세

매입가(KRW) = 매입가(CNY) x 환율(KRW/CNY)
지사관리비 = 매입가(KRW) x 0.10
관세 = (매입가(KRW) + 국제운송비) x HS Code별 관세율
```

### 8.5 번들 마진 계산

```
[번들 가격/마진 설계]
개별 합산가 = A판매가 + B판매가 + C판매가
번들 판매가 = 개별 합산가 x (1 - 번들 할인율)
번들 COGS = A(COGS) + B(COGS) + C(COGS) - 패키징 절감
번들 순마진 = 번들 공급가 - 번들 COGS - 채널 수수료 - 배송비(1건)
```

### 8.6 파트너클래스 수수료 계산

```
[파트너클래스 수수료 (확정 정책)]
수수료 금액 = 수강료 x 수수료율
적립금 금액 = 수수료 금액 x 적립금 전환율(80%)
PRESSCO21 현금 = 수수료 금액 - 적립금 금액

SILVER(20%): 10만원 수강료 -> 수수료 2만원 -> 적립금 1.6만원 -> 현금 0.4만원
GOLD(25%):   10만원 수강료 -> 수수료 2.5만원 -> 적립금 2만원 -> 현금 0.5만원
PLATINUM(30%): 10만원 수강료 -> 수수료 3만원 -> 적립금 2.4만원 -> 현금 0.6만원
```
