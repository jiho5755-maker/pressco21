# 로즈 프롬프트 템플릿 (영상 팀장 - 콘텐츠/SNS)

> 텔레그램에서 플로라 봇(@pressco21_openclaw_bot)에게 아래처럼 말하면 됩니다.

---

## 1. 유튜브 영상 기획

```
압화 액자 만들기 유튜브 영상 스크립트 써줘.
타겟: 공예 초보자, 길이: 10분 내외
```

```
이번 달 유튜브 영상 주제 5개 추천해줘.
최근 인기 있는 꽃공예 트렌드 반영해서.
```

```
이 영상 제목 후보 3개 만들어줘:
내용은 레진 하바리움 만들기, 초보자용
```

```
영상 설명란(디스크립션) 써줘:
제목: "압화 핸드폰케이스 만들기 | 5분 완성"
```

---

## 2. SNS 콘텐츠

```
인스타그램 게시글 캡션 써줘:
사진은 압화 액자 완성작, 봄 시즌 느낌으로
```

```
이번 주 인스타 릴스 아이디어 3개 추천해줘
```

```
해시태그 추천해줘: 압화꽃 관련 인스타그램용, 10개 정도
```

---

## 3. 카피라이팅

```
신상품 소개 문구 써줘:
상품: 프리저브드 장미 미니 부케, 가격: 25,000원
특징: 1년 이상 보존 가능, 선물 포장 포함
```

```
이벤트 배너 문구 써줘:
봄맞이 전품목 15% 할인, 기간: 4/1~4/15
```

```
쿠팡 상품명 최적화해줘:
현재: "압화꽃 액자"
카테고리: 인테리어소품 > 액자
```

---

## 4. 콘텐츠 캘린더

```
4월 콘텐츠 캘린더 짜줘:
유튜브 주 1회, 인스타 주 3회, 블로그 주 1회
시즌 키워드: 봄, 어버이날, 스승의날
```

```
이번 달 클래스 일정 알려줘
```

---

## 5. 경쟁사/트렌드

```
요즘 꽃공예 유튜브에서 인기 있는 콘텐츠 유형이 뭐야?
```

```
압화 관련 검색 트렌드 알려줘
```

---

## 팁

- **구체적일수록 좋아요**: "캡션 써줘"보다 "봄 느낌 압화 액자 인스타 캡션 써줘"가 더 좋은 결과
- **수정 요청 가능**: "좀 더 캐주얼하게", "이모지 추가해서" 같이 피드백하면 수정해줘요
- **클래스 정보 조회 가능**: 파트너클래스 일정이나 정보 물어보면 알려줄 수 있어요
