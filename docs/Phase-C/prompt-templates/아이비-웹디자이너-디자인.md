# 아이비 프롬프트 템플릿 (웹 디자이너 - 디자인/카피)

> 텔레그램에서 플로라 봇(@pressco21_openclaw_bot)에게 아래처럼 말하면 됩니다.

---

## 1. 상품 상세페이지 카피

```
상세페이지 카피 써줘:
상품명: 압화꽃 레진 트레이
가격: 45,000원
특징: 천연 압화꽃, 에폭시 레진, 핸드메이드, 크기 20x15cm
```

```
이 상품 특징을 불릿포인트로 정리해줘:
- 프리저브드 장미 사용
- 유리돔 케이스 포함
- 선물 포장 무료
- 1년 이상 보존
```

```
상세페이지 구매 유도 문구 써줘. "지금 주문하면 ~" 스타일로.
```

---

## 2. 배너/이벤트 카피

```
메인 배너 카피 써줘:
이벤트: 봄맞이 신상품 출시
느낌: 따뜻하고 고급스러운
글자 수: 메인 카피 15자 이내, 서브 카피 30자 이내
```

```
카테고리 배너 카피 3종 써줘:
1. 압화 카테고리
2. 레진공예 카테고리
3. 하바리움 카테고리
```

```
팝업 배너 문구 써줘:
내용: 회원가입 시 3,000원 쿠폰, 첫 구매 무료배송
```

---

## 3. 상품명 최적화

```
이 상품명 검색 잘 되게 다듬어줘:
현재: "예쁜 꽃 액자"
실제 상품: 압화꽃이 들어간 A4 크기 원목 액자
```

```
메이크샵 상품명 규칙에 맞게 써줘:
상품: 레진 플루이드 아트 시계
```

---

## 4. 색상/디자인 참고

```
프레스코21 브랜드 색상 알려줘
```

```
이번 시즌 꽃공예 트렌드 컬러 추천해줘
```

---

## 5. 클래스 관련

```
현재 진행 중인 클래스 목록 알려줘
```

```
클래스 소개 문구 써줘:
클래스명: 압화 캔들홀더 원데이클래스
대상: 초보자, 소요시간: 2시간, 가격: 55,000원
```

## 팁

- **브랜드 톤 자동 적용**: 프레스코21 스타일에 맞는 카피를 써줘요
- **글자 수 지정 가능**: "20자 이내로" 같이 제한하면 딱 맞춰줘요
- **여러 버전 요청**: "3가지 버전으로 써줘" 하면 선택할 수 있어요
- **수정 요청**: "좀 더 고급스럽게", "젊은 느낌으로" 등 톤 조정 가능
