# 프레스코21 AI 비서 설정 가이드

## 왜 하는 건가요?

회사에서 업무를 도와주는 **AI 비서**를 도입합니다.
텔레그램 앱에서 AI 비서에게 말을 걸면 업무에 필요한 답변을 바로 받을 수 있어요.

설정은 딱 **5분**이면 끝납니다.

---

## 순서대로 따라하세요

### STEP 1. 텔레그램 앱 설치

- **아이폰**: App Store에서 "Telegram" 검색 → 설치
- **안드로이드**: Play Store에서 "Telegram" 검색 → 설치
- **PC**: https://desktop.telegram.org 에서 다운로드 (선택)

설치 후 전화번호로 회원가입하세요. (카카오톡 가입하는 것과 비슷합니다)

---

### STEP 2. 내 Chat ID 확인하기

AI 비서가 "이 사람이 누구인지" 알아보려면 본인의 고유번호(Chat ID)가 필요합니다.

1. 텔레그램 앱을 엽니다
2. 상단 검색창(돋보기)을 누릅니다
3. **@userinfobot** 을 정확히 입력하고 검색합니다
4. "UserInfoBot"이라는 봇이 나옵니다 → 누릅니다
5. 하단의 **시작(Start)** 버튼을 누릅니다
6. 봇이 자동으로 메시지를 보내줍니다

메시지에서 **Id** 옆에 있는 숫자가 본인의 Chat ID입니다.

예시:
```
Id: 1234567890
First: 재혁
Last: 이
```

이 숫자(**1234567890** 부분)를 장지호 팀장에게 알려주세요.

---

### STEP 3. 장지호 팀장에게 보내기

아래 내용을 카톡이나 문자로 보내주세요:

```
이름:
직함:
텔레그램 Chat ID 숫자:
```

예시:
```
이름: 이재혁
직함: 물류운영 과장
텔레그램 Chat ID 숫자: 1234567890
```

---

### STEP 4. AI 비서 연결 (장지호 팀장이 해줍니다)

팀장이 설정을 완료하면 안내가 갑니다.
그 후에 텔레그램에서 AI 비서 봇을 검색해서 대화를 시작하면 됩니다.

---

## 자주 묻는 질문

**Q. 텔레그램이 뭔가요?**
카카오톡 같은 메신저 앱입니다. 전 세계에서 많이 쓰는 앱이에요.

**Q. Chat ID가 뭔가요?**
텔레그램에서 나를 구별하는 고유 숫자입니다. 전화번호가 아니라 텔레그램 내부 번호예요.

**Q. 개인정보가 노출되나요?**
Chat ID는 텔레그램 안에서만 쓰는 숫자입니다. 전화번호나 개인정보와는 다릅니다.

**Q. 비용이 드나요?**
아니요. 텔레그램 앱도 무료, AI 비서도 회사에서 제공합니다.

**Q. 꼭 해야 하나요?**
네, 업무 효율화를 위해 전 직원이 설정해주셔야 합니다. 5분이면 끝나요!
