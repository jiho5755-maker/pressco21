# 웹디자이너용 덕테이프 활용 가이드 — 브랜드스토리/상세페이지/배너

작성일: 2026-04-23  
대상: PRESSCO21 웹디자이너  
목적: ChatGPT Pro의 ChatGPT Images 2.0을 API 없이 웹/앱에서 수동 사용해 상세페이지·배너·브랜드스토리 이미지 제작 효율과 완성도를 높인다.

## 0. 한 줄 결론

디자이너에게 덕테이프는 “그림을 대신 그려주는 도구”가 아니라 **실물 사진을 더 좋은 상세페이지·배너 장면으로 바꿔주는 보조 디자이너**로 쓰게 해야 한다.

현재처럼 실물 사진 1장을 기준으로 보정/배경 합성을 하되, 앞으로는 아래 3가지를 추가한다.

1. **사진을 바꾸지 말고 장면만 바꾼다.**
2. **프롬프트를 매번 새로 쓰지 말고 템플릿에 끼워 넣는다.**
3. **한 번 만든 좋은 프롬프트를 다음 상세페이지에도 재사용한다.**

## 1. 디자이너 사용 수준 진단

### 현재 사용 방식

- 실물 사진 1장을 넣고 배경 합성 또는 약한 보정 위주로 사용한다.
- 결과물이 이질감이 적어서 실제 운영 만족도는 있다.
- 다만 프롬프트 관리, 버전 관리, 실패 원인 기록, 사용처별 규격화는 약하다.

### 목표 사용 방식

- “감으로 요청”이 아니라 “업무 유형별 고정 프롬프트”로 요청한다.
- ChatGPT 결과물을 최종본으로 보지 않고, 포토샵/피그마/캔바에서 마감할 중간 재료로 본다.
- 상세페이지, 배너, 브랜드스토리마다 다른 기준을 적용한다.
- 성공 프롬프트와 실패 프롬프트를 남겨 다음 작업 시간을 줄인다.

## 2. 반드시 지킬 원칙

### 원칙 A. 제품은 보존, 배경만 개선

실물 제품 사진을 넣을 때는 항상 아래 문장을 넣는다.

```text
첨부한 실제 제품의 형태, 색상, 질감, 크기, 꽃의 종류, 구성품은 절대 바꾸지 말고 그대로 유지해 주세요. 배경, 조명, 그림자, 주변 소품, 여백만 상세페이지에 어울리게 정리해 주세요.
```

### 원칙 B. 글자는 이미지 안에 많이 넣지 않기

덕테이프는 한글 표현력이 좋아졌지만, 상세페이지 운영에서는 아직 **중요 문구는 HTML/CSS 또는 편집 프로그램에서 직접 넣는 방식**이 안전하다.

이미지 안에 넣어도 되는 것:

- 짧은 제목 1개
- 2~4글자 라벨
- 장식적 소제목

이미지 안에 넣지 말 것:

- 가격
- 할인율
- 배송/반품 조건
- 구성품 상세
- 긴 설명문
- 법적/인증 표현

### 원칙 C. 3안만 만들고 고른다

한 작업에서 이미지를 무한히 뽑으면 오히려 시간이 낭비된다.

- 1차: 3안 생성
- 선택: 가장 좋은 1안 선택
- 2차: 선택한 1안만 수정
- 최종: 포토샵/피그마/캔바에서 마감

### 원칙 D. 파일명과 프롬프트를 같이 저장

이미지가 마음에 들면 프롬프트도 자산이다.

- 이미지 파일
- 사용 프롬프트
- 원본 사진명
- 사용처
- 수정한 이유

이 5개를 같이 남긴다.

## 3. 덕테이프로 먼저 맛볼 브랜드스토리 작업 5개

브랜드스토리는 상품 판매보다 “신뢰와 품격”이 중요하므로, AI 티가 적고 사진 기반인 작업부터 시작한다.

| 우선순위 | 작업 | 입력 | 결과물 | 목적 |
|---|---|---|---|---|
| 1 | 대표 히어로 배경 고도화 | 기존 브랜드/작품 사진 1장 | 넓은 여백의 고급 히어로 이미지 | 첫인상 개선 |
| 2 | 압화 700종+ 연구 무드컷 | 압화/작업대 사진 | 연구실/아카이브 느낌 배경 | 전문성 강조 |
| 3 | 1982~현재 헤리티지 타임라인 배경 | 옛 사진 또는 작품 사진 | 오래된 기록/아카이브 톤 | 업력 전달 |
| 4 | 출판/교육 섹션 보조 이미지 | 책/수업 사진 | 따뜻한 교육 현장 무드 | 교육 신뢰 |
| 5 | 갤러리 썸네일 배경 통일 | 작품 사진 여러 장 | 같은 톤의 썸네일 배경 | 페이지 완성도 |

## 4. 디자이너 5단계 작업 루틴

### Step 1. 원본 사진 고르기

좋은 원본 기준:

- 제품/작품이 선명하다.
- 그림자가 너무 강하지 않다.
- 색이 실제와 가깝다.
- 배경이 복잡해도 제품 윤곽은 살아 있다.
- 사진 한 장에 목적이 하나만 있다.

피해야 할 원본:

- 흔들린 사진
- 이미 과보정된 사진
- 제품 일부가 잘린 사진
- 실제 색과 너무 다른 사진
- 다른 브랜드 로고나 사람 얼굴이 섞인 사진

### Step 2. 사용처를 먼저 정하기

프롬프트 전에 아래 중 하나를 먼저 고른다.

- 브랜드스토리 히어로
- 상세페이지 섹션 이미지
- 모바일 배너
- PC 와이드 배너
- SNS/카카오 공지 이미지
- 상품 옵션 설명 이미지

### Step 3. 고정 프롬프트에 끼워 넣기

매번 새로 쓰지 말고, 아래 템플릿에서 대괄호만 바꾼다.

```text
첨부한 실제 사진을 기준으로 [사용처]에 사용할 이미지를 만들어 주세요.
실제 제품/작품의 형태, 색상, 질감, 크기, 꽃 종류, 구성품은 절대 바꾸지 마세요.
바꿔도 되는 것은 배경, 조명, 그림자, 주변 소품, 여백, 전체 분위기뿐입니다.
브랜드 톤은 우아함, 따뜻함, 신뢰, 자연스러움입니다.
결과는 [비율]로 만들어 주세요.
텍스트는 넣지 말고, 나중에 문구를 얹을 수 있는 여백을 남겨 주세요.
AI 이미지처럼 과장된 색감, 비현실적인 꽃, 과한 광택, 가짜 로고는 피해주세요.
```

### Step 4. 1차 결과를 A/B/C로 고르기

- A: 바로 편집 프로그램에서 마감 가능
- B: 배경/크롭/조명만 다시 요청
- C: 제품이 바뀌었거나 AI 티가 강해서 폐기

### Step 5. 좋은 결과만 프롬프트 저장

좋은 이미지가 나오면 아래 양식으로 남긴다.

```text
작업명:
원본 사진:
사용처:
사용 프롬프트:
좋았던 점:
다음에 반복할 점:
주의할 점:
```

## 5. 복사해서 쓰는 프롬프트 템플릿

### 5-1. 브랜드스토리 히어로 배경

```text
첨부한 실제 사진을 기준으로 PRESSCO21 브랜드스토리 페이지 히어로 이미지로 사용할 와이드 비주얼을 만들어 주세요.
실제 작품/제품의 형태, 색상, 질감, 꽃 종류는 절대 바꾸지 마세요.
배경만 고급스럽고 조용한 브랜드 헤리티지 분위기로 정리해 주세요.
느낌은 26년 업력, 꽃공예 연구, 압화 아카이브, 따뜻한 장인정신입니다.
색감은 세이지 그린, 크림, 베이지, 짙은 그린 계열로 차분하게 맞춰 주세요.
비율은 16:9입니다.
중앙 또는 우측에는 제목 문구를 얹을 수 있는 여백을 남겨 주세요.
이미지 안에 글자, 로고, 가격, 가짜 인증마크는 넣지 마세요.
AI 느낌이 강한 초현실적 꽃, 과한 빛번짐, 비현실적인 광택은 피해주세요.
```

### 5-2. 상세페이지 섹션 이미지

```text
첨부한 실제 제품 사진을 기준으로 쇼핑몰 상세페이지 중간 섹션에 들어갈 고급 제품 연출 이미지를 만들어 주세요.
제품의 형태, 색상, 크기, 질감, 구성품은 절대 바꾸지 말고 그대로 유지해 주세요.
배경은 깔끔한 공방 테이블 또는 자연광이 들어오는 작업 공간처럼 정리해 주세요.
주변 소품은 꽃공예 도구, 종이, 작은 유리 용기 정도만 최소한으로 사용해 주세요.
텍스트는 넣지 말고, 왼쪽 또는 오른쪽에 설명 문구를 넣을 여백을 남겨 주세요.
비율은 4:5입니다.
실제보다 더 고급스럽게 보이되, 상품 구성이나 품질을 과장하지 마세요.
```

### 5-3. PC 와이드 배너 배경

```text
첨부한 실제 사진을 사용해 쇼핑몰 PC 와이드 배너 배경을 만들어 주세요.
실제 제품/작품은 바꾸지 말고, 배경과 조명만 고급스럽게 정리해 주세요.
비율은 3:1 와이드 배너입니다.
모바일에서 중앙 크롭되어도 핵심 피사체가 잘리지 않도록 중앙 안전영역을 지켜 주세요.
왼쪽 40% 영역은 카피를 얹을 수 있도록 너무 복잡하지 않은 여백으로 남겨 주세요.
텍스트, 로고, 가격, 할인 문구는 넣지 마세요.
브랜드 톤은 세이지 그린, 크림, 자연광, 차분한 공예 스튜디오입니다.
```

### 5-4. 모바일 배너 배경

```text
첨부한 실제 사진을 사용해 모바일 쇼핑몰 배너 이미지를 만들어 주세요.
비율은 4:5입니다.
제품/작품의 형태, 색상, 질감은 절대 바꾸지 마세요.
핵심 피사체는 중앙에 크게 두고, 위쪽에는 짧은 제목을 넣을 수 있는 여백을 남겨 주세요.
배경은 단정한 크림색 또는 세이지 그린 계열로 정리해 주세요.
작은 화면에서도 제품이 바로 보이게 단순한 구도로 만들어 주세요.
텍스트, 로고, 가격은 넣지 마세요.
```

### 5-5. 배경만 제거/정리

```text
첨부한 실제 사진에서 제품/작품은 픽셀 단위로 최대한 유지하고, 배경만 깔끔하게 정리해 주세요.
제품의 색상, 꽃의 위치, 질감, 그림자는 실제와 다르게 바꾸지 마세요.
배경은 투명 또는 아주 밝은 크림색으로 만들어 주세요.
가장자리 윤곽이 뭉개지지 않게 해 주세요.
결과는 상세페이지 합성용으로 사용할 예정입니다.
```

### 5-6. 실패한 결과 수정 요청

```text
방금 결과에서 제품 형태와 색상이 원본과 달라졌습니다.
제품/작품 자체는 원본 사진과 동일하게 되돌리고, 배경과 조명만 다시 정리해 주세요.
특히 꽃의 색, 위치, 크기, 질감은 원본과 같아야 합니다.
```

```text
방금 결과는 AI 느낌이 강합니다.
더 자연스러운 실제 촬영 사진처럼 바꿔 주세요.
과한 빛번짐, 비현실적인 꽃 색감, 플라스틱 같은 질감, 과도한 광택을 줄여 주세요.
```

```text
방금 결과는 배너로 쓰기엔 여백이 부족합니다.
제품은 그대로 두고, [왼쪽/오른쪽/위쪽]에 문구를 얹을 수 있는 깨끗한 여백을 더 만들어 주세요.
```

## 6. 덕테이프를 쓰면 좋은 업무와 쓰면 안 되는 업무

### 적극 사용

- 실제 사진의 배경 정리
- 상세페이지 무드컷 제작
- 배너 배경 시안 3안 만들기
- 브랜드스토리 섹션별 분위기 통일
- 오래된 사진의 톤 보정
- 썸네일 배경 통일
- 포토샵 작업 전 합성 방향 잡기

### 신중 사용

- 실제 상품 대표 이미지
- 색상/크기/구성품이 중요한 옵션 설명
- 협회/인증/수상 관련 이미지
- 인물 얼굴이 포함된 사진
- 법적 문구가 들어가는 안내 이미지

### 사용 금지

- 실제와 다른 구성품을 추가한 상품 이미지
- 허위 인증마크/로고 생성
- 가격/할인율/배송조건이 들어간 이미지 자동 생성
- 타 브랜드 스타일을 직접 지목한 모방
- 고객 사진/개인정보가 포함된 이미지 업로드

## 8. 디자이너에게 줄 하루 루틴

### 오전 10분

- 오늘 만들 이미지 1~3개만 고른다.
- 원본 사진과 사용처를 정한다.
- 위 템플릿 중 하나를 복사한다.

### 작업 중 30~60분

- ChatGPT에서 3안 생성
- 가장 좋은 1안 선택
- 수정 요청 1~2회만 진행
- 결과 저장

### 마감 10분

- 좋은 프롬프트 저장
- 실패 이유 한 줄 기록
- 최종본/원본/프롬프트를 같은 폴더에 정리

## 9. 폴더 정리 규칙

Nextcloud 또는 로컬 작업 폴더에서 아래처럼 단순하게 운영한다.

```text
디자인/AI이미지/브랜드스토리/
  01_원본사진/
  02_생성시안/
  03_승인본/
  04_프롬프트/
  05_폐기/
```

파일명 예시:

```text
brandstory_hero_archive_20260423_v01.webp
brandstory_hero_archive_20260423_prompt.txt
brandstory_hero_archive_20260423_source.jpg
```

## 10. 품질 체크 10문항

이미지를 쓰기 전 아래 10개에 “예”가 8개 이상이어야 한다.

1. 원본 제품/작품이 변형되지 않았는가?
2. 실제 색상과 크게 다르지 않은가?
3. AI 티가 과하지 않은가?
4. 브랜드 컬러와 어울리는가?
5. 텍스트를 얹을 여백이 있는가?
6. 모바일에서 핵심이 잘리지 않는가?
7. 상세페이지에서 실제 사진과 섞어도 이질감이 적은가?
8. 불필요한 가짜 로고/문자/마크가 없는가?
9. 고객이 실제 구성품으로 오해할 요소가 없는가?
10. 같은 페이지의 다른 이미지와 톤이 맞는가?

## 11. 교육 방식

디자이너에게 처음부터 AI 이론을 설명하지 않는다. 아래 순서로 훈련한다.

### 1회차 — “제품은 그대로, 배경만 바꾸기”

- 원본 사진 3장으로 배경 정리만 실습
- 목표: 제품 변형 없이 결과 뽑기

### 2회차 — “상세페이지 여백 만들기”

- PC 배너/모바일 배너 각각 생성
- 목표: 문구 얹을 공간 확보

### 3회차 — “프롬프트 저장하고 재사용하기”

- 성공 프롬프트 5개를 `04_프롬프트` 폴더에 저장
- 목표: 다음 작업 때 복붙해서 재사용

### 4회차 — “브랜드스토리 1섹션 완성”

- 히어로 또는 철학 섹션용 이미지 1세트 제작
- 목표: 실제 페이지에 넣을 수 있는 후보 만들기

## 12. 대표/관리자 피드백 문장

디자이너에게 피드백할 때는 추상적으로 “AI 티 난다”라고만 말하지 말고 아래처럼 말한다.

- 제품은 좋은데 배경이 너무 광고 사진 같아요. 자연광 공방 느낌으로 낮춰주세요.
- 꽃 색이 실제보다 진해졌어요. 원본 색을 유지하라고 다시 요청해 주세요.
- 배너 문구를 얹을 공간이 부족해요. 왼쪽 40% 여백을 만들어달라고 해주세요.
- 이건 상세페이지 대표 이미지가 아니라 무드컷으로만 쓰는 게 맞겠어요.
- 이 프롬프트는 좋으니 저장해두고 다음 상세페이지에도 재사용해 주세요.

## 13. 성공 기준

첫 2주 성공 기준은 아래와 같다.

- 브랜드스토리 후보 이미지 18안 이상 생성
- 실제 사용할 승인 후보 3~5장 확보
- 재사용 가능한 프롬프트 5개 확보
- 디자이너가 혼자서 “원본 보존 + 배경 개선” 프롬프트를 작성 가능
- 상세페이지/배너 작업 시작 시간이 30% 이상 줄었다고 체감

## 15. 코드 적용 주의

이 가이드는 이미지 생성/디자인 작업용이다. 브랜드스토리 코드 파일은 현재 참조용 폴더 성격이 있으므로, 이미지 후보를 실제 쇼핑몰에 반영할 때는 별도 작업 브랜치에서 메이크샵 제약을 다시 확인한다.

- 메이크샵 가상태그는 수정하지 않는다.
- HTML/CSS/JS 붙여넣기 전 PC/모바일 크롭을 확인한다.
- 최종 이미지는 WebP 우선, 필요 시 JPG/PNG 폴백을 준비한다.
- 이미지 안 중요 문구보다 HTML/CSS 텍스트를 우선한다.
- 운영 대표 이미지와 AI 연출 이미지는 내부 파일명/폴더에서 구분한다.
