# ChatGPT Images 2.0 프롬프트 팩 v1

작성일: 2026-04-23

## 공통 프롬프트 원칙

- ChatGPT Pro 웹/앱의 `Create image` 또는 Images 화면에서 사용한다.
- API를 사용하지 않는다.
- 한 번에 후보는 최대 3안만 만든다.
- 한글 텍스트는 짧게 쓴다.
- 상세페이지의 사실 정보는 이미지 안에 과도하게 넣지 말고 HTML/CSS 텍스트로 분리한다.
- 실제 상품 사진이 필요한 곳에는 AI 이미지를 “연출 이미지”로만 사용한다.

## 공통 금지 문구

- 국내 1위, 완벽 보장, 누구나 월 000만원, 공식 인증, 유일한 수익 보장
- 실제 협회/공방/강사 동의 없이 특정 협회 추천처럼 보이는 표현
- 실제 구성품에 없는 도구/꽃/소품을 포함하는 표현

## PC-HERO-001 — 파트너클래스 상세 히어로

```text
PRESSCO21 파트너클래스 상세페이지 상단 히어로 이미지.
대상은 꽃공예 클래스를 처음 예약하려는 수강생.
목표는 “돈 값 할까” 불안을 낮추고, 전문 강사에게 배우는 따뜻한 신뢰감을 주는 것.
분위기: 우아함, 따뜻함, 자연스러움, 전문가의 정돈된 작업대.
시각 요소: 압화, 레진 도구, 하바리움 용기 중 하나를 중심에 배치. 손은 있더라도 왜곡 없이 자연스럽게. 실제 상품 사진처럼 과장하지 말 것.
한글 텍스트는 넣지 말고, 텍스트가 들어갈 여백을 중앙 안전영역 밖에 남겨둘 것.
비율: 16:9 웹 배너, 모바일 크롭을 고려해 핵심 피사체는 중앙에 둔다.
금지: 과도한 AI 느낌, 비현실적 꽃 색감, 실제 수업에 없는 고급 장비 과장, 로고 임의 생성.
```

## PC-BANNER-001 — 파트너클래스 모집 배너

```text
PRESSCO21 파트너클래스의 강사/공방 파트너 모집용 배너.
대상은 꽃공예 강사, 공방 운영자, 문화센터 강사.
핵심 메시지는 “강의 운영 부담은 줄이고, 수업에 집중하세요.”
분위기: 전문적이지만 따뜻하고 부담 없는 파트너십.
시각 요소: 정돈된 공방 책상, 수업 준비 키트, 일정 체크 느낌의 노트, 꽃공예 재료.
이미지 안 한글 문구: “수업에 더 집중하세요”만 크게, 나머지 설명은 넣지 말 것.
비율: 4:5와 16:9 둘 다 재생성하기 쉽도록 중앙 구도.
금지: 과장된 수익, 돈다발, 고압적인 세일즈 느낌, 플랫폼 수수료 장사처럼 보이는 표현.
```

## SHOP-HERO-001 — 재료/키트 상세 히어로

```text
PRESSCO21 자사몰 상품 상세페이지용 히어로 이미지.
대상은 압화/레진/하바리움 재료를 구매하려는 강사와 공방 운영자.
목표는 “이 재료로 어떤 완성작이 가능한지”를 빠르게 이해시키는 것.
분위기: 고급스럽고 실용적, 실제 공방에서 바로 쓸 수 있는 느낌.
시각 요소: 상품 재료와 완성작 예시를 함께 배치하되, 실제 상품 구성과 다른 구성품은 넣지 말 것.
텍스트: 이미지 안 텍스트 없음. 텍스트 여백만 확보.
비율: 1:1 또는 4:5.
금지: 실제보다 큰 크기/과장된 광택/구성품 오해/상표 임의 생성.
```

## SHOP-INFO-001 — 구성품 비교 인포그래픽

```text
PRESSCO21 상품 상세페이지에 들어갈 구성품 비교 인포그래픽 스타일 이미지.
대상은 초보 강사/공방 운영자.
목표는 어떤 재료를 언제 쓰는지 쉽게 보여주는 것.
스타일: 깔끔한 에디토리얼 인포그래픽, 한국 쇼핑몰 상세페이지에 어울리는 밝은 배경.
한글 텍스트는 정확해야 하며 다음 3개만 넣는다: “입문용”, “수업용”, “전문가용”.
아이콘/도식은 단순하게, 실제 제품처럼 보이는 사진풍은 피한다.
비율: 세로형 4:5.
금지: 가격, 할인율, 성능 보장, 실제 브랜드 로고 임의 생성.
```

## ASSOC-001 — 협회 제안 카드

```text
PRESSCO21 협회 제휴 제안서에 넣을 카드형 이미지.
대상은 꽃공예 협회 운영진.
목표는 협회원 전용 혜택, 공동 홍보, 자사몰 재방문 구조를 신뢰감 있게 보여주는 것.
분위기: B2B 제안서용, 단정하고 격식 있지만 차갑지 않음.
시각 요소: 협회 일정 카드, 회원 혜택 카드, 꽃공예 재료 패키지, 세미나 이미지가 하나의 생태계처럼 연결된 구조.
한글 텍스트: “회원 혜택”, “공동 홍보”, “전용 재료”만 사용.
비율: 16:9.
금지: 특정 협회 로고/명칭 임의 사용, 공식 인증처럼 보이는 문구, 수익 과장.
```

## EDU-001 — 교육자료 단계별 가이드

```text
PRESSCO21 교육자료용 단계별 작업 가이드 이미지.
대상은 꽃공예 초보자와 신규 직원.
목표는 작업 순서를 한눈에 이해시키는 것.
스타일: 깨끗한 교육용 일러스트/인포그래픽, 과한 장식 없음.
단계는 4단계로 구성: “준비”, “배치”, “고정”, “마감”.
각 단계에는 단순한 아이콘과 작은 작업 장면을 넣는다.
비율: A4 세로 또는 4:5.
금지: 위험한 작업 방식, 실제 도구와 다른 도구, 손가락 왜곡, 너무 작은 글씨.
```
