## 2. 컬러 시스템

### 2-1. Primary (주요 브랜드 컬러)

메인 디자인에 반드시 1개 이상 사용합니다.

| 이름 | HEX | 용도 |
|------|-----|------|
| **Soft Pink** | `#F8E8E0` | 배경, 카드, 섹션 구분 |
| **Ivory** | `#FFF8F0` | 메인 배경, 상세페이지 바탕 |
| **Green** | `#8BA888` | 포인트, CTA 버튼, 강조 |
| **Deep Brown** | `#5C4033` | 제목, 핵심 텍스트, 로고 |

### 2-2. Accent (보조 컬러)

시즌/이벤트에 따라 선택적으로 사용합니다.

| 이름 | HEX | 용도 |
|------|-----|------|
| **Gold** | `#C9A96E` | 프리미엄/고급 라인, 수료증 |
| **Rose** | `#D4907E` | 봄/어버이날 시즌 |
| **Sage** | `#A8BFA0` | 자연/힐링 콘텐츠 |
| **Lavender** | `#B8A9C9` | 특별 이벤트, 한정 라인 |

### 2-3. Neutral (중립 컬러)

텍스트, 구분선, 배경에 사용합니다.

| 이름 | HEX | 용도 |
|------|-----|------|
| **White** | `#FFFFFF` | 기본 배경 |
| **Light Gray** | `#F5F5F5` | 섹션 구분 배경 |
| **Gray** | `#999999` | 보조 텍스트, 캡션 |
| **Dark** | `#333333` | 본문 텍스트 |
| **Black** | `#1A1A1A` | 강조 제목 (제한적 사용) |

### 2-4. Semantic (기능 컬러)

UI 상태 표시용입니다.

| 이름 | HEX | 용도 |
|------|-----|------|
| **Success** | `#6B9E6B` | 완료, 성공, 재고 있음 |
| **Warning** | `#E0A84B` | 주의, 품절 임박 |
| **Error** | `#C75C5C` | 오류, 품절, 필수 입력 |
| **Info** | `#5C8DC7` | 안내, 도움말 |

### 2-5. 컬러 조합 가이드

**자주 쓰는 조합 (권장)**:
- 배경 Ivory + 텍스트 Deep Brown + 버튼 Green
- 배경 Soft Pink + 텍스트 Dark + 포인트 Gold
- 배경 White + 카드 Soft Pink + 텍스트 Deep Brown

**금지 조합**:
- Green + Rose (직접 인접 배치 — 대비 부족)
- Black 배경 + 어두운 텍스트 (브랜드 톤과 맞지 않음)
- 형광색, 네온색은 어떤 경우에도 사용 금지

## 6. 이미지 가이드

### 6-1. 사진 톤앤무드

| 구분 | 기준 |
|------|------|
| **조명** | 소프트 자연광 또는 스튜디오 라이팅. 강한 그림자 배제 |
| **색온도** | 따뜻함 (5500K~6000K 느낌) |
| **배경** | 순백, 아이보리, 밝은 우드톤, 린넨 질감 |
| **구도** | 정갈한 플랫레이 또는 45도 라이프스타일 |
| **소품** | 자연 소재 (나무, 린넨, 유리) — 플라스틱/메탈 최소화 |

### 6-2. AI 이미지 생성 vs 실촬영 판단

| AI 생성 가능 | 반드시 실촬영 |
|-------------|-------------|
| 이벤트/시즌 배너 | 신제품 메인 사진 |
| SNS 분위기 이미지 | 압화 작품 디테일 (실제 질감) |
| 키트 구성도 / 인포그래픽 | 수업 현장 / 인물 사진 |
| YouTube 썸네일 배경 | 제품 색상이 핵심인 이미지 |
| 카테고리 대표 이미지 | 원장님/강사님 인물 |

**핵심 원칙: "구매 결정에 직결되는 이미지 = 실촬영, 분위기/정보 전달 = AI 가능"**

## 8. 작업 프로세스

### 8-1. AI 이미지 제작 플로우

```
1. 승해 사원: 위 템플릿으로 ChatGPT Pro에서 이미지 3~5장 생성
2. 승해 사원: 그 중 2장 선별 → 다경 팀장에게 공유
3. 다경 팀장: 브랜드 톤 검수 → OK / 수정 피드백
4. (수정 시) 승해 사원: 피드백 반영 재생성 — 최대 2회전
5. 3회 이상 수정이 필요하면 → 실촬영 또는 후보정 전환
6. 최종 OK → 이미지 다운로드 → 메이크샵 어드민 업로드
```

### 8-2. 검수 체크리스트

이미지 승인 전 아래 항목을 확인합니다:

- [ ] 브랜드 컬러 범위 안에 있는가?
- [ ] 한글 텍스트가 깨지거나 이상한 글자가 없는가?
- [ ] 꽃/재료의 색상이 자연스러운가? (비현실적 색상 아닌가?)
- [ ] 톤이 "우아+따뜻"인가? (저렴하거나 차가운 느낌 아닌가?)
- [ ] 로고 삽입 여백이 확보되어 있는가?
- [ ] 지정된 사이즈 규격에 맞는가?
- [ ] AI 생성 인물 얼굴이 포함되어 있지 않은가?

## 9. Figma 파일 사용법

### 9-1. 브랜드 마스터 파일

- **파일 위치**: https://www.figma.com/design/tw2xLLJ7mSLdUgZKnbYwGJ
- **역할**: 모든 디자인의 컬러/폰트/간격 기준
- 새 디자인 시 이 파일의 변수(Variables)를 참조하여 사용
- 변수 패널 (오른쪽 사이드바 > Local Variables)에서 컬러/간격 확인

### 9-2. 변수 사용 방법

1. 디자인 작업 시 색상 선택할 때 → 색상 피커 하단의 "Variables" 탭 클릭
2. "PRESSCO21 Colors" 컬렉션에서 선택
3. 간격/여백 설정 시 → 숫자 입력란 옆 변수 아이콘 클릭 → "PRESSCO21 Spacing"에서 선택

### 9-3. 수정 권한

| 항목 | 수정 가능 | 비고 |
|------|----------|------|
| 변수 값 (컬러 HEX 등) | 다경 팀장만 | 변경 시 지호님에게 알림 |
| 새 컬러/스타일 추가 | 다경 팀장만 | 시즌 컬러 등 |
| 가이드 페이지 레이아웃 | 다경 팀장, 승해 사원 | |
| 작업 파일 (배너/상세페이지) | 자유 | 별도 파일에서 작업 |

## 10. 자주 묻는 질문

**Q. ChatGPT가 정확한 HEX 색상을 안 지키면?**
A. 프롬프트에 HEX를 명시해도 100% 일치하지 않을 수 있습니다. "비슷한 톤" 수준이면 OK. 정확한 색상이 필요하면 후보정(Figma/Canva에서 색상 오버레이)으로 보정합니다.

**Q. 한글이 깨져서 나오면?**
A. Images 2.0은 한글이 거의 완벽하지만, 간혹 글자가 이상할 수 있습니다. 그런 경우 텍스트 없이 이미지만 생성한 후 Figma/Canva에서 텍스트를 따로 올립니다.

**Q. 실제 제품 사진을 참고시키고 싶으면?**
A. ChatGPT 대화창에 실제 제품 사진을 첨부하고 "이 제품을 참고해서 만들어줘"라고 하면 됩니다. Thinking 모드를 켜면 더 정확합니다.

**Q. 같은 시리즈로 여러 장 만들고 싶으면?**
A. "위와 동일한 스타일로 [다른 제품/다른 문구]로 1장 더 만들어줘"라고 하면 일관된 시리즈가 됩니다.
