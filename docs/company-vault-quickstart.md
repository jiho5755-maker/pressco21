## 1. 이게 뭔가요?

회사 자료(상품 사진, 브랜드 파일, 세무 서류, 교육 자료 등)를
**한 곳에 모아서 누구든 찾을 수 있게** 만든 시스템입니다.

- 브라우저에서 접속 (앱 설치 불필요)
- 사무실·외부 어디서든 접근 가능
- 파일 올리면 자동으로 분류
- 상품 사진 올리면 **배경 제거 + 4채널 규격 자동 생성**

## 5. 파일 올리는 법

### 방법 A: 바로 올리기 (내가 어디에 넣을지 아는 경우)

1. 로그인
2. 맞는 폴더로 이동 (예: `products/forever-love/`)
3. 파일을 **드래그 앤 드롭**으로 올리기

### 방법 B: inbox에 던져넣기 (모르겠는 경우)

1. 로그인
2. `inbox` 폴더 열기
3. 한국어 폴더 중 맞는 곳 선택:

```
inbox/
├── 상품사진/       → 자동으로 products/에 정리됨
├── 교육자료/       → 자동으로 classes/에 정리됨
├── 마케팅소재/     → 자동으로 marketing/에 정리됨
├── 고객후기/       → 자동으로 marketing/ugc/에 정리됨
├── 세무증빙/       → 자동으로 finance/에 정리됨
├── 거래처자료/     → 자동으로 sourcing/에 정리됨
├── 브랜드자료/     → 자동으로 brand/에 정리됨
└── 기타/           → 관리자가 수동으로 분류
```

4. 파일 업로드
5. **10분 후 자동으로 정리됨**

## 6. 상품 사진 자동 처리 (디자인팀 필독)

`inbox/상품사진/`에 사진을 올리면 **자동으로**:

1. 배경 제거 (AI)
2. 4개 채널 규격으로 리사이즈:
   - 메이크샵: 860x860
   - 스마트스토어: 1000x1000
   - 쿠팡: 780x780
   - 11번가: 640x640
3. `vault/products/{브랜드}/{상품명}/` 에 자동 정리

### 파일명 규칙 (중요!)

```
{브랜드}_{상품명}_{용도}.확장자

예시:
  forever-love_로즈오일_상세메인.jpg     → products/forever-love/로즈오일/
  resiners_투명레진키트_쿠팡.jpg         → products/resiners/투명레진키트/
  레지너스_플라워코스터_상세.png          → products/resiners/플라워코스터/
```

파일명을 이 규칙대로 지으면 브랜드와 상품이 자동으로 분류됩니다.
규칙을 안 지키면 `_unsorted/` 폴더에 들어가고, 관리자가 수동 정리합니다.

## 7. 일반 파일 이름 짓는 법

하나만 기억하세요:

```
날짜_내용_용도.확장자

예시:
  20260402_로즈오일_쿠팡.jpg
  20260402_봄맞이배너_인스타.png
  20260402_한진택배_세금계산서.pdf
```

- 공백 대신 언더바 `_` 사용
- 카메라 기본 파일명 (`IMG_4023.jpg`) 은 가능하면 바꿔주세요

## 8. 파일 찾는 법

1. File Browser 상단의 **검색창**에 파일명 일부 입력
2. 또는 폴더를 탐색:
   - 상품 사진 → `vault/products/{브랜드}/`
   - 교육 자료 → `vault/classes/`
   - 마케팅 → `vault/marketing/`

## 9. 디자인팀 추가 설정 — Syncthing (선택)

Syncthing을 설치하면 **작업 폴더가 미니 PC와 자동 동기화**됩니다.
파일을 저장하면 자동으로 미니 PC에도 백업됩니다.

### 맥북 설치

1. https://syncthing.net/downloads/ 에서 macOS 버전 다운로드
2. 설치 후 실행
3. 브라우저에서 `http://127.0.0.1:8384` 열기
4. 원격 장치 추가 → 미니 PC Device ID 입력:
```
3XE4MY5-RZZL5WU-2LQMH2Z-7MGQWLU-IUZVOIL-3WJXJEW-LWEQDA2-4MHRAAC
```
5. 동기화할 폴더 설정 (대표님에게 문의)

### 윈도우 설치

1. https://syncthing.net/downloads/ 에서 Windows 64-bit 다운로드
2. 압축 풀고 실행
3. 브라우저에서 `http://127.0.0.1:8384` 열기
4. 원격 장치 추가 → 위 Device ID 입력
5. 동기화할 폴더 설정

> Syncthing은 선택사항입니다. File Browser만으로도 충분히 사용할 수 있습니다.
> Syncthing은 "저장하면 자동 백업" 이 필요한 디자이너/편집자에게 권장합니다.

## 10. 자주 묻는 질문

**Q. 파일을 잘못된 폴더에 올렸어요.**
→ 경영진 계정으로 로그인해서 맞는 폴더로 옮기면 됩니다.

**Q. 올린 파일이 안 보여요.**
→ inbox에 올렸으면 10분 후에 vault 해당 폴더를 확인하세요.

**Q. 외부에서 접속이 안 돼요.**
→ Tailscale이 켜져 있는지 확인하세요.

**Q. 파일을 실수로 삭제했어요.**
→ 직원 계정은 삭제 권한이 없으므로 걱정 없습니다.
→ 경영진이 삭제한 경우 관리자에게 문의하세요.

**Q. 미니 PC가 꺼져 있으면?**
→ 접속이 안 됩니다. 미니 PC는 항상 켜두는 것이 원칙입니다.
→ 모니터와 키보드는 연결 안 해도 됩니다.

### 비상 복구 (Oracle 서버 장애 시)

미니 PC 터미널에서:
```
sudo bash /srv/pressco21-backup-node/dr/restore.sh
```
화면 안내에 따라 진행하면 n8n + NocoDB가 미니 PC에서 임시 가동됩니다.

### 계정 비밀번호 변경이 필요할 때

미니 PC 터미널에서 (예: 디자인팀 비밀번호 변경):
```
sudo filebrowser users update pressco21_design -p "새비밀번호" -d /var/lib/pressco21-content-browser/filebrowser.db
sudo systemctl restart pressco21-content-browser
```
