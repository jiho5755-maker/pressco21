## 2. 기존 콘텐츠 활용 방안

### 2-1. 인스타그램 208개 게시물 활용

**광고 소재로 적합한 유형 (우선순위)**:

| 순위 | 게시물 유형 | 활용 방법 | 적합 세트 |
|------|-----------|---------|---------|
| 1 | **완성 작품 사진** (압화 액자, 레진 소품, 하바리움 등) | 캐러셀 구성, 단일 이미지 광고 | B, D |
| 2 | **만들기 과정 릴스** (타임랩스, 스텝바이스텝) | 그대로 광고 소재로 전환 (기존 게시물 광고) | C |
| 3 | **키트 언박싱/구성품 사진** | 세트 A 이미지 소재 | A |
| 4 | **시즌 특별 게시물** (어버이날, 크리스마스 등) | 시즌 캠페인 소재 | E |
| 5 | **고객 후기/리그램** | 사회적 증거 (social proof) 소재 | D |

**활용 불가/비추천 유형**:
- 텍스트 중심 공지성 게시물
- 단순 상품 나열 (카탈로그형)
- 해상도 낮은 초기 게시물

**기존 게시물 광고 (Boost) vs 신규 소재 제작**:
- 인게이지먼트가 높은 기존 게시물 → "기존 게시물 광고"로 바로 집행 가능 (추가 제작비 0원)
- 인게이지먼트가 낮지만 시각적으로 우수한 게시물 → 이미지만 추출해 새 광고 소재로 재구성

### 2-2. 유튜브 467개 영상 활용

**릴스/숏폼 전환 가능한 유형**:

| 유형 | 영상 수(추정) | 전환 방법 | 적합 세트 |
|------|-------------|---------|---------|
| **만들기 튜토리얼** | ~200개 | 핵심 과정 30~60초 하이라이트 추출 → 세로(9:16) 재편집 | C |
| **상품 소개/리뷰** | ~100개 | 상품 사용법 핵심 15~30초 발췌 | A, D |
| **시즌 특별 영상** | ~30개 | 시즌 키트 과정 편집 | E |
| **원장님 강의** | ~50개 | 전문성 어필 클립 (30초) | D |
| **이벤트/전시** | ~30개 | 브랜드 스토리 소재 | D |

**편집 방침**:
- 유튜브 가로(16:9) → 인스타 세로(9:16) 전환 시 핵심 작업 부분을 크롭
- 자막 필수 추가 (무음 시청 80% 이상)
- 처음 3초에 완성 결과물 먼저 보여주고 → 과정 역순 편집 (후킹)
- 음악은 Meta 사운드 컬렉션 무료 음원 사용

### 2-3. 자사몰 상품 이미지 활용

| 활용 가능 여부 | 조건 | 비고 |
|-------------|------|------|
| 바로 사용 가능 | 배경이 깔끔하고 해상도 1080px+ | 키트 상품, 주요 재료 |
| 편집 후 사용 | 배경 정리, 색 보정 필요 | 대부분의 상품 이미지 |
| 사용 불가 | 저해상도, 텍스트 과다, 카탈로그 나열형 | 구형 등록 상품 |

**권장 처리 방법**:
- 이미지 톤 통일: 자연광 느낌 + 따뜻한 아이보리 배경 (#fff9f0 근사)
- 브랜드 컬러 오버레이: 포레스트 그린(#425b51) 반투명 띠에 카피 배치
- 골드(#d4a373) 악센트로 CTA 영역 강조

## 5. A/B 테스트 설계

### 5-1. 첫 2주 테스트 구조 (4/21 ~ 5/4)

**구조: CBO(캠페인 예산 최적화) + 5개 광고 세트 동시 집행**

```
캠페인: [전환] 프레스코21_자사몰_꽃공예키트
  일예산: 12,000원 (CBO)
  |
  +-- 광고 세트 A: 입문키트 (이미지 2개 + 캐러셀 1개)
  +-- 광고 세트 B: 갤러리 (캐러셀 2개)
  +-- 광고 세트 C: 과정영상 (릴스 2개)
  +-- 광고 세트 D: 신뢰 (이미지 1개 + 영상 1개)
  +-- 광고 세트 E: 시즌-어버이날 (이미지 1개 + 릴스 1개)
```

### 5-2. 1주차 (4/21 ~ 4/27): 학습 기간

| 일정 | 행동 |
|------|------|
| Day 1~3 | 모든 세트 동시 시작. Meta 머신러닝 학습 기간이므로 수정 금지 |
| Day 4 | 중간 점검: 노출수, CTR, CPC 확인. 아직 수정 금지 |
| Day 7 | 1주차 결산. 아래 기준으로 1차 판단 |

**1주차 종료 시 판단 기준**:

| 지표 | PASS | WATCH | FAIL |
|------|------|-------|------|
| CTR | 1.5%+ | 1.0~1.5% | 1.0% 미만 |
| CPC | 300원 이하 | 300~500원 | 500원 초과 |
| 전환 | 1건+ | 0건 (노출 1,000+ 기준) | 0건 (노출 3,000+) |

### 5-3. 2주차 (4/28 ~ 5/4): 최적화 기간

| 상황 | 조치 |
|------|------|
| **FAIL 세트 있음** | 해당 세트 OFF → 예산이 PASS 세트로 자동 재배분 |
| **WATCH 세트** | 카피 변경 또는 소재 교체 (한 번에 하나만 변경) |
| **PASS 세트** | 유지. 소재 하나 추가하여 소재 내 A/B 테스트 |
| **전체 FAIL** | 타겟 재검토 (연령대 확장 25-60 또는 관심사 추가) |

### 5-4. 최종 승패 판단 기준

**2주 종합 ROAS 500% 달성 여부로 판단**:

| 판단 | 조건 | 다음 행동 |
|------|------|---------|
| **성공** | ROAS 500%+ | 일예산 15,000~20,000원 스케일업 |
| **부분 성공** | ROAS 300~500% | 전환 세트만 남기고 카피/소재 최적화 |
| **실패** | ROAS 300% 미만 | 타겟/소재/랜딩 전면 재검토. CFO 보고 후 예산 축소 또는 일시 중단 |

### 5-5. 테스트 후 최적화 방향

**Phase 1 (2주 테스트 후)**:
- 승리 세트 확정 → 해당 세트에 예산 70% 집중
- 패배 세트 OFF 또는 소재 전면 교체

**Phase 2 (3~4주차)**:
- 승리 세트 내 소재 A/B 테스트 (같은 컨셉, 다른 이미지/카피)
- 유사 타겟 (Lookalike) 1% 테스트 (Purchase 기반)

**Phase 3 (5주차~)**:
- 유사 타겟 비중 확대
- 리타겟팅 세트 추가 (사이트 방문 7일 + 장바구니 미구매)
- DPA(Dynamic Product Ads) 카탈로그 광고 검토

## 부록: UTM 파라미터 규칙

| 파라미터 | 값 | 설명 |
|---------|------|------|
| `utm_source` | `meta` | 트래픽 소스 |
| `utm_medium` | `cpc` | 유료 광고 |
| `utm_campaign` | `conversion_craft_kit` / `conversion_mothers_day` / `conversion_teachers_day` | 캠페인명 |
| `utm_content` | `set_a_beginner` / `set_b_gallery` / `set_c_process` / `set_d_trust` / `set_e_carnation` / `set_e_teacher` | 소재 세트 |

GA4(G-CWQYWF1580)에서 트래픽 획득 > 세션 소스/매체 보고서로 Meta 광고 유입과 전환을 추적한다.
