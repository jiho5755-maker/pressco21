## 작업 1: 잠긴 캠페인 9개 삭제

**예상 시간: 5분**

> 2016~2017년에 만들어진 오래된 캠페인들입니다. 이미 잠긴 상태라 광고가 나가지 않지만, 관리 화면을 지저분하게 만들어 삭제합니다.

### 삭제 대상 목록

1. 프레스코21(스마트스토어 파워링크) - 2017년
2. 프레스코21(에폭시레진)
3. 프레스코21네이버쇼핑 - 2017년
4. (구)프레스코21 - 2017년
5. #. 프레스코21 압화 TEST
6. 01. 에폭시레진(X)
7. -#에폭시레진 2024.11.26
8. 포에버러브 - 2016년
9. 프레스코21 (cmp-m002) - 2016년

### 진행 방법

1. 캠페인 목록에서 위 캠페인명을 찾아 **왼쪽 체크박스 클릭**
2. 9개 모두 체크 후 목록 상단 **"삭제"** 버튼 클릭
3. 확인 팝업에서 **"삭제"** 클릭

### 주의사항

- 잠긴 캠페인은 삭제가 안 될 수도 있습니다. 그럴 경우 해당 캠페인만 건너뛰고 가능한 것부터 삭제하세요.
- 삭제 후 복구 불가 — 하지만 어차피 운영 중인 캠페인이 아니니 걱정 안 하셔도 됩니다.

## 작업 2: 컨탠츠지면 캠페인 일시정지

**예상 시간: 1분**

> 30일 동안 클릭이 0건인 캠페인입니다. 광고비만 낭비할 수 있으니 일시정지합니다.

### 진행 방법

1. 캠페인 목록에서 **"컨탠츠지면"** 찾기 (SHOPPING 타입)
2. 캠페인명 왼쪽 **상태 토글**(초록색 원 버튼)을 클릭
3. 팝업이 뜨면 **"일시정지"** 확인

### 확인 방법

- 토글이 회색으로 바뀌고 상태가 "일시정지"로 표시되면 완료

## 작업 3: 벌크 캠페인 일예산 변경 (3,000원 → 5,000원)

**예상 시간: 2분**

> 쇼핑 벌크 캠페인의 하루 광고 예산을 높여 노출 기회를 늘립니다.

### 진행 방법

1. 캠페인 목록에서 **"벌크"** 찾기 (SHOPPING 타입)
2. 캠페인명을 **클릭**해서 상세 화면 진입
3. 우측 상단 또는 캠페인 정보 옆 **"수정"** 버튼 클릭
4. **"일예산"** 항목에서 3,000 → **5,000** 으로 수정
5. **"저장"** 클릭

### 확인 방법

- 수정 후 일예산 항목에 "5,000원"이 표시되면 완료

## 작업 4: toup.net 추적 URL 제거

**예상 시간: 5분**

> 기존 대행사(toup.net)의 추적 URL이 현재 활성 캠페인에 남아 있습니다. 대행 계약이 끝났으므로 제거합니다.

**대상 캠페인: "-# 프레스코21 파워링크"**

### 진행 방법

1. 캠페인 목록에서 **"-# 프레스코21 파워링크"** 클릭
2. 좌측 탭에서 **"광고그룹"** 클릭
3. 광고그룹 목록 상단 **"전체 선택"** 체크박스 클릭
4. **"수정"** → **"추적 URL 수정"** 클릭
5. 추적 URL 입력창에서 toup.net이 포함된 URL 전체를 **지우기** (빈 칸으로 만들기)
6. **"저장"** 클릭

### 주의사항

- 추적 URL은 비워도 광고 노출에 영향 없습니다.
- URL을 지우지 않고 다른 URL로 바꾸면 toup.net 쪽에 클릭 데이터가 계속 전송됩니다.
- 광고그룹별로 개별 추적 URL이 설정되어 있을 수 있습니다. 각 그룹에 들어가서 광고 소재 > 추적 URL 항목도 확인하세요.

### 개별 광고 소재 확인 경로

캠페인 → 광고그룹 → 광고 탭 → 광고 소재 클릭 → "URL 관리" 또는 "추적 URL" 항목 확인

## 작업 5: 압화 메인 캠페인 입찰가 하향

**예상 시간: 10분**

> 입찰가가 너무 높아 광고비 대비 효율이 낮습니다. 아래 4개 광고그룹의 입찰가를 낮춥니다.

**대상 캠페인: "-압화 메인"**

### 중요: 반드시 이 순서로 진행하세요!

> 네이버SA는 "컨텐츠 매체 입찰가 <= 기본 입찰가" 규칙이 있습니다.
> 컨텐츠 매체 입찰가를 먼저 낮추지 않으면 기본 입찰가 저장이 실패합니다.

**각 광고그룹마다 아래 순서 반복:**
1단계: 컨텐츠 매체 입찰가 먼저 낮추기
2단계: 기본 입찰가 낮추기

### 변경 대상 및 목표값

| 광고그룹명 | 현재 기본 입찰가 | 변경 목표 | 현재 컨텐츠 입찰가 | 처리 순서 |
|-----------|--------------|---------|-----------------|---------|
| PC_시간별 순위확인 | 800원 | 100원 | 확인 필요 | 컨텐츠 먼저 낮추기 |
| 전체시간 노출 | 870원 | 150원 | 확인 필요 | 컨텐츠 먼저 낮추기 |
| 00. 종합팩(압화)_PC/MO | 800원 | 100원 | 확인 필요 | 컨텐츠 먼저 낮추기 |
| 01. 세부(압화)_PC/MO | 800원 | 100원 | 확인 필요 | 컨텐츠 먼저 낮추기 |

### 진행 방법 (광고그룹 1개 기준, 4번 반복)

#### 1단계: 컨텐츠 매체 입찰가 먼저 낮추기

1. 캠페인 목록에서 **"-압화 메인"** 클릭
2. **"광고그룹"** 탭 클릭
3. 대상 광고그룹명 클릭 (예: "PC_시간별 순위확인")
4. **"기본 설정"** 또는 **"입찰가"** 탭 클릭
5. **"컨텐츠 매체 입찰가"** 항목 찾기
6. 현재 값을 **100원 이하**(예: 70원)로 수정
7. **"저장"** 클릭

#### 2단계: 기본 입찰가 낮추기

8. 같은 화면에서 **"기본 입찰가"** 항목 수정
9. 목표값(100원 또는 150원)으로 입력
10. **"저장"** 클릭

### 주의사항

- 저장 시 "컨텐츠 입찰가가 기본 입찰가보다 높습니다" 오류가 뜨면 → 컨텐츠 입찰가를 먼저 낮추지 않은 것입니다. 1단계부터 다시 시작하세요.
- 입찰가를 너무 낮게 설정하면 광고 노출 순위가 밀릴 수 있습니다. 100~150원은 최소 노출이 가능한 범위입니다.
