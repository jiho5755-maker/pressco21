# YouTube 영상 제작 가이드 — 상품 자동 연동 최적화

> 이 가이드는 PRESSCO21 유튜브 채널 영상 편집자/업로더를 위한 매뉴얼입니다.
> 영상 업로드 시 아래 규칙을 따르면 **foreverlove.co.kr 메인페이지에 관련 재료가 자동으로 표시**됩니다.

---

## 자동 연동 작동 원리

메인페이지 YouTube 섹션("영상으로 배우고, 바로 만들어보세요")은 3단계 자동 매칭 시스템으로 운영됩니다.

| 단계 | 방식 | 정확도 | 헤더 문구 |
|------|------|--------|-----------|
| 1단계 | 영상 설명·댓글의 상품 링크 직접 감지 | **높음** | "이 영상에 사용된 재료" |
| 2단계 | 제목·설명·태그 키워드 자동 매칭 | 중간 | "추천 재료" |
| 3단계 | 연동 상품 없을 때 안내 문구 표시 | - | "인기 재료" (준비 중) |

**최선의 결과를 위해 1단계 링크 삽입을 권장합니다.**

---

## 필수 체크리스트 (업로드 전)

### ✅ 제목 규칙

- **핵심 재료명 1~2개 포함** (영상에 실제 사용한 재료)
- 좋은 예: `압화 레진 트레이 만들기 | 초보도 쉽게!`
- 나쁜 예: `클래스 영상 #12` (재료명 없음)
- 키워드 참조: [카테고리별 키워드표](#카테고리별-키워드-참조표) 참고

### ✅ 영상 설명란 규칙

#### 상단 (첫 3줄 — 접힘 미리보기에 표시됨)

```
이번 영상에서 사용한 재료:
▷ [재료명1] — [구매 링크]
▷ [재료명2] — [구매 링크]
```

#### 하단 (필수: 상품 링크)

```
━━━━━ 이 영상 사용 재료 구매 ━━━━━
▷ 상품명1 → https://www.foreverlove.co.kr/shop/shopdetail.html?branduid=XXXXX
▷ 상품명2 → https://www.foreverlove.co.kr/shop/shopdetail.html?branduid=XXXXX

※ 링크 형식: https://www.foreverlove.co.kr/shop/shopdetail.html?branduid=상품번호
※ 상품번호(branduid)는 관리자에게 문의
```

**링크 삽입이 가장 중요합니다** — 링크가 있으면 자동으로 "이 영상에 사용된 재료" 섹션에 표시됩니다.

### ✅ 태그 규칙

재료명·카테고리명 태그 필수 (매칭 점수에 영향):

```
#압화 #레진 #캔들 #석고 #비즈 (해당 카테고리)
#핸드메이드 #DIY #공예 #취미
#foreverlove #포에버러브
```

## 댓글 활용 (보조 방법)

영상 게시 후 **채널 관리자 계정으로 댓글**에 상품 링크를 달아두면, 6시간마다 자동으로 스캔되어 연동됩니다.

```
이 영상에서 사용한 재료는 여기서 구매하실 수 있어요 😊
👉 https://www.foreverlove.co.kr/shop/shopdetail.html?branduid=XXXXX
```

## 자주 묻는 질문

**Q. 영상 업로드 후 얼마나 지나야 메인페이지에 표시되나요?**
A. 최대 30분 이내 (WF-YT-SYNC가 30분 주기로 자동 동기화).

**Q. 상품 링크를 설명란에 넣었는데 표시가 안 돼요.**
A. `https://www.foreverlove.co.kr/shop/shopdetail.html?branduid=숫자` 형식인지 확인해 주세요. 단축URL이나 bit.ly 형식은 감지되지 않습니다.

**Q. 여러 개 상품 링크를 넣으면 다 표시되나요?**
A. 최대 4개까지 표시됩니다. 가장 중요한 재료 4개를 넣어주세요.

**Q. 영상 제목에 재료 이름이 없으면 어떻게 되나요?**
A. 상품 링크가 없을 때는 키워드 자동 매칭(2단계)으로 관련 재료를 찾아 표시합니다. 하지만 1단계(링크 직접 삽입)보다 정확도가 낮습니다.

## 시스템 아키텍처 (기술 참고용)

```
업로드된 영상
    │
    ▼ (30분 이내)
WF-YT-SYNC (n8n 스케줄 워크플로우)
    ├─ YouTube RSS 폴링 → 신규 영상 감지
    ├─ YouTube API: 제목/설명/태그 수집
    ├─ 설명란 링크 파싱 → tbl_VideoProducts (source: description)
    └─ 키워드 자동매칭 → tbl_VideoProducts (source: keyword)

WF-YT-COMMENTS (6시간 주기)
    └─ 댓글 링크 파싱 → tbl_VideoProducts (source: comment)

메인페이지 방문자
    └─ POST /webhook/youtube-api → WF-YT → NocoDB 조회 → 영상+상품 응답
```
