## 확정된 원칙

1. **무료 전용**: Oracle Cloud Free Tier만 사용. 유료 클라우드 절대 안 감. 부족하면 새 무료 계정으로 서버 추가.
2. **서버 명명**: 본진(oracle, 운영), 플로라(openclaw, AI+테스트), 금고(minipc, 백업+파일)
3. **계정 분리**: 본진/플로라가 **별도 Oracle 계정** → 각각 4코어/24GB/200GB 한도. 현재 각 2코어/12GB/45GB만 사용 중 (155GB 디스크 미사용).
4. **역할 분리**: 본진=돈 버는 것만, 플로라=AI+실험실, 금고=데이터 보호

### C. 추가 분석 요청

이전 세션에서 3가지 관점으로 진단했는데, 새 세션에서 **더 깊이 파고들 질문들**:

3. **하네스 엔지니어**: L5(자율 협업)로 가려면 뭐가 필요한가? Flora가 n8n 모니터링→이상 감지→Claude Code에 디버깅 요청하는 자율 루프가 현실적인가?
