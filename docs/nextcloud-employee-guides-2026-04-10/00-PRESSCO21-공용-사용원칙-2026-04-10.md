# PRESSCO21 공용 저장소 사용 원칙

작성일: 2026-04-10

## 무엇을 쓰는가

- 평소 브랜드, 디자인, 사진, 내부문서는 Nextcloud에서 업로드하고 다운로드한다.
- 대용량 영상 `raw`, `project`, `export`는 장지호와 장다경만 Syncthing으로 mini PC에 동기화한다.
- Syncthing으로 들어간 영상은 Nextcloud의 `영상자료`에서 다시 내려받아 확인할 수 있다.

## 접속 주소

- 공통 접속 주소: https://backup.pressco21.com
- 사무실이든 외부든 같은 주소를 사용한다.
- 브라우저는 Chrome 또는 Edge 사용을 권장한다.

## 로그인 방법

1. 브라우저에서 `https://backup.pressco21.com` 접속
2. 본인 아이디와 비밀번호 입력
3. 로그인 후 본인 권한에 맞는 폴더 확인

## 폴더 사용 원칙

- `00_INBOX`: 어디에 넣을지 애매한 자료를 넣는 접수함
- `브랜드`: 로고, 브랜드 가이드, 템플릿, 공용 에셋
- `디자인`: 디자인 결과물, 시안, 템플릿, 공유 이미지
- `사진`: 촬영본, 선별본, 썸네일용 이미지
- `내부문서`: 공용 운영 문서, 소개자료, 업무 참고자료
- `영상자료`: Syncthing으로 들어온 영상 자료 열람/다운로드용
- `업무가이드`: 직원이 읽는 기준 문서, 표준답변, 체크리스트
- `관리자문서함`: 매출자료, 계약서, 기밀문서. 관리자만 사용

## 공통 운영 규칙

- 정리할 때는 파일 성격을 알 수 있는 자료는 맞는 폴더에 바로 넣는다.
- `브랜드`, `디자인`, `사진`, `내부문서`, `관리자문서함` 루트에는 직접 업로드하지 않는다.
- 반드시 한 단계 들어가서 맞는 하위 폴더에 업로드한다.
- 어디에 넣을지 애매한 자료는 `00_INBOX/직원업로드`에 넣는다.
- 대용량 영상은 Nextcloud에 직접 올리지 않는다.
- 파일 이름은 가능하면 `날짜_무슨자료인지_상태_v번호` 형식으로 저장한다.
- 한 파일의 최신본이 무엇인지 헷갈릴 때는 새 폴더를 만들지 말고 기존 위치에서 버전명을 붙여 관리한다.
- 공용 자료는 삭제보다 이동과 이름 정리를 우선한다.
- 계약, 세무, 매출, 인사, 고객 개인정보, 계정 정보는 공용 폴더에 넣지 않고 관리자에게 전달한다.
- 관리자문서함은 관리자 계정에서만 보이는 것이 정상이다.
