## 1. 목적

오픈마켓 등록과 상세페이지 제작 전에 `섹션별 멘트 + 상품 리스트 + 채널별 상품명`이 정리된 기획안을 먼저 만든다.

이 문서의 목표는 다음 3가지다.

1. 웹디자이너가 작업 시작 전에 필요한 정보를 한 번에 받게 한다.
2. 자사몰 원본과 오픈마켓 복사등록 흐름을 같은 기준으로 맞춘다.
3. 이후 AI 초안 생성으로 확장할 수 있는 최소 입력 구조를 만든다.

---

## 2. 현재 문제

현재 흐름은 `사진/카피 수집 -> 상세 제작 -> 자사몰 등록 -> 오픈마켓 복사등록` 이지만, 중간의 기획안 단계가 느슨하다.

회의에서 확인된 병목은 아래와 같다.

- 사전 기획안이 없어 디자이너가 섹션 구성부터 다시 잡아야 한다.
- 오픈마켓 등록 시 상품명, 가격, 이미지 수정 기준이 담당자 머릿속에만 있다.
- `단일상품`과 `모음전/기획전`이 다른데도 같은 방식으로 요청이 들어온다.
- 11번가 부가이미지 규격 등 채널 차이가 작업 후반에 다시 반영된다.

---

## 3. 운영 원칙

1. 자사몰 상세가 마스터 원본이다.
2. 기획안은 `디자인 전 단계`에서 확정한다.
3. 오픈마켓은 상세 이미지보다 `상품명/키워드/가격/썸네일 우선순위`를 먼저 맞춘다.
4. 브랜드 톤은 유지하되, 검색 키워드 구조는 채널별로 달리 쓴다.
5. 최종 결정은 사람 담당자가 한다. AI는 초안까지만 맡는다.

---

## 4. 산출물 타입

### A. 단일상품 기획안

대상:
- 압화 재료
- 레진/오일
- 도구/부자재
- 키트 단품

필수 결과물:
- 자사몰용 상품명 1안
- 네이버/쿠팡/11번가용 상품명 각 1안
- 상세페이지 4~6섹션 구성
- 핵심 특징 3가지
- 사용법/주의사항
- 촬영/보정 필요 이미지 목록

### B. 모음전/기획전 기획안

대상:
- 시즌 추천 모음전
- 초보자 입문 세트
- 카테고리 묶음전

필수 결과물:
- 기획전 제목 2안
- 포함 상품 리스트
- 상품 선정 이유
- 상단 도입 카피
- 섹션별 진열 순서
- 교차판매 연결 포인트

## 5. 기본 입력 항목

기획안 작성 전에 아래 항목이 먼저 모여야 한다.

| 항목 | 설명 | 담당 |
|------|------|------|
| 상품코드/SKU | 내부 식별값 | 경영기획 |
| 카테고리 | 압화/레진/도구/키트 등 | 경영기획 |
| 판매 채널 | 자사몰, 네이버, 쿠팡, 11번가 | 경영기획 |
| 핵심 특징 3개 | 왜 사야 하는지 | 대표/경영기획 |
| 필수 스펙 | 용량, 수량, 사이즈, 재질 | 경영기획 |
| 사용 대상 | 초보/취미/강사/B2B | 경영기획 |
| 주의사항 | 안전, 보관, 사용 제한 | 대표 |
| 보유 이미지 현황 | 대표컷/클로즈업/활용컷 유무 | 디자이너 확인 |
| 가격 정책 | 자사몰 기준가, 오픈마켓 반영가 | 대표/경영기획 |
| 채널 특이사항 | 11번가 부가이미지, 쿠팡 키워드 등 | 등록 담당 |

입력 항목이 80% 이상 채워지지 않으면 디자인 작업으로 넘기지 않는다.

## 8. 주간 운영 루틴

### 월요일

- 이번 주 등록/개편 대상 상품 5~10개 선정
- 단일상품/모음전 유형 분류
- 누락 정보 체크

### 화요일

- 경영기획이 기획안 초안 작성
- 채널별 상품명 1차 작성
- 이미지 추가 촬영 필요 여부 표시

### 수요일

- 대표가 특징, 스펙, 가격 정책 확인
- 디자이너가 이미지 보유 현황과 제작 난이도 확인

### 목요일

- 상세페이지 제작
- 자사몰용 카피 최종 반영

### 금요일

- 자사몰 등록
- 오픈마켓 복사등록
- 채널별 상품명/가격/썸네일 반영

---

## 9. 4주 실행 계획

### 1주차

- 단일상품 템플릿 확정
- 우선순위 SKU 10개 선정
- 압화/레진 각 3개씩 파일럿 작성

### 2주차

- 모음전 템플릿 확정
- 네이버/쿠팡/11번가 제목 규칙 보완
- 11번가 부가이미지 체크포인트 문서화

### 3주차

- 카테고리별 문장 라이브러리 구축
- `압화`, `레진`, `키트`, `도구` 공통 문구 정리
- 자주 쓰는 주의사항/보관법 문구 표준화

### 4주차

- 스프레드시트 입력폼으로 전환
- AI 초안 생성 실험 시작
- 기획안 작성 시간과 수정 횟수 측정

---

## 10. 우선순위 상품군

1차 파일럿은 아래 순서로 진행한다.

1. 압화 재료
2. UV레진/레진 보조재
3. 하바리움 오일
4. 초보자 키트
5. 도구/부자재

이유:
- 검색 수요가 명확하다.
- 초보자 설명 니즈가 크다.
- 오픈마켓 상품명 최적화 효과를 바로 보기 쉽다.

## 11. KPI

초기 4주 동안 아래 지표를 본다.

| 지표 | 현재 추정 | 1차 목표 |
|------|----------|----------|
| 기획안 작성 시간 | 불규칙 | 상품당 30분 이내 |
| 상세 제작 전 수정 요청 | 많음 | 1회 이내 |
| 디자이너 정보 재문의 | 잦음 | 50% 이상 감소 |
| 오픈마켓 복사등록 리드타임 | 담당자 의존 | 주간 일괄 등록 가능 |

## 12. 다음 단계

문서만으로 끝내지 말고 아래 순서로 이어간다.

1. 파일럿 SKU 10개를 실제 템플릿으로 채운다.
2. 자주 쓰는 문장 라이브러리를 별도 파일로 분리한다.
3. 이후 스프레드시트 입력 -> AI 초안 생성 흐름으로 자동화한다.

이 문서는 `운영 계획` 문서이며, 실제 카피 룰은 기존 `openmarket-guide.md` 와 `product-listing-sop.md` 를 우선 기준으로 삼는다.
