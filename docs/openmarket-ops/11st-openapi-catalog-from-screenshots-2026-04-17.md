## 현재 캡처만으로 부족한 점

- 상품 신규 등록 API가 캡처 목록에 명확히 보이지 않는다.
- 상품 이미지/상세설명/고시정보 등록 가능 여부는 각 상세 URL 확인이 필요하다.
- 각 API의 HTTP method, endpoint path, request body, rate limit, 필수 권한명은 상세 문서 URL을 받아야 확정할 수 있다.
