## 1. 교육 아카데미 개요

### 1-1. 목적

PRESSCO21 파트너 교육 아카데미는 파트너(강사/공방)가 플랫폼에 합류하기 전에 반드시 이수하는 온보딩 교육입니다. 교육을 통해 브랜드 가치를 함께 공유하고, 안전한 수업 환경을 조성하며, 플랫폼을 원활히 활용할 수 있도록 돕습니다.

### 1-2. 교육 구조

```
파트너 승인 → 교육 안내 이메일 수신
                    |
                    v
        [모듈 1] 브랜드 스토리 & 가이드라인 (YouTube, 약 7분)
                    |
                    v
        [모듈 2] 수업 환경 & 안전 수칙 (YouTube, 약 6분)
                    |
                    v
        [모듈 3] 플랫폼 사용 완전 정복 (YouTube, 약 9분)
                    |
                    v
        [확인 퀴즈] 15문항, 70점(10.5/15) 이상 합격 (Google Forms)
                    |
              ------+------
              |            |
           합격          불합격
              |            |
     인증서 이메일     재응시 안내
     SILVER 활성화     이메일 발송
```

### 1-3. 핵심 수치

| 항목 | 값 |
|------|------|
| 필수 모듈 | 3개 |
| 총 영상 시간 | 약 22분 |
| 퀴즈 문항 | 15문항 (모듈당 5문항) |
| 합격 기준 | 70% 이상 (15문항 중 11문항 정답) |
| 재응시 | 횟수 제한 없음, 즉시 가능 |
| 교육 유효기간 | 영구 (1회 이수 시 유지) |

### 1-4. 교육 안내 페이지 카피

**파트너 대시보드 교육 섹션 헤더:**

> **PRESSCO21 파트너 아카데미**
> 함께 성장하기 위한 첫 번째 단계

**교육 안내 메시지:**

> 파트너 활동을 시작하기 전에 3개의 교육 영상을 시청하고 확인 퀴즈를 완료해주세요.
> PRESSCO21의 브랜드 가치, 안전한 수업 환경, 플랫폼 사용법을 안내해드립니다.
> 영상은 총 22분 정도이며, 퀴즈는 15문항이에요. 70점 이상이면 합격입니다.

**교육 미이수 상태 표시:**

> 교육을 완료하면 SILVER 파트너로 활성화되어 첫 클래스를 등록할 수 있어요.
> 아래 영상을 순서대로 시청한 후 퀴즈를 풀어주세요.

**교육 완료 상태 표시:**

> 교육을 모두 이수하셨습니다! SILVER 파트너로 활성화되었어요.
> 이제 첫 클래스를 등록해볼까요?

## 2. 필수 교육 3모듈 커리큘럼 상세

### 모듈 1: PRESSCO21 브랜드 스토리 & 가이드라인

| 항목 | 내용 |
|------|------|
| **영상 길이** | 약 7분 |
| **형식** | YouTube 비공개(unlisted) 영상 |
| **학습 목표** | PRESSCO21의 역사와 가치를 이해하고, 파트너로서 지켜야 할 브랜드 약속을 숙지한다 |

#### 커리큘럼 구성

| 순서 | 주제 | 시간 | 내용 상세 |
|------|------|------|----------|
| 1-1 | 오프닝 | 0:00~0:30 | 인사 + "왜 이 교육이 중요한가요?" 동기 부여 |
| 1-2 | 브랜드 역사 | 0:30~2:00 | 30년간의 여정 -- 압화 연구, 투명 식물표본 특허, 방송/전시/국제 교류 이야기 |
| 1-3 | 브랜드 가치 | 2:00~3:30 | "Forever and ever and Blooming"의 의미, 꽃의 아름다움을 영원히 보존하는 철학 |
| 1-4 | 품질 기준 | 3:30~4:30 | PRESSCO21이 추구하는 수업 품질 -- 재료의 신선함, 작품의 완성도, 수강생 경험 |
| 1-5 | 파트너 브랜드 약속 | 4:30~6:00 | 로고 사용 규칙, 홍보 시 표현 가이드, 금지 사항 (과장 광고, 경쟁사 비교, 허위 정보) |
| 1-6 | 성공 파트너 사례 | 6:00~6:40 | 실제 파트너의 수업 운영 사례 (인용문 또는 인터뷰 발췌) |
| 1-7 | 마무리 | 6:40~7:00 | 핵심 요약 3가지 + 다음 모듈 안내 |

#### 핵심 포인트 3가지

1. **30년 전통**: PRESSCO21은 단순한 쇼핑몰이 아닌, 꽃 공예의 발전을 위해 연구하고 교류해온 전문 브랜드입니다.
2. **함께하는 여정**: 파트너는 PRESSCO21과 동반자 관계입니다. 함께 성장하며 더 많은 분들에게 꽃의 아름다움을 전합니다.
3. **일관된 품질**: 어떤 파트너의 수업이든, PRESSCO21 이름으로 진행되는 수업은 동일한 품질 기대치를 충족해야 합니다.

#### 파트너 브랜드 약속 상세

| 항목 | DO | DON'T |
|------|------|-------|
| 소개 | "PRESSCO21 파트너 [공방명]" | "PRESSCO21 공식 인증 1급 강사" |
| 홍보 | "PRESSCO21과 함께하는 클래스" | "PRESSCO21 본사 직영 수업" |
| 재료 | "PRESSCO21 재료로 진행합니다" | "PRESSCO21 독점 재료, 여기서만 가능" |
| 가격 | 수강료를 투명하게 안내 | "원래 10만원인데 특별히 5만원" (허위 할인) |
| 비교 | 자신의 수업 장점을 강조 | "다른 공방보다 훨씬 낫습니다" |
| SNS | 작품 사진에 @pressco21 태그 | PRESSCO21 로고를 무단 변형/합성 |

### 모듈 2: 수업 환경 & 안전 수칙

| 항목 | 내용 |
|------|------|
| **영상 길이** | 약 6분 |
| **형식** | YouTube 비공개(unlisted) 영상 |
| **학습 목표** | 안전하고 쾌적한 수업 환경을 조성하는 방법을 숙지하고, 돌발 상황에 대처할 수 있다 |

#### 커리큘럼 구성

| 순서 | 주제 | 시간 | 내용 상세 |
|------|------|------|----------|
| 2-1 | 오프닝 | 0:00~0:20 | "안전한 수업이 좋은 수업의 시작이에요" |
| 2-2 | 작업 공간 요건 | 0:20~1:30 | 조명(자연광 권장), 환기(밀폐 공간 주의), 테이블 크기(1인당 60cm 이상), 수도 시설 접근성 |
| 2-3 | 재료 취급 주의 | 1:30~3:00 | 건조제(실리카겔 분진 주의), 레진/접착제(환기 필수, 피부 접촉 주의), UV 램프(눈 보호), 도료(유기용제 환기) |
| 2-4 | 알레르기 대응 | 3:00~4:00 | 꽃가루/라텍스/화학물질 알레르기 사전 확인 방법, 수강 신청서에 알레르기 정보 항목 추가 권장 |
| 2-5 | 응급 대처 | 4:00~5:00 | 화상(레진 발열), 절상(가위/칼), 알레르기 반응(두드러기, 호흡 곤란) 시 즉각 대처 + 119 연락 기준 |
| 2-6 | 어린이 수업 주의 | 5:00~5:40 | 보호자 동반 기준(만 12세 미만 권장), 안전 도구 사용(둥근 가위 등), 소재 제한(레진/유기용제 제외) |
| 2-7 | 마무리 | 5:40~6:00 | 핵심 요약 3가지 + 안전 체크리스트 소개 |

#### 핵심 포인트 3가지

1. **환기가 가장 중요합니다**: 레진, 접착제, 건조제를 다루는 공예 수업에서 환기는 선택이 아닌 필수입니다.
2. **사전 확인이 사고를 예방합니다**: 수강생의 알레르기 정보를 수업 전에 반드시 확인해주세요.
3. **어린이 수업은 별도 기준이 필요합니다**: 보호자 동반, 안전 도구, 재료 제한을 꼭 지켜주세요.

#### 안전 체크리스트 (파트너 자가 점검용)

- [ ] 작업 공간 환기 가능 여부 확인
- [ ] 구급 키트 비치 (밴드, 소독약, 화상 연고, 라텍스 프리 장갑)
- [ ] 수강생 알레르기/건강 정보 수집 안내
- [ ] 어린이 참여 시 보호자 동반 안내
- [ ] 화학 재료 사용 시 보안경/장갑 비치
- [ ] 가위/칼 등 날카로운 도구 안전 보관
- [ ] 작업대 1인당 60cm 이상 공간 확보
- [ ] 비상 연락처(119) 및 가장 가까운 병원 위치 숙지

3. **궁금한 점은 언제든 물어보세요**: 대시보드에서 대부분의 정보를 확인할 수 있고, 그 외에는 파트너 전용 문의 채널로 연락하시면 돼요.

```
자연의 아름다움을 보존하고 전하는 방법을 연구해온 거죠.

파트너 프로그램을 통해 더 많은 분들에게
꽃 공예의 즐거움을 전하고 있습니다.


--- 1-3. 브랜드 가치 (2:00~3:30) ---

[화면: "Forever and ever and Blooming" 타이포그래피 + 꽃이 피는 타임랩스]

PRESSCO21의 슬로건은 "Forever and ever and Blooming"이에요.
"영원히, 그리고 영원히, 꽃은 피어납니다."

PRESSCO21 파트너의 수업은 어떤 기준을 지켜야 할까요?

foreverlove.co.kr에서 재료를 구매하시면

수강생이 집에 가져갈 작품은 "내가 만들었다"는 자부심을 느낄 수 있어야 해요.
적절한 난이도 설정과 꼼꼼한 지도가 그 열쇠입니다.

셋, 수강생 경험.
수업 내용뿐 아니라, 예약부터 수업 후까지의 전체 경험이 중요해요.
친절한 응대, 깨끗한 공간, 즐거운 분위기.
이 세 가지가 수강생을 다시 오게 만들어요.


--- 1-5. 파트너 브랜드 약속 (4:30~6:00) ---

이제 파트너로서 지켜주실 브랜드 약속에 대해 안내해드릴게요.

SNS에 작품 사진을 올리실 때는 @pressco21을 태그해주시면
서로에게 좋은 홍보가 돼요.
```

### 모듈 2 스크립트: 수업 환경 & 안전 수칙

```
[영상 시간: 약 6분]
[톤: 신뢰감 있고 차분한 안내자]

--- 인트로 (0:00~0:20) ---

[화면: 깨끗하고 정돈된 공방 내부]

안녕하세요, 파트너 아카데미 두 번째 시간이에요.
오늘은 "안전한 수업"에 대해 이야기해볼게요.

좋은 수업의 시작은 안전한 환경입니다.
재미있으면서도 안전한 수업을 만드는 방법, 함께 알아볼까요?


--- 2-2. 작업 공간 요건 (0:20~1:30) ---

[화면: 이상적인 공방 레이아웃 일러스트, 실제 공방 사진]

먼저, 수업 공간에 대해 알아볼게요.

조명은 자연광이 가장 좋아요.
꽃의 색감을 정확히 볼 수 있거든요.
자연광이 어렵다면 색온도 5000K 이상의 조명을 추천합니다.

환기가 가장 중요해요.
레진, 접착제, 건조제를 다루는 수업에서는
창문을 열거나 환풍기를 반드시 가동해주세요.
밀폐된 공간에서의 수업은 절대 피해야 합니다.

테이블은 1인당 가로 60센티미터 이상의 공간을 확보해주세요.
옆 사람과 팔이 부딪히면 작업에 집중하기 어렵거든요.

그리고 수도 시설 접근성.
재료가 손에 묻거나 흘렸을 때 바로 씻을 수 있어야 해요.
물티슈만으로는 부족한 경우가 있어요.


--- 2-3. 재료 취급 주의 (1:30~3:00) ---

[화면: 재료별 주의사항 아이콘 + 실제 재료 사진]

이제 재료별 주의사항을 짚어볼게요.

건조제, 특히 실리카겔.
건조 분말 형태이기 때문에 미세한 분진이 발생할 수 있어요.
직접 피부에 닿지 않도록 장갑을 착용하고,
가루가 날리지 않도록 조심히 다뤄주세요.

레진과 접착제.
레진은 경화 과정에서 열이 발생하고, 특유의 냄새가 나요.
반드시 환기가 되는 공간에서 사용하고,
피부에 직접 닿지 않도록 일회용 장갑을 준비해주세요.
접착제도 마찬가지예요.

UV 램프.
레진 경화에 사용하는 UV 램프는 직접 눈으로 바라보지 않도록 안내해주세요.
필요하다면 보안경을 비치해두는 것도 좋아요.

도료나 유기용제를 사용하는 수업이 있다면,
환기는 물론이고 마스크 착용도 권장합니다.


--- 2-4. 알레르기 대응 (3:00~4:00) ---

[화면: 수강 신청서 예시 화면, 알레르기 정보 항목 하이라이트]

꽃 공예 수업에서 알레르기는 놓치기 쉬운 부분이에요.

꽃가루 알레르기가 있는 수강생이 생화를 다루면
재채기, 눈 충혈, 심하면 호흡 곤란까지 올 수 있어요.
라텍스 알레르기가 있는 분은 라텍스 장갑 대신
니트릴 장갑을 준비해야 하고요.

그래서 수강 신청 시 알레르기 정보를 미리 확인하는 것이 중요합니다.
수업 안내에 "알레르기가 있으신 경우 미리 알려주세요"라는
한 줄만 추가해도 많은 상황을 예방할 수 있어요.


--- 2-5. 응급 대처 (4:00~5:00) ---

[화면: 구급 키트 내용물, 대처 순서 플로차트]

아무리 조심해도 예상치 못한 상황이 생길 수 있어요.
대처 방법을 미리 알아두면 당황하지 않아요.

레진 발열로 인한 화상.
레진을 한 번에 많이 붓거나 두껍게 경화하면 열이 많이 발생해요.
화상이 발생하면 즉시 흐르는 찬물에 10분 이상 식혀주세요.
심한 경우 병원 방문이 필요합니다.

가위나 칼에 의한 절상.
깨끗한 거즈로 상처 부위를 누르고, 출혈이 멈추면 소독 후 밴드를 붙여주세요.
출혈이 5분 이상 멈추지 않으면 병원에 가셔야 해요.

알레르기 반응.
두드러기, 부종, 호흡 곤란이 나타나면
원인 재료를 즉시 치우고 환기시킨 뒤,
증상이 심하면 119에 연락해주세요.

구급 키트를 항상 수업 공간에 비치해두세요.
밴드, 소독약, 화상 연고, 니트릴 장갑 정도면 충분해요.


--- 2-6. 어린이 수업 주의 (5:00~5:40) ---

[화면: 어린이 수업 장면, 안전 도구 예시]

어린이 수강생을 받으실 때는 추가 주의사항이 있어요.

만 12세 미만은 보호자 동반을 권장합니다.
가위는 둥근 끝 어린이용 가위를 사용하고,
레진이나 유기용제가 포함된 재료는 어린이 수업에서 사용하지 않아요.

대신 압화 스티커, 압화 책갈피, 압화 액자 등
접착제만으로 완성할 수 있는 프로그램이 안전하고 인기도 좋아요.

수업 안내에 "만 12세 미만은 보호자 동반이 필요합니다"라고
명시해두시면 사전에 정리가 돼요.


--- 2-7. 마무리 (5:40~6:00) ---

[화면: 핵심 3가지 자막 + 안전 체크리스트 다운로드 안내]

오늘의 핵심 세 가지를 정리합니다.

첫째, 환기가 가장 중요합니다. 밀폐 공간 수업은 절대 금지.
둘째, 수강생의 알레르기 정보를 사전에 확인해주세요.
셋째, 어린이 수업은 별도 기준으로 안전하게 운영해주세요.

안전 자가 점검 체크리스트를 파트너 가이드에서 다운로드하실 수 있어요.
수업 전에 한 번씩 점검해보시면 좋겠어요.

다음 영상에서는 플랫폼 사용법을 알아볼게요.
그럼 다음 영상에서 만나요!
```

### 모듈 3 스크립트: 플랫폼 사용 완전 정복

```
[영상 시간: 약 9분]
[톤: 친절하고 실용적인 안내자]

--- 인트로 (0:00~0:20) ---

[화면: 파트너 대시보드 화면 캡처]

안녕하세요, 파트너 아카데미 마지막 시간이에요!
오늘은 PRESSCO21 플랫폼 사용법을 처음부터 끝까지 알려드릴게요.

수업에만 집중하실 수 있도록, 나머지는 저희가 도울게요.
자, 하나씩 살펴볼까요?


--- 3-2. 강의 등록 (0:20~2:30) ---

[화면: Google Forms 강의 등록 화면 실제 캡처, 단계별 하이라이트]

첫 수업을 등록하는 방법을 알아볼게요.

강의 등록은 Google Forms를 통해 진행됩니다.
파트너 코드를 입력하고, 강의 정보를 작성하면 끝이에요.

카테고리 선택부터 볼게요.
압화, 보존화, 레진아트, 하바리움, 캔들, 기타 중에서 선택합니다.
여러 분야를 다루신다면 주요 분야를 선택해주세요.

수강료 설정.
재료비가 포함된 가격인지, 별도인지를 꼭 명시해주세요.
수강생이 결제 전에 총비용을 정확히 알 수 있어야 해요.

사진 업로드 팁을 하나 드릴게요.
자연광에서 촬영한 작품 사진이 가장 효과적이에요.
배경은 깔끔하게, 작품이 주인공이 되도록.
수업 과정 사진도 함께 올리시면 수강생이 수업을 미리 상상할 수 있어서 좋아요.

소개글 작성 요령.
"이 수업에서 무엇을 배우는지, 어떤 작품을 가져갈 수 있는지"를
구체적으로 써주세요.
"예쁜 수업입니다" 보다는
"생화 3송이를 직접 압화하여 나만의 에코백을 완성하는 수업입니다"가
훨씬 매력적이에요.


--- 3-3. 검수 과정 (2:30~3:30) ---

[화면: 검수 항목 체크리스트 그래픽]

강의 정보를 제출하시면 관리자가 1~2 영업일 내에 검수합니다.

확인하는 항목은 다섯 가지예요.
브랜드 적합성, 가격 적정성, 이미지 품질, 안전성 안내, 소개글 톤.

검수를 통과하면 메이크샵에 상품으로 등록되고,
클래스 목록 페이지에 자동으로 노출됩니다.

보완이 필요한 경우에는 수정 요청 이메일을 보내드려요.
"사진 밝기를 조금만 높여주세요" 같은 구체적인 피드백이니
부담 갖지 않으셔도 돼요.


--- 3-4. 예약 관리 (3:30~5:00) ---

[화면: 파트너 예약 알림 이메일 예시, 대시보드 예약 목록]
```

### 8-2. Google Forms 퀴즈 연결 설정

#### Forms 생성 단계

1. Google Forms 새 양식 생성
2. 제목: "PRESSCO21 파트너 아카데미 확인 퀴즈"
3. 설명: 섹션 4-2의 안내 카피 입력
4. **설정 > 퀴즈 만들기** 활성화
5. 점수 공개 시점: "제출 직후"
6. 오답 피드백: 각 문항에 섹션 4의 오답 피드백 입력

#### 필수 수집 항목

| 필드 | 형식 | 유효성 검사 |
|------|------|-----------|
| 파트너 코드 | 단답형 | 필수, "PC_" 로 시작하는 형식 안내 |
| 이메일 주소 | 이메일 | 필수, 이메일 형식 검증 |

#### GAS Trigger 연결

1. Google Forms의 스크립트 편집기 열기
2. `onFormSubmit` 트리거 함수 설정
3. 제출 시 GAS의 `handleEducationComplete` 엔드포인트 호출
4. 점수 확인 -> 합격/불합격 분기 -> 해당 이메일 발송

```
// 의사 코드 -- gas-backend-expert가 구현
FUNCTION onFormSubmit(event):
    partnerCode = event.values[0]   // 파트너 코드
    email = event.values[1]         // 이메일
    score = calculateScore(event)   // 퀴즈 점수 계산

    IF score >= 11:   // 합격 (15문항 중 11문항 = 약 73%)
        // 파트너 상세 시트 education_completed = 'Y'
        updatePartnerEducation(partnerCode, 'Y', score)
        // 합격 축하 + 인증서 이메일 발송
        sendAcademyPassEmail(email, partnerCode, score)
    ELSE:
        // 불합격 재응시 안내 이메일 발송
        sendAcademyFailEmail(email, partnerCode, score)
```
