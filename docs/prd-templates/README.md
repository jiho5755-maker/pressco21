# PRD 템플릿 시스템

## 사용법

**텔레그램/Claude Code에서 한줄 지시:**
```
기획안 상세페이지 레지너스 화이트 프레임 신상품
기획안 이벤트 봄맞이 압화 클래스 30% 할인
기획안 랜딩 파트너클래스 신규 강사 모집
기획안 SNS 인스타 릴스 레진아트 완성작 모음
```

**출력물 3종:**
1. PRD (마크다운) — 기획 의도, 구조, 카피 초안
2. 디자인 브리프 — 색상, 폰트, 레이아웃 규격
3. 레퍼런스 리스트 — 참고할 기존 페이지/경쟁사

## 템플릿 유형

| 유형 | 파일 | 담당 에이전트 | 용도 |
|------|------|-------------|------|
| 상품 상세페이지 | `product-detail.md` | makeshop-expert | 자사몰 상품 상세 |
| 이벤트/프로모션 | `event-promo.md` | content-brand-planner | 할인, 기획전 |
| 랜딩 페이지 | `landing-page.md` | makeshop-expert | 모집, 브랜드, 소개 |
| SNS 콘텐츠 | `sns-content.md` | content-brand-planner | 인스타/유튜브/블로그 |

## 브랜드 규격 (공통)
