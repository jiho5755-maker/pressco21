## PRD 섹션

### 1. 이벤트 개요

- **이벤트명**: {{event_name}}
- **기간**: {{start_date}} ~ {{end_date}}
- **목표**: (매출 목표, 신규 고객 수, 재방문율 등)
- **타겟**: (기존 고객 리텐션 / 신규 유입 / 특정 등급)
- **예산**: (할인 원가, 쿠폰 비용, 광고비)

### 2. 혜택 구조

| 혜택 | 내용 | 조건 |
|------|------|------|
| 메인 혜택 | {{discount}}% 할인 | 기간 내 구매 |
| 추가 혜택 | 무료 배송 / 사은품 | N만원 이상 |
| 한정 혜택 | 선착순 N명 추가 쿠폰 | 조기 구매 |

### 3. 페이지 구조

| 순서 | 섹션 | 내용 |
|------|------|------|
| 1 | 히어로 배너 | 이벤트명 + 기간 + 메인 혜택 (카운트다운 선택) |
| 2 | 혜택 요약 | 3칸 아이콘 카드 (할인/배송/사은품) |
| 3 | 추천 상품 | 할인 적용 상품 그리드 (4열 PC / 2열 MO) |
| 4 | 참여 방법 | 스텝 1-2-3 (구매 → 리뷰 → 추첨) |
| 5 | 유의사항 | 약관, 교환/반품 조건 |

### 4. 카피 초안

- **히어로**: (감성 + 긴급성, 15자 이내)
- **서브**: (혜택 요약, 2줄)
- **CTA**: "지금 할인받기" / "이벤트 상품 보기"
- **유의사항**: (법적 고지 포함)

### 5. 운영 체크리스트

- [ ] 메이크샵 할인 설정 (관리자 > 프로모션)
- [ ] 쿠폰 코드 발급 (있을 경우)
- [ ] SNS 사전 공지 (D-3, D-1)
- [ ] 이벤트 종료 후 페이지 비활성화 일정

## 디자인 브리프

### 컬러 (이벤트 특화)

| 용도 | 색상 | HEX |
|------|------|-----|
| 배너 배경 | 시즌별 변경 | 봄:#f0e6d3 여름:#d3e6f0 가을:#e6d3c0 겨울:#d3d3e6 |
| 할인 강조 | 코럴/레드 | #E85D4A |
| 기간 카운트다운 | 다크 | #333333 배경 + #FFFFFF 텍스트 |
| CTA | 프라이머리 | #7d9675 |
| 브랜드 톤 유지 | 세이지 그린 | #637561 (헤더/푸터) |

| 요소 | 크기(PC) | 크기(MO) | 굵기 |
|------|---------|---------|------|
| 이벤트명 | 36px | 24px | 800 |
| 할인율 | 48px | 32px | 900 |
| 기간 | 16px | 14px | 500 |
| 상품명 | 15px | 14px | 500 |
| 가격 (취소선) | 14px | 13px | 400 |
| 할인가 | 18px | 16px | 700 |

### 배너 규격

- **PC**: 1200 x 400px (3:1)
- **모바일**: 720 x 480px (3:2)
- **포맷**: JPG (사진) / PNG (일러스트)
- **텍스트 영역**: 좌측 50% 또는 중앙
