# 랜딩 페이지 기획안 템플릿

> 유형: 강사 모집, 브랜드 소개, 서비스 안내, 캠페인
> 에이전트: makeshop-expert (opus)
> 입력: 페이지 목적, 타겟, 핵심 전환 액션

---

## PRD 섹션

### 1. 페이지 개요

- **페이지명**: {{page_name}}
- **목적**: {{goal}} (신청, 가입, 문의, 인지)
- **핵심 전환**: {{cta_action}} (폼 제출 / 페이지 이동 / 전화)
- **타겟**: {{target_audience}}
- **메이크샵 페이지 ID**: (신규 생성 또는 기존 ID)

### 2. 페이지 구조 (원페이저)

| 순서 | 섹션 | 높이(PC) | 전환 역할 |
|------|------|---------|---------|
| 1 | 히어로 | 100vh | 관심 유발 — 한줄 카피 + CTA |
| 2 | 문제 제기 | 400px | 공감 — "이런 고민 있으셨나요?" |
| 3 | 솔루션 | 500px | 해결책 제시 — 서비스/프로그램 소개 |
| 4 | 혜택 3종 | 300px | 구체적 가치 — 아이콘 카드 3개 |
| 5 | 사회적 증거 | 400px | 신뢰 — 후기, 수치, 파트너 로고 |
| 6 | FAQ | auto | 의문 해소 — 아코디언 5개 |
| 7 | 최종 CTA | 300px | 전환 — 폼 또는 버튼 (반복) |

### 3. 카피 초안

- **히어로 헤드라인**: (가치 제안, 15자 이내)
- **히어로 서브**: (구체적 혜택, 2줄)
- **문제 제기**: ("혹시 ~하신 적 있나요?" 패턴)
- **솔루션 헤드**: (브랜드명 + 핵심 가치)
- **혜택 3종**: (키워드 + 1줄 설명)
- **CTA 버튼**: "지금 신청하기" / "무료 상담 받기" / "자세히 보기"

### 4. 메이크샵 기술 참고

- **페이지 유형**: 자유 페이지 (page.html?id=XXXX)
- **URL**: `/shop/page.html?id={{page_id}}` (반드시 /shop/ 포함)
- **모바일 CTA**: position fixed, bottom 0, z-index 999
- **폼 연동**: NocoDB Webhook 또는 메이크샵 게시판 API
