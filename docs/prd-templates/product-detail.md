# 상품 상세페이지 기획안 템플릿

> 유형: 자사몰(메이크샵) 상품 상세페이지
> 에이전트: makeshop-expert (opus)
> 입력: 상품명, 카테고리, 주요 특징, 가격대

---

## PRD 섹션

### 1. 기획 개요

- **상품명**: {{product_name}}
- **카테고리**: {{category}} (압화/레진/캔들/석고/비즈/기타)
- **목표**: 이 페이지가 달성해야 할 것 (전환율, 체류시간 등)
- **타겟 고객**: 누구에게 팔 것인가

### 2. 페이지 구조 (섹션 순서)

| 순서 | 섹션 | 높이(PC) | 내용 |
|------|------|---------|------|
| 1 | 히어로 이미지 | 600px | 대표 이미지 + 상품명 + 한줄 카피 |
| 2 | 핵심 가치 3종 | 200px | 아이콘 + 키워드 + 설명 (3컬럼) |
| 3 | 상세 설명 | auto | 상품 스토리 + 사용법 + 소재 |
| 4 | 포함 내역 | 300px | 키트 구성품 리스트 (이미지+텍스트) |
| 5 | 리뷰/후기 | auto | 별점 + 포토 리뷰 (가상태그) |
| 6 | CTA | 80px | "지금 구매하기" 버튼 |
| 7 | 관련 상품 | 400px | 추천 상품 4개 (가상태그) |

### 3. 카피 초안

- **히어로 카피**: (1줄, 감성적, 20자 이내)
- **서브 카피**: (2줄, 기능적 가치 강조)
- **핵심 가치 3종**: (키워드 3개 + 각 1줄 설명)
- **CTA 버튼**: "지금 구매하기" / "키트 구성 보기" / "클래스 신청하기"

### 4. 메이크샵 기술 참고

- **페이지 유형**: 상품 상세 (shopdetail.html)
- **가상태그**: `<!--/상품이미지/-->`, `<!--/상품명/-->`, `<!--/판매가격/-->`, `<!--/상품설명/-->`
- **반응형**: 768px 브레이크포인트, 모바일 이미지 100% width
- **주의**: `${var}` 이스케이프 필수 → `\${var}`

- **이미지 비율**: 히어로 4:3, 상세 자유
- **버튼**: border-radius 8px, padding 16px 32px
