# SNS 콘텐츠 기획안 템플릿

> 유형: 인스타그램, 유튜브, 블로그, 네이버 포스트
> 에이전트: content-brand-planner (opus)
> 입력: 채널, 콘텐츠 형태, 주제, 목표

---

## PRD 섹션

### 1. 콘텐츠 개요

- **채널**: {{channel}} (인스타 피드/릴스/스토리, 유튜브, 블로그)
- **형태**: {{format}} (이미지, 영상, 카드뉴스, 숏폼)
- **주제**: {{topic}}
- **목표**: (도달, 참여, 유입, 전환)
- **게시 예정일**: {{publish_date}}
- **연관 상품/이벤트**: (있을 경우 링크)

### 2. 채널별 규격

#### 인스타그램

| 형태 | 비율 | 크기 | 길이 |
|------|------|------|------|
| 피드 이미지 | 1:1 | 1080x1080 | — |
| 카드뉴스 | 1:1 | 1080x1080 x N장 | 최대 10장 |
| 릴스 | 9:16 | 1080x1920 | 15~90초 |
| 스토리 | 9:16 | 1080x1920 | 15초 |

#### 유튜브

| 형태 | 비율 | 길이 | 썸네일 |
|------|------|------|--------|
| 숏츠 | 9:16 | 60초 이하 | 자동 |
| 일반 영상 | 16:9 | 5~15분 | 1280x720 |

#### 블로그

| 항목 | 규격 |
|------|------|
| 대표 이미지 | 16:9, 1200x675 |
| 본문 이미지 | 가로 860px 이상 |
| 글자 수 | 1500~3000자 (SEO) |

### 3. 콘텐츠 구성

#### 이미지/카드뉴스
- **1장(커버)**: 후킹 카피 + 비주얼
- **2~N장(본문)**: 정보/스토리 전개
- **마지막 장**: CTA ("팔로우" / "링크 클릭" / "댓글")

#### 영상 (릴스/숏츠)
- **0~3초**: 후킹 (질문 or 결과물 먼저)
- **3~20초**: 과정/정보
- **20~30초**: CTA + 브랜드 태그

#### 블로그
- **제목**: 키워드 포함, 30자 이내
- **도입**: 공감/질문 (2~3문장)
- **본문**: 소제목 3~5개 + 이미지 교차
- **마무리**: CTA + 관련 링크

### 4. 카피 초안

- **인스타 캡션**: (2~3줄 + 해시태그 15개)
- **유튜브 제목**: (키워드 + 호기심, 40자 이내)
- **유튜브 설명**: (요약 3줄 + 타임스탬프 + 링크)
- **블로그 제목**: (SEO 키워드 + 가치 제안)
- **해시태그 풀**: #프레스코21 #꽃공예 #{{category}} + 10개

### 5. 촬영/제작 체크리스트

- [ ] 완성작 촬영 (자연광, 배경 통일)
- [ ] 과정 촬영 (탑뷰 or 3/4 앵글)
- [ ] 브랜드 워터마크/로고
- [ ] BGM 선정 (저작권 프리)
- [ ] 자막 (릴스/숏츠 필수)

## 디자인 브리프

### 컬러 (SNS 특화)

| 용도 | 색상 | HEX |
|------|------|-----|
| 텍스트 오버레이 배경 | 반투명 화이트 | rgba(255,255,255,0.85) |
| 텍스트 | 다크 그린 | #425B51 |
| 포인트 강조 | 프라이머리 | #7d9675 |
| CTA 배경 | 세이지 | #637561 |
| 카드뉴스 배경 | 크림 | #f8f6f0 |

### 폰트 (이미지 내)

| 요소 | 크기 | 굵기 |
|------|------|------|
| 메인 카피 | 48~60pt | 800 |
| 서브 카피 | 24~32pt | 500 |
| 정보 텍스트 | 20~24pt | 400 |
| CTA 텍스트 | 28~32pt | 700 |

### 비주얼 톤

- **사진 톤**: 따뜻한 자연광, 채도 약간 낮춤 (빈티지 아님)
- **배경**: 우드/린넨/대리석 (소재감 있는 단색)
- **소품**: 꽃, 잎, 리본 (브랜드 연관 소재)
- **필터**: VSCO A6 계열 또는 무보정 자연톤
