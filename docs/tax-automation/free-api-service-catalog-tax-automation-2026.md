# 세무 자동화 무료 API·무료 조회 서비스 카탈로그 2026

> 작성일: 2026-04-25  
> 상태: Draft v1  
> 목적: 세무 자동화 시스템에서 활용 가능한 무료 API, 무료 조회 서비스, 수동 다운로드 서비스를 구분한다.  
> 주의: 세무 판단·신고 제출·납부 실행은 자동화 대상이 아니라 대표/세무사 승인 대상이다.

---

## 1. 분류 기준

| 분류 | 의미 | 자동화 전략 |
|---|---|---|
| 무료 API | API 키나 이용기관 등록 후 프로그램 호출 가능 | n8n/서버에서 dry-run 후 자동 수집 |
| 조건부 API | 무료 또는 공식 API이나 이용기관, 동의, 계약, 보안 심사가 필요 | PoC 전 이용조건 확인 |
| 무료 조회 서비스 | 본인인증 후 웹에서 무료 조회 가능하지만 API가 아님 | 비전공자 가이드와 수동 체크리스트로 활용 |
| 수동 다운로드 포털 | 로그인 후 PDF/XLSX/CSV를 직접 다운로드 | 파일 업로드 MVP와 파서로 처리 |
| 유료/벤더 후보 | 자동화 효율은 높지만 비용·계약 필요 | Phase 4 이후 비교 검토 |

## 2. 우선순위 요약

| 우선순위 | 소스 | 분류 | 세무 자동화 효용 | 판단 |
|---|---|---|---|---|
| P0 | UNI-PASS API001/API049 | 무료 API | 수입·통관·관세 증빙 | 이미 초안 있음, 실제 dry-run 필요 |
| P0 | 홈택스 다운로드 자료 | 수동 다운로드 포털 | 종소세 핵심 자료 | 수동 가이드와 파서가 먼저 필요 |
| P0 | 은행/카드 파일 다운로드 | 수동 다운로드 포털 | 계좌·카드 지출 증빙 | 수동 업로드 MVP 우선 |
| P1 | 국세청 사업자등록정보 API | 무료 API | 거래처 사업자 상태 검증 | 증빙 보조, 직접 세액 자료는 아님 |
| P1 | 오픈마켓/PG API 또는 export | 조건부 API/수동 다운로드 | 매출·수수료·정산 대조 | 채널별 확인 필요 |
| P1 | 금융결제원 오픈뱅킹 | 조건부 API | 계좌 거래 자동 수집 | 이용기관·동의 요건 검토 필요 |
| P2 | 계좌정보통합관리서비스 | 무료 조회 서비스 | 누락 계좌 찾기 | 자동 수집보다는 발견용 |
| P2 | 내보험찾아줌 | 무료 조회 서비스 | 보험계약·숨은보험금 확인 | 개인 공제/보험 자료 보조 |
| P2 | 카드포인트 통합조회 | 무료 조회 서비스 | 카드사 보유 여부와 포인트 확인 | 세무 핵심은 아님 |
| P2 | 사회보험통합징수포털 | 수동 다운로드 포털 | 4대보험 납부확인서 | 인증 후 증명서 발급 |
| P2 | 정부24 | 수동 다운로드 포털 | 국세증명·민원증명 | 발급은 사람 승인 유지 |

## 3. API 후보 상세

### 3.1 UNI-PASS API001 화물통관 진행정보

| 항목 | 내용 |
|---|---|
| 운영기관 | 관세청/UNI-PASS |
| 분류 | 무료 API |
| 용도 | 화물관리번호 또는 BL 기반 통관 진행상태 조회 |
| 세무 효용 | 수입 상품의 통관 상태, 수입신고번호 후보, 관세 증빙 연결 |
| 인증 | UNI-PASS Open API 키 |
| 자동화 방식 | n8n 정기 조회, raw XML 저장, 통관 상태 정규화 |
| 주의 | BL 조회 시 입항년도 등 파라미터가 필요할 수 있음 |
| 우선순위 | P0 |

### 3.2 UNI-PASS API049 수입제세 납부여부 조회

| 항목 | 내용 |
|---|---|
| 운영기관 | 관세청/UNI-PASS |
| 분류 | 무료 API |
| 용도 | 수입신고번호 기준 수입제세 납부여부와 납부일자 확인 |
| 세무 효용 | 관세·수입부가세 납부 증빙 누락 방지 |
| 인증 | UNI-PASS Open API 키 |
| 자동화 방식 | API001 또는 별도 신고번호 목록에서 `impDclrNo` 확보 후 단건 조회 |
| 주의 | 월간 기간 조회 API로 쓰면 안 됨. 수입신고번호가 선행되어야 함 |
| 우선순위 | P0 |

### 3.3 국세청 사업자등록정보 진위확인 및 상태조회 API

| 항목 | 내용 |
|---|---|
| 운영기관 | 국세청, 공공데이터포털 |
| 분류 | 무료 API |
| 용도 | 사업자등록정보 진위확인, 휴폐업/과세유형 등 상태조회 |
| 세무 효용 | 거래처 사업자번호 검증, 세금계산서/증빙 메타데이터 보조 검증 |

| 주의 | 직접 매출·매입 금액을 제공하는 API가 아니므로 보조 검증용 |
| 우선순위 | P1 |
| 공식 링크 | https://www.data.go.kr/data/15081808/openapi.do |

### 3.4 금융결제원 오픈뱅킹 거래내역조회 API

| 항목 | 내용 |
|---|---|
| 운영기관 | 금융결제원 |
| 분류 | 조건부 API |
| 용도 | 사용자가 등록한 계좌의 거래내역 조회 |
| 세무 효용 | 사업용 계좌 입출금 자동 수집, 큰 금액 거래 설명 후보 탐지 |
| 인증 | 이용기관 등록, 사용자 계좌등록, Access Token 등 |

| 자동화 방식 | 이용기관 요건 충족 시 서버 수집, 아니면 은행 CSV 업로드 fallback |
| 주의 | 개인/사업 계좌 접근 권한과 동의, 보안 심사가 핵심 병목 |
| 우선순위 | P1 검토 |
| 공식 링크 | https://developers.kftc.or.kr/dev/openapi/open-banking/transaction |

## 7. 개발 우선순위

### 7.1 즉시 PoC

1. UNI-PASS API001/API049 실제 dry-run
2. 국세청 사업자등록정보 API 샘플 조회
3. 홈택스 XLSX/CSV/PDF 파일 파서
4. 파일명/해시/manifest 생성기

### 7.2 조건 확인 후 PoC

1. 금융결제원 오픈뱅킹 이용기관 등록 가능성
2. 오픈마켓/PG 공식 API 이용조건
3. 홈택스 공식 API 또는 벤더 API 비용/보안 조건

### 7.3 수동 가이드 유지

1. 내보험찾아줌
2. 계좌정보통합관리서비스
3. 카드포인트 통합조회
4. 휴면예금 찾아줌
5. 정부24 증명서 발급
6. 사회보험통합징수포털 제증명 발급

## 8. 출처
