# 종합소득세 자료 수집 가이드 2026

> 대상: 비전공자 직원 또는 대표가 세무사에게 자료를 넘기기 전 따라 하는 실무 가이드  
> 기준: 2025년 귀속 종합소득세 자료 수집, 2026년 신고 준비  
> 작성일: 2026-04-25  
> 상태: Draft v1  
> 중요: 이 문서는 세무 신고·절세 판단 가이드가 아니라 자료 수집 가이드다. 경비 인정, 공제 가능 여부, 신고 방식, 세액 확정은 세무사에게 확인한다.

---

## 0. 먼저 이해하기

### 0.1 왜 이렇게 자료가 많은가

종합소득세는 “내가 1년 동안 번 돈”과 “그 돈을 벌기 위해 쓴 비용”, “이미 낸 세금”, “공제받을 수 있는 자료”를 모두 모아 계산한다. 그래서 단순히 홈택스 지급명세서만 받으면 끝나지 않는다.

PRESSCO21처럼 자사몰, 오픈마켓, 도소매, 제조, 수입/통관, 인건비가 섞인 사업은 아래 자료가 함께 필요하다.

- 홈택스 소득·매출·매입 자료
- 오픈마켓/자사몰/PG 매출·수수료·정산 자료
- 은행 계좌 거래내역
- 카드 사용내역
- 현금영수증 지출증빙
- 4대보험/급여/원천세 자료
- 통관·관세 자료
- 전년도 신고서와 세금 납부내역
- 보험료, 의료비, 기부금, 연금저축 등 공제 자료
- 세무사가 오해할 수 있는 거래에 대한 설명 메모

### 0.2 신고 기간 기준

국세청 종합소득세 안내는 종합소득금액이 있는 사람이 다음 해 5월에 종합소득세를 신고·납부하는 구조를 안내한다. 실무에서는 휴일, 성실신고확인 대상 여부, 개별 안내에 따라 마감일이 달라질 수 있으므로, 2025년 귀속분의 정확한 마감일은 홈택스 안내와 세무사 확인을 따른다.

### 0.3 이 가이드의 목표

이 가이드를 끝까지 따르면 다음 산출물이 만들어진다.

```text
2025_종합소득세_세무사전달/
  00_세무사전달_manifest/
  01_기본정보/
  02_소득자료/
  03_매출자료/
  04_비용증빙/
  05_공제자료/
  06_세금신고이력/
  07_통관관세/
  08_급여4대보험/
  09_설명메모/
  99_확인필요/
```

## 1. 준비물

### 1.1 로그인 수단

- 홈택스 로그인 수단
  - 공동인증서
  - 금융인증서
  - 간편인증
  - 홈택스 ID
- 은행/카드사 로그인 수단
- 오픈마켓/자사몰/PG 관리자 로그인 수단
- 정부24 로그인 수단
- 사회보험통합징수포털/4대사회보험 관련 로그인 수단
- UNI-PASS 로그인 또는 Open API 키 정보

주의: 공동인증서, 금융인증서, 비밀번호, API 키는 문서에 적지 않는다. 필요한 담당자만 본인 기기에서 직접 로그인한다.

## 3. 0단계: 무료 일괄조회 서비스로 빠진 계좌·보험·카드 확인

이 단계는 세무 신고 자료 자체를 바로 만들어 주는 단계가 아니라, “내가 잊고 있던 계좌, 보험, 카드, 휴면자산이 있는지”를 먼저 확인하는 단계다. 누락 계좌나 카드가 있으면 사업 관련 거래가 빠질 수 있다.

### 3.1 계좌정보통합관리서비스 / 어카운트인포

목적:

- 내 명의 계좌를 한 번에 확인한다.
- 비활동성 계좌와 자동이체를 확인한다.
- 사업 관련 입출금이 있었던 계좌를 빠뜨리지 않는다.

절차:

1. https://www.payinfo.or.kr 접속
2. `계좌정보통합관리서비스` 또는 `내 계좌 한눈에` 메뉴로 이동
3. 본인인증 진행
4. 은행·제2금융권 계좌 목록 확인
5. 사업 관련으로 보이는 계좌가 있으면 별도 메모
6. 이후 해당 은행 사이트에서 2025년 거래내역을 다운로드

저장 파일:

```text
09_설명메모/2025_계좌정보통합관리_사업관련계좌메모.xlsx
```

주의:

- 계좌정보통합관리서비스에서 보이는 화면 캡처만 세무사에게 넘기기보다, 실제 거래내역은 각 은행에서 기간을 지정해 다운로드한다.

### 3.2 내보험찾아줌

목적:

- 가입 보험과 숨은보험금을 확인한다.
- 보험료 공제자료와 실제 보험계약이 맞는지 확인한다.
- 환급/보험금 수령 내역이 있으면 세무사에게 알려야 할 수 있다.

절차:

1. https://cont.insure.or.kr 접속
2. 본인인증 진행
3. 보험가입내역 확인
4. 숨은보험금 또는 보험금 청구 가능 내역 확인
5. 공제자료와 차이가 있거나 보험금 수령이 있으면 메모

저장 파일:

```text
05_공제자료/2025_내보험찾아줌_보험확인메모.pdf
09_설명메모/2025_보험금수령_세무사확인필요.xlsx
```

주의:

- 보험료 공제 가능 여부와 보험금 과세 여부는 세무사에게 확인한다.

### 3.3 카드포인트 통합조회

목적:

- 내가 쓰는 카드사를 빠짐없이 확인한다.
- 카드포인트 현금화가 있었으면 기타 입금 메모에 남긴다.

절차:

1. https://www.cardpoint.or.kr 접속
2. `통합조회/계좌입금/기부`로 이동
3. 본인인증 진행
4. 카드사별 포인트 확인
5. 2025년에 사업 관련 카드가 있었는지 카드사 목록을 확인

저장 파일:

```text
09_설명메모/2025_카드사목록_확인메모.xlsx
```

주의:

- 세무 핵심 자료는 카드포인트가 아니라 카드 사용내역이다. 카드포인트 조회 후 각 카드사에서 연간 사용내역을 따로 다운로드한다.

### 3.4 휴면예금 찾아줌 / 서민금융진흥원

목적:

- 휴면예금이나 오래된 계좌를 확인한다.
- 사업 관련 입출금이 있었던 과거 계좌가 있는지 점검한다.

절차:

1. 서민금융진흥원 휴면예금 찾아줌 또는 정부24/어카운트인포의 휴면예금 조회 경로 접속
2. 본인인증 진행
3. 휴면예금 조회
4. 사업 관련 계좌 가능성이 있으면 메모

저장 파일:

```text
09_설명메모/2025_휴면예금_확인메모.xlsx
```

## 4. 1단계: 기본 정보 수집

저장 폴더:

```text
01_기본정보/
```

### 4.1 사업자등록증명

목적:

- 현재 사업자 정보, 상호, 사업자등록번호, 업태/종목, 사업장 주소를 세무사에게 확인시킨다.

방법 A: 홈택스

1. 홈택스 https://www.hometax.go.kr 접속
2. 로그인
3. 검색창에 `사업자등록증명` 입력
4. 민원증명 발급 화면으로 이동
5. 발급 용도는 `관공서 제출` 또는 세무사 요청 용도에 맞게 선택
6. PDF 저장

방법 B: 정부24

1. 정부24 https://www.gov.kr 접속
2. 검색창에 `사업자등록증명` 입력
3. 민원 안내 및 신청으로 이동
4. 로그인/본인인증 후 발급
5. PDF 저장

파일명:

```text
2025_사업자등록증명_프레스코21.pdf
```

### 4.2 전년도 종합소득세 신고서와 납부서

목적:

- 세무사가 전년도 이월사항, 기납부세액, 신고 방식, 비교 자료를 확인할 수 있게 한다.

절차:

1. 홈택스 로그인
2. 검색창에 `종합소득세 신고내역` 또는 `신고서 조회` 입력
3. 전년도 귀속 신고서 조회
4. 신고서, 접수증, 납부서를 PDF로 저장
5. 지방소득세 신고/납부 내역도 있으면 함께 저장

파일명 예시:

```text
2024_종합소득세_신고서.pdf
2024_종합소득세_납부서.pdf
2024_지방소득세_신고납부내역.pdf
```

### 4.3 환급 계좌 정보

목적:

- 환급이 발생할 경우 세무사가 계좌 정보를 확인할 수 있게 한다.

저장 방식:

```text
01_기본정보/2025_환급계좌_메모.xlsx
```

내용:

- 은행명
- 예금주
- 계좌 끝 4자리
- 대표 확인 여부

주의:

- 계좌 전체번호를 문서에 넣을지는 세무사 요청 기준으로 결정한다. 공유 파일에는 불필요한 전체번호 노출을 피한다.

## 5. 2단계: 홈택스 소득 자료 다운로드

저장 폴더:

```text
02_소득자료/
```

### 5.1 지급명세서

목적:

- 근로소득, 사업소득, 기타소득 등 원천징수된 소득 자료를 확인한다.

절차:

1. 홈택스 로그인
2. 검색창에 `지급명세서` 입력
3. `지급명세서 등 제출내역` 또는 유사 메뉴로 이동
4. 귀속연도 `2025` 선택
5. 조회되는 항목을 확인
   - 근로소득 지급명세서
   - 사업소득 지급명세서
   - 기타소득 지급명세서
   - 퇴직소득 지급명세서
   - 일용근로소득 지급명세서
   - 간이지급명세서
6. 항목별로 PDF 또는 Excel 저장

파일명 예시:

```text
2025_홈택스_지급명세서_근로소득.pdf
2025_홈택스_지급명세서_사업소득.pdf
2025_홈택스_지급명세서_기타소득.pdf
2025_홈택스_지급명세서_간이지급명세서.pdf
```

확인 포인트:

- 대표 개인 근로소득이나 기타소득이 있으면 사업소득과 함께 세무사에게 알려야 한다.
- 홈택스에 안 보이는 소득이 있어도 실제 입금이 있으면 설명 메모에 적는다.

## 6. 3단계: 홈택스 매출 자료 다운로드

저장 폴더:

```text
03_매출자료/
```

### 6.1 전자세금계산서 매출

절차:

1. 홈택스 로그인

## 13. 10단계: 통관·관세 자료 수집

### 13.1 UNI-PASS 수동 조회

목적:

- 수입 상품의 통관 진행, 수입신고번호, 관세/부가세 납부 여부를 확인한다.

절차:

1. UNI-PASS https://unipass.customs.go.kr 접속
2. 메인 또는 정보조회 메뉴에서 화물진행정보 조회
3. 화물관리번호 또는 M B/L, H B/L, 입항연도 입력
4. 통관 진행 상태 확인
5. 수입신고번호가 보이면 별도 메모
6. 수입제세 납부 관련 자료가 있으면 저장

파일명:

```text
2025_UNIPASS_화물통관진행_연간.xlsx
2025_UNIPASS_수입신고번호목록_연간.xlsx
2025_UNIPASS_수입제세납부확인_연간.pdf
```

### 13.2 자동화 예정 흐름

현재 개발 방향은 아래 순서다.

```text
화물관리번호/BL 목록 확보
→ UNI-PASS API001 통관 진행 조회
→ 수입신고번호 dclrNo 후보 추출
→ UNI-PASS API049 수입제세 납부여부 조회
→ customs_clearance_status / customs_tax_payments 저장
→ 세무사 패키지 07_통관관세에 포함
```

주의:

- API049는 수입신고번호 단건 조회가 필요하므로, 먼저 수입신고번호 목록이 있어야 한다.
- 관세, 수입부가세 처리 방식은 세무사 확인이 필요하다.

### 14.2 4대보험 납부확인서

1. https://si4n.nhis.or.kr 접속
2. 사업장 회원 로그인

6. 보험 구분 선택
   - 건강보험
   - 국민연금
   - 고용산재보험
   - 합산보험
7. 기간 `2025년 1월~12월` 선택
8. PDF 발급/저장

## 16. 13단계: 세무사 전달 전 검산

### 16.1 월 누락 확인

아래 항목은 1월~12월이 모두 있는지 확인한다.

| 자료 | 1월~12월 확인 |
|---|---|
| 은행 거래내역 | 월별 누락 없음 |
| 카드 사용내역 | 카드별 누락 없음 |
| 홈택스 전자세금계산서 | 연간 또는 월별 전체 있음 |
| 현금영수증 | 연간 또는 월별 전체 있음 |
| 오픈마켓/PG 정산 | 월별 누락 없음 |
| 급여/4대보험 | 해당 월 전체 있음 |
| 통관/관세 | 수입 건별 누락 없음 |

### 16.2 금액 대조 확인

정확한 세무 판단은 세무사가 하지만, 아래 차이는 미리 표시한다.

- 홈택스 카드 매출과 PG 정산액 차이
- 오픈마켓 총매출과 정산 입금액 차이
- 부가세 신고서 매출과 플랫폼 매출 합계 차이
- 은행 입금액과 정산자료 차이
- 카드 사용내역과 홈택스 사업용 카드 자료 차이
- 전자세금계산서와 카드 결제 중복 가능성

## 18. 자주 놓치는 자료

- 개인계좌로 받은 사업 매출
- 개인카드로 결제한 사업 비용
- 사업용 카드 등록 전 사용분
- 현금영수증을 개인 소득공제용으로 받은 사업 지출
- 오픈마켓 정산액만 있고 총매출/수수료 자료가 없는 경우
- PG 수수료와 광고비
- 환불/취소 내역
- 배송비, 물류비, 포장재, 재료비
- 수입 통관 관련 관세/수입부가세
- 전년도 이월결손금, 중간예납, 기납부세액
- 4대보험 납부확인서
- 외주비 원천징수/계약서/송금내역
- 부양가족 자료 제공 동의 미완료
- 간소화 PDF에 없는 기부금/월세/의료비
- 보증금, 대출, 가족 간 이체 설명 메모

## 19. 세무사 확인 필요 항목

아래는 자동으로 판단하지 않는다.

- 종합소득세 신고 대상 소득 범위
- 사업소득과 기타소득 구분
- 간편장부/복식부기/추계신고 선택
- 성실신고확인 대상 여부
- 업종코드 적정성
- 개인카드 사업 비용 인정 범위
- 차량 비용 인정 범위
- 자택 사무실 비용 인정 범위
- 가족 인건비 인정 요건
- 접대비/식대/회의비 처리 기준
- 전자세금계산서와 카드 결제 중복 여부
- 현금영수증 개인공제용/지출증빙용 정정 필요 여부
- 부양가족 공제 가능 여부
- 금융소득 종합과세 여부
- 해외소득 신고 여부
- 전년도 결손금 이월공제 가능 여부
- 지방소득세 신고/납부 처리 주체
