## 🎯 개정 배경과 방향

### 개정 이유
1. **근기법 제93조 필수 기재 사항 보완** — 현행 인사규정에 임금·퇴직급여·모성보호·직장 내 괴롭힘·안전보건·차별 금지 조항이 누락되어 있음
2. **유연근로시간제 도입** — 사무실 선택근로, 재택 재량근로
3. **모성보호·일·가정 양립 제도 정비** — 현재 출산휴가 중인 직원 지원 + 가족친화인증 등 정부 지원 신청 기반 확보
4. **2025.10.01 개정 남녀고용평등법 반영** — 배우자 출산휴가 20일, 육아휴직 만 8세/초2 이하 확대, 육아기 근로시간 단축 만 12세/초6 이하 확대, 난임치료휴가 연 6일

### 작성 원칙 (최소안 방침)
- **법적 의무 + 정부 지원 신청 최소 요건**에 맞춘 최소안 작성
- 회사에 추가 부담이 생기는 세부 규정은 포함하지 않음
- 세부 운영은 개별 근로계약서·관계 법령에 위임
- "관계 법령에 따른다" 수준의 문구 활용 (불이익 변경 리스크 회피)
- 09번 옵션 대조표에 [FM안]을 함께 제시하여 노무사가 가감 판단할 수 있도록

## 🏗️ 3-Track 개편 구조

| Track | 제도 | 대상 | 근거 | 비고 |
|-------|------|------|------|------|
| A | 선택적 근로시간제 | 사무실 근무자 | 근기법 §52 | Core Time 11~16시, 포괄임금 병존 |
| B | 재량근로시간제 | 재택 디자인·영상 업무 | 근기법 §58 ③ + 시행령 §31 | 1일 8시간 간주, 간주근로 만료 후 전환 |

장지호(경영기획 팀장)는 Track A(사무실 선택근로)에 포함. 장준혁(공동대표 전환 예정)은 근로자 신분 이탈 예정으로 HR 대상 제외.

- 비용이 발생하는 경우 사전 견적 확인
