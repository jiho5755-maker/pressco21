# G03 — 가족친화기업 인증 신청 가이드

> 작성일: 2026년 4월 15일
> 근거 법령: 가족친화 사회환경의 조성 촉진에 관한 법률 제15조
> 대상: 프레스코21 (상시 8명, 서울 소재)

---

## 🎯 가족친화인증이란?

**성평등가족부장관**이 가족친화제도를 모범적으로 운영하는 기업·공공기관에 부여하는 국가 인증입니다.

### 법적 근거
- **가족친화 사회환경의 조성 촉진에 관한 법률 제15조 (가족친화기업 등 인증)**
  - 성평등가족부장관이 가족친화제도를 모범적으로 운영하고 있는 기업 또는 공공기관에 대하여 인증을 할 수 있음
  - 인증받은 기업은 인증 표시 사용 가능
  - 인증받지 않은 기업이 인증 표시 또는 유사 표시 사용 금지

### 운영 기관
- **주관**: 성평등가족부
- **심사·운영**: 한국건강가정진흥원
- **지원**: 노사발전재단 (사전 컨설팅)

---

## 💎 인증 혜택

### 직접 혜택
1. **인증 마크 사용** — 명함·홈페이지·카탈로그·제품 포장 등에 사용 가능. 브랜드 가치 향상.
2. **인증서 발급** — 유효기간 3년.

### 정부·공공기관 연계 혜택
3. **정부 조달 입찰 가점** — 중소기업 조달 입찰 시 가산점 부여.
4. **정부 사업 선정 가점** — 여러 정부 지원 사업 신청 시 우대.
5. **금융 우대** — 일부 금융기관에서 대출 금리 인하·보증료 우대.
6. **세무조사 유예** — 일정 기간 세무조사 유예 혜택 가능.
7. **출입국 우수기업 우대** — 외국인 채용 시 비자 처리 우대.
8. **근로감독 면제** — 고용노동부 정기 근로감독 면제 (일정 기간).

### 간접 혜택
9. **직원 채용 경쟁력 향상** — 구직자가 선호하는 인증 기업으로 인식.
10. **직원 몰입도·만족도 향상** — 일·가정 양립 문화 확립.
11. **이직률 감소** — 특히 여성·육아기 직원 유지.

→ **인증 신청 가능한 상태**가 됩니다 (이번 인사규정 개정 완료 후).

## 🔍 인증 평가 지표 (대략)

가족친화인증의 평가 지표는 여러 차원으로 구성되며, 주요 항목은 다음과 같습니다 (상세 배점은 매년 한국건강가정진흥원에서 고지).

### A. CEO·최고경영자 의지
- 가족친화경영 비전 선포
- 경영 방침에 일·가정 양립 반영

### B. 자녀 출산·양육 지원
- 배우자 출산휴가 20일 유급 운영
- 출산전후휴가 법정 이상 제공
- 육아휴직 사용률
- 육아기 근로시간 단축 사용률
- 수유시간·태아검진시간 운영

### C. 유연근무제
- 시차출퇴근
- 선택적 근로시간제
- 재량근로시간제
- 재택근무
- 원격근무

### D. 가족친화직장문화
- 정시 퇴근 유도
- 연차 사용 촉진
- 직장 내 괴롭힘 예방
- 양성평등 문화

### E. 가족친화 참여 지원
- 가족돌봄휴직·휴가
- 가족 행사 지원
- 근로자 심리상담

### F. 제도 운영의 실효성
- 실제 사용률
- 근로자 만족도

→ 프레스코21은 이번 인사규정 개정으로 **A·B·C·D 항목의 기본 요건 충족** 상태가 됩니다.

## 📅 신청 절차 (대략)

```
① 사전 준비 (2~3개월)
   ├ 인사규정 정비 (모성보호·일가정양립·직장내괴롭힘·차별금지)
   ├ 신청서·관리대장·복귀협의서 등 내부 양식 구비
   ├ CEO 경영방침·비전 선포
   └ 노사발전재단 사전 컨설팅 신청 (무료 권장)

② 자가진단
   └ 한국건강가정진흥원 자가진단표 작성 → 예상 점수 산정

③ 신청서 제출 (매년 모집 시기)
   └ 온라인 또는 우편 접수

④ 서면 심사
   └ 제출 서류 + 자가진단 근거 검증

⑤ 현장 심사
   └ 심사위원 사업장 방문 + 직원 면담

⑥ 인증 심의
   └ 심사위원회 종합 평가

⑦ 인증서 발급 (통상 연말)
   └ 유효기간 3년, 갱신 심사 필요
```

**매년 신청 시기·심사 일정**은 해당 연도 한국건강가정진흥원 공지를 확인하세요.

1. **인증받지 않은 기업이 인증 표시 사용 금지** (가족친화법 §15 ④) — 인증 후에만 마크 사용.
