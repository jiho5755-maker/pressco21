## 1. 채널별 마진 목표 설정

### 1-1. 채널별 역할 정의

8개 채널은 동일한 역할을 수행하지 않는다. 각 채널의 **전략적 역할**에 따라 마진 목표도 달라져야 한다.

| 채널 | 전략적 역할 | 핵심 가치 | 마진 전략 방향 |
|------|-----------|----------|--------------|
| 메이크샵(자사몰) | **수익 기반** | 최저 수수료, LTV 극대화 | 최고 마진 추구 |
| 쿠팡 윙 | **볼륨 확보** | 대규모 트래픽, 브랜드 인지도 | 중간 마진, 볼륨 우선 |
| 쿠팡 로켓배송 | **인지도 확산** | 로켓배송 배지, 신뢰도 | 납품가 협상, 최소 마진 확보 |
| 쿠팡 로켓그로스 | **소형 상품 특화** | 빠른 배송, CS 부담 절감 | 풀필먼트비 포함 중간 마진 |
| 네이버 스마트스토어 | **신규 유입 핵심** | 네이버 검색 트래픽 | 중상 마진, 검색 광고 연계 |
| 11번가 | **추가 노출** | SKT 멤버십 고객, 추가 채널 | 중간 마진, 판매가 인상 허용 |
| 오늘의집 | **프리미엄 포지셔닝** | 인테리어/홈리빙 고관여 고객 | 고단가 상품 집중, 높은 절대 마진 |
| 카페24 유튜브쇼핑 | **콘텐츠 커머스** | 유튜브 채널 시너지, 충성 고객 | 중상 마진, 자사몰 다음 수익 채널 |

### 1-2. 이중 마진 기준 체계 (Go/No-Go + Target)

각 채널에 **최소선(Go/No-Go)**과 **목표선(Target)**을 설정한다. 최소선 미달 상품은 해당 채널에서 판매를 중단하거나 판매가를 인상해야 한다.

| 채널 | 최소선 (Go/No-Go) | 목표선 (Target) | 근거 |
|------|-------------------|----------------|------|
| 메이크샵(자사몰) | 20% | **30%** | 수수료 3.3%로 최고 마진 가능. 목표도 가장 높게 설정 |
| 쿠팡 윙 | 12% | **20%** | 수수료 10.8% + 배송비 부담. 트래픽 보상분 마진 양보 |
| 쿠팡 로켓배송 | 8% | **15%** | 납품가 기반. 대량 납품 시 COGS 절감 효과 반영 |
| 쿠팡 로켓그로스 | 10% | **18%** | 수수료+풀필먼트. 배송 CS 부담 제거 감안 |
| 네이버 스마트스토어 | 15% | **25%** | 수수료 5.7%로 비교적 양호. 검색 광고비 여력 확보 필요 |
| 11번가 | 12% | **20%** | 수수료 12% 부담. 추가 노출 채널로 판매가 인상 허용 |
| 오늘의집 | 10% | **20%** | 수수료 17.5% 고부담. 반드시 고단가 상품만 입점 |
| 카페24 유튜브쇼핑 | 18% | **28%** | 수수료 4~6%로 양호. 콘텐츠 제작비 내재화 필요 |

### 1-3. 최소선(Go/No-Go) 적용 규칙

```
판정 프로세스:

1. 상품 X의 채널 Y 순마진율 산출
2. 순마진율 < 최소선?
   ├─ YES → 3단계 검토 진입
   └─ NO  → 정상 판매 유지

3단계 검토:
  Step 1. 판매가 인상 여력 확인
    → 경쟁사 대비 +10% 이내 인상 가능한가?
    → YES: 인상 후 재계산
    → NO: Step 2로

  Step 2. COGS 절감 가능성 확인
    → 대량 구매, 조달처 변경, 패키징 간소화 등
    → YES: 적용 후 재계산
    → NO: Step 3으로

  Step 3. 해당 채널 판매 중단 결정
    → 단, "인지도 확산" 역할 채널은 전략적 유지 검토
    → 최종 결정권: 대표
```

### 1-4. 신규 채널 입점 타당성 평가 기준

새로운 판매 채널 입점을 검토할 때 적용하는 기준표:

| 평가 항목 | 가중치 | 기준 | 채점 (1~5점) |
|----------|-------|------|-------------|
| **예상 순마진율** | 30% | 15% 이상 = 5점, 10% = 3점, 10% 미만 = 1점 |
| **월간 예상 주문 수** | 20% | 50건+ = 5점, 20건 = 3점, 10건 미만 = 1점 |
| **운영 복잡도** | 15% | 기존 시스템 연계 = 5점, 별도 시스템 = 1점 |
| **브랜드 적합성** | 15% | 꽃 공예/인테리어 카테고리 = 5점, 무관 = 1점 |
| **고객 중복도** | 10% | 기존 채널과 중복 낮음 = 5점, 높음 = 1점 |
| **성장 잠재력** | 10% | 카테고리 성장세 = 5점, 정체/하락 = 1점 |

**입점 기준**:
- 가중 평균 3.5점 이상: **입점 진행**
- 가중 평균 2.5~3.5점: **3개월 테스트 후 재평가**
- 가중 평균 2.5점 미만: **입점 보류**

**필수 조건 (점수 무관하게 충족해야 함)**:
- 예상 순마진율 10% 이상 (절대 하한)
- 월 관리 공수 8시간 이내 (운영 부담 한계)

## 2. 판매가 전략 설계

### 2-1. 채널별 판매가 차등 vs 동일 가격 전략

#### 전략 A: 동일 가격 전략 (모든 채널 동일 판매가)

| 장점 | 단점 |
|------|------|
| 가격 일관성으로 브랜드 신뢰 유지 | 고수수료 채널에서 마진 압박 |
| 가격 비교 사이트에서 혼란 없음 | 저수수료 채널의 마진 잠재력 미활용 |
| 운영 관리 단순 | 채널별 마진 편차 큼 (3.3% vs 17.5%) |
| 고객 불만 최소화 (타채널 더 싸다는 항의 없음) | 오늘의집 등 고수수료 채널에서 적자 가능 |

#### 전략 B: 차등 가격 전략 (채널별 판매가 상이)

| 장점 | 단점 |
|------|------|
| 채널별 마진 최적화 가능 | 가격 비교 사이트에서 최저가 채널로 쏠림 |
| 고수수료 채널도 수익성 확보 | "왜 여기가 더 비싸요?" 고객 문의 발생 |
| 채널 특성에 맞는 가격 포지셔닝 | 운영 관리 복잡도 증가 |
| 프리미엄 채널에서 고가 설정 가능 | 채널 정책 위반 가능성 (최저가 보장 등) |

#### PRESSCO21 권장: **혼합 전략 (Hybrid)**

```
[혼합 전략 규칙]

기본 원칙: 메이크샵 자사몰 판매가 = 기준가 (100%)

채널별 가격 조정:

  메이크샵 자사몰   : 기준가 (최저가)
  카페24 유튜브쇼핑  : 기준가 (자사몰과 동일 -- 충성 고객 배려)
  네이버 스마트스토어 : 기준가 ~ 기준가+3% (미세 인상 허용)
  쿠팡 윙          : 기준가+5~8% (수수료 차이분 보전)
  쿠팡 로켓그로스    : 기준가+5~8% (풀필먼트비 보전)
  11번가           : 기준가+5~10% (수수료 12% 보전)
  오늘의집          : 기준가+10~15% (프리미엄 포지셔닝 + 수수료 보전)
  쿠팡 로켓배송     : 쿠팡 결정 (셀러 관여 불가)
```

**핵심 이유**:
- 꽃 공예 재료는 "최저가 검색" 비율이 가전/생활용품 대비 현저히 낮다
- 압화/레진/캔들 재료 구매자는 "신뢰할 수 있는 판매자"를 더 중시한다
- 오늘의집 고객은 인테리어 맥락에서 구매하므로 가격 민감도가 상대적으로 낮다
- 자사몰+유튜브쇼핑을 최저가로 설정하여 자사 채널 유입 유도

### 2-2. 가격 라운딩 포인트 활용

꽃 공예 재료의 가격대별 최적 라운딩 포인트:

```
[가격대별 라운딩 전략]

1,000원 미만 (소형 부자재: 글리터, 와이어 등)
  → 990원 단위: 990원, 1,990원
  → 심리적 1,000원 미만 효과 극대화

1,000~5,000원 (중소 부자재: 꽃잎 팩, 레진 소분 등)
  → 900원 단위: 2,900원, 3,900원, 4,900원
  → 천 단위 미만 가격 효과

5,000~15,000원 (주력 재료: 압화 세트, 레진 키트, 몰드 등)
  → 800~900원 단위: 7,800원, 9,900원, 12,800원, 14,900원
  → 만원 단위 심리 경계선 활용

15,000~30,000원 (프리미엄/세트: 대용량, 종합 키트 등)
  → 500~900원 단위: 19,500원, 24,800원, 29,900원
  → 무료배송 기준 금액과 연계

30,000원 이상 (대용량/프로용 세트, 오늘의집 프리미엄)
  → 000원 단위: 35,000원, 49,000원, 58,000원
  → 고가 상품은 깔끔한 가격이 오히려 프리미엄 느낌
```

**라운딩 시 마진 역산 검증**:
```
반드시 라운딩 "전" 역산 가격과 "후" 실제 판매가의 마진 차이를 확인한다.

예시:
  역산 판매가 = 10,232원
  → 라운딩 후보: 9,900원 vs 10,800원

  9,900원 선택 시: 순마진율 22.1% (목표 대비 -2.9%p)
  10,800원 선택 시: 순마진율 26.8% (목표 대비 +1.8%p)

  → 목표 마진율 25%이면 10,800원 채택
  → 경쟁사 가격이 10,500원이면 9,900원 + 대안 검토
```

### 2-3. 채널 정책에 의한 가격 통제 대응

| 채널 | 가격 통제 정책 | PRESSCO21 대응 |
|------|-------------|--------------|
| 쿠팡 로켓배송 | 쿠팡이 판매가 결정 (셀러 관여 불가) | 납품가 협상에 집중. COGS+목표마진 기반 최소 납품가 설정 |
| 쿠팡 윙 | 최저가 매칭 압박 (경고/노출 하락) | 자사몰보다 5~8% 높게 설정하되, 타 오픈마켓과 동일 수준 유지 |
| 네이버 | 가격 비교 노출 (네이버 쇼핑) | 네이버 내 최저가 노출이 트래픽 핵심. 동종 셀러 대비 경쟁력 유지 |
| 오늘의집 | 가격 제약 낮음 (버티컬 특성) | 프리미엄 가격 설정 가능. 구성/패키징 차별화로 정당화 |
| 11번가 | 가격 비교 제도 운영 | 11번가 전용 구성(약간 다른 세트)으로 가격 비교 회피 |

**채널별 SKU 차별화 전략** (가격 통제 우회):

```
같은 상품이라도 채널별로 구성을 미세 조정하여 "다른 상품"으로 인식시킨다.

예시: 압화 꽃잎 세트
  메이크샵/네이버: "압화 꽃잎 10종 세트" → 9,900원
  쿠팡 윙/11번가: "압화 꽃잎 10종 세트 + 핀셋" → 12,800원
  오늘의집: "인테리어 압화 꽃잎 프리미엄 12종 세트" → 15,800원

효과:
  - 가격 비교 사이트에서 "동일 상품"으로 잡히지 않음
  - 채널별 가격 차등을 자연스럽게 정당화
  - 오늘의집은 "프리미엄" 네이밍 + 추가 구성으로 고가 포지셔닝
```

### 2-4. 카테고리별 판매가 전략 매트릭스

PRESSCO21의 주요 상품 카테고리별 가격 전략:

| 카테고리 | 주요 가격대 | COGS 비율 | 권장 채널 우선순위 | 가격 전략 핵심 |
|---------|-----------|----------|-----------------|-------------|
| 압화 재료 | 3,000~15,000 | 30~45% | 자사몰 > 네이버 > 쿠팡 | 세트 구성으로 객단가 상승 |
| 레진 재료 | 5,000~25,000 | 35~50% | 자사몰 > 네이버 > 오늘의집 | 대용량 할인으로 재구매 유도 |
| 캔들 재료 | 8,000~30,000 | 40~55% | 오늘의집 > 자사몰 > 유튜브 | 프리미엄 홈데코 포지셔닝 |
| 석고 재료 | 5,000~20,000 | 35~50% | 네이버 > 쿠팡 > 자사몰 | 무게 기반 배송비 전략 중요 |
| 비즈 재료 | 1,000~10,000 | 25~40% | 네이버 > 쿠팡 > 자사몰 | 소액 다량 구매 패턴. 묶음 가격 |
| 도구류 | 3,000~50,000 | 40~60% | 쿠팡 > 네이버 > 자사몰 | 가격 비교 민감. 경쟁력 유지 필수 |
| 종합 키트 | 20,000~80,000 | 35~50% | 오늘의집 > 유튜브 > 자사몰 | 체험+교육 가치 포함 고가 설정 |

## 3. 광고비 전략 (채널별 ROAS 목표)

### 3-1. 광고비 여력 계산 구조

```
[광고비 여력 계산 공식]

순마진 = 실질 매출 - COGS - 채널 수수료 - 배송비
최소 확보 이익 = 순마진 x 최소 마진 보존율
최대 광고비/건 = 순마진 - 최소 확보 이익
목표 ROAS = 판매가 / 최대 광고비/건

최소 마진 보존율 기준:
  - 보수적: 60% (순마진의 60%는 반드시 확보)
  - 표준: 50% (순마진의 50%를 광고에 투자 가능)
  - 공격적: 30% (성장기 채널, 신규 고객 확보 목적)
```

## 4. 배송비 전략

### 4-1. 무료배송 vs 유료배송 마진 임계점 계산

```
[무료배송 마진 임계점 공식]

무료배송 시 순마진 = 실질매출 - COGS - 채널수수료 - 배송원가
유료배송 시 순마진 = 실질매출 - COGS - 채널수수료

무료배송 "가능" 조건: 무료배송 시 순마진율 >= 최소선

임계 판매가(무료배송 가능 최저가):
  = (COGS + 배송원가) / (1 - 수수료율 - 목표마진율) x 1.1
```

**실제 배송 원가 기준**: 택배비 3,000원 (일반 택배 기준)

| 채널 | COGS 5,000원 상품 | COGS 8,000원 상품 | COGS 15,000원 상품 |
|------|------------------|------------------|-------------------|
| | 무료배송 최저 판매가* | 무료배송 최저 판매가* | 무료배송 최저 판매가* |
| 메이크샵 (수수료 3.3%) | 11,492원 | 15,800원 | 25,851원 |
| 네이버 (수수료 5.7%) | 12,119원 | 16,660원 | 27,260원 |

| 11번가 (수수료 12%) | 13,756원 | 18,910원 | 30,940원 |
| 오늘의집 (수수료 17.5%) | 15,339원 | 21,085원 | 34,497원 |

*순마진율 15%(최소선) 기준으로 역산한 무료배송 가능 최저 판매가

**핵심 시사점**:

### 4-2. 조건부 무료배송 기준 금액 설정

```
[조건부 무료배송 기준 금액 결정 방법]

Step 1. 현재 평균 객단가(AOV) 확인
Step 2. AOV x 1.3 = 조건부 무료배송 기준 금액 후보
Step 3. 기준 금액에서의 평균 마진 검증
Step 4. 기준 금액이 배송비(3,000원)를 흡수할 수 있는지 확인

예시:
  현재 AOV = 18,000원
  기준 금액 후보 = 18,000 x 1.3 = 23,400원 → 25,000원 (라운딩)

  검증:
  - 25,000원 주문 시 평균 순마진 = 약 6,000원 (24%)
  - 배송비 3,000원 부담 후 순마진 = 3,000원 (12%)
  - 12% > 최소선(대부분 채널 10~15%) → 무료배송 가능

  → 결론: 25,000원 이상 무료배송 설정
```

## 5. 마진 경보 및 모니터링 체계

### 5-1. 위험 신호 자동 감지 기준

| 경보 등급 | 조건 | 색상 | 자동 조치 |

**추가 경보 조건**:

|------|-----------|------|

| COGS 상승 경보 | COGS가 직전 분기 대비 10% 이상 상승 | 원가 구조 변화. 판매가 조정 필요 |

| 광고 ROAS 경보 | 실제 ROAS < 목표 ROAS의 70% | 광고 효율 급락. 캠페인 재검토 |

### 5-2. 마진 하락 시 즉각 대응 체크리스트

마진율이 최소선 이하로 하락한 상품이 감지되면 아래 체크리스트를 순서대로 실행한다:

```
[마진 하락 즉각 대응 체크리스트]

--- 1단계: 원인 진단 (당일) ---

[ ] 1-1. 해당 상품의 COGS 변동 확인
    → 원자재 가격 상승? 환율 변동? 운송비 인상?
    → 확인 방법: 원가구성 시트 vs 직전 분기 비교

[ ] 1-2. 채널 수수료 변경 확인
    → 채널에서 수수료율을 인상했는가?
    → 확인 방법: 판매플랫폼 시트 vs 채널 관리자 공지

[ ] 1-3. 배송비 변동 확인
    → 택배사 요금 인상? 무게/부피 기준 변경?
    → 확인 방법: 택배 계약서, 최근 청구서

[ ] 1-4. 판매가 변동 확인
    → 쿠폰/할인 적용? 경쟁 가격 하락으로 인한 가격 인하?
    → 확인 방법: 각 채널 상품 페이지 확인

--- 2단계: 단기 대응 (1주 이내) ---

[ ] 2-1. 판매가 인상 가능 여부 검토
    → 경쟁사 동일/유사 상품 가격 조사 (최소 3곳)
    → 인상 가능 범위: 현 가격 대비 +5~10%

[ ] 2-2. 해당 채널 광고 일시 중단
    → 마진 미확보 상태에서 광고비 집행은 이중 손실
    → ROAS 재계산 후 재개 여부 결정

[ ] 2-3. 무료배송 → 조건부 무료배송 전환 검토
    → 배송비가 마진 하락 주원인인 경우

[ ] 2-4. 세트/묶음 상품 재구성
    → 저마진 상품 + 고마진 상품 묶어서 평균 마진 개선

--- 3단계: 중기 대응 (1개월 이내) ---

[ ] 3-1. COGS 절감 방안 실행
    → 대량 구매 → 단가 할인 협의
    → 대체 조달처 탐색 (국내 vs 중국 vs 직제조)
    → 패키징 간소화 / 용량 조정

[ ] 3-2. 채널 비중 조정
    → 해당 채널 노출 축소, 고마진 채널 노출 확대
    → 예: 오늘의집 마진 하락 → 네이버/자사몰 집중

[ ] 3-3. 상품 단종/리뉴얼 결정
    → 마진 회복 불가 판단 시 단종
    → 구성/디자인 변경 후 신상품으로 재출시

--- 4단계: 기록 ---

[ ] 4-1. 의사결정 로그 기록
    → 구글시트 "마진 의사결정 로그" 탭에 기록
    → 날짜 / 상품 / 원인 / 조치 / 결과 / 담당자
```

### 6-1. 판매가 역산 시뮬레이터 (COGS 5,000원 상품 기준)

**시뮬레이터 시트 구조**:

```
=== 입력 영역 (노란색 셀) ===

[B2] COGS (원가)          : 5,000
[B3] 목표 순마진율 (%)     : 25%
[B4] 배송비 (원)           : 3,000
[B5] 배송비 부담            : 무료배송(셀러 부담) / 착불(구매자 부담)

=== 자동 계산 영역 (회색 셀) ===
```

**8개 채널 동시 비교 결과 테이블**:

목표 순마진율 25% 기준, COGS 5,000원, 배송비 3,000원(무료배송):

| 채널 | 수수료율 | 필요 공급가 | 필요 판매가(VAT) | 라운딩 판매가 | 실제 순마진율 | 순마진(원) | 판정 |
|------|---------|-----------|----------------|------------|------------|----------|------|
| 메이크샵 | 3.3% | 11,156 | 12,271 | **12,800** | 28.5% | 3,321 | EXCELLENT |
| 카페24 유튜브 | 5.0% | 11,429 | 12,571 | **12,800** | 27.2% | 3,164 | HEALTHY |
| 네이버 | 5.7% | 11,547 | 12,702 | **12,800** | 25.6% | 2,978 | HEALTHY |
| 쿠팡 윙 | 10.8% | 12,469 | 13,716 | **13,800** | 25.5% | 3,200 | HEALTHY |
| 쿠팡 로켓그로스 | 10.8%+500 | 12,851 | 14,136 | **14,800** | 29.1% | 3,916 | EXCELLENT |
| 11번가 | 12.0% | 12,698 | 13,968 | **13,900** | 24.3% | 3,068 | HEALTHY |
| 오늘의집 | 17.5% | 13,913 | 15,304 | **15,800** | 25.5% | 3,662 | HEALTHY |

*쿠팡 로켓배송은 별도 계산: 목표 납품가 = COGS x (1 + 목표마진율) = 5,000 x 1.25 = **6,250원** 이상

**역산 수식 상세**:

```
필요 공급가 = (COGS + 배송비) / (1 - 수수료율 - 목표마진율)

[네이버 예시]
  = (5,000 + 3,000) / (1 - 0.057 - 0.25)
  = 8,000 / 0.693
  = 11,547원

필요 판매가(VAT 포함) = 필요 공급가 x 1.1
  = 11,547 x 1.1
  = 12,702원

라운딩 후 검증:
  판매가 = 12,800원
  실질 매출 = 12,800 / 1.1 = 11,636원
  수수료 = 11,636 x 0.057 = 663원
  순마진 = 11,636 - 5,000 - 663 - 3,000 = 2,973원
  순마진율 = 2,973 / 11,636 = 25.5% -- 목표 달성
```

### 6-2. 구글시트 실제 수식 (복사/붙여넣기용)

시뮬레이터 시트의 핵심 수식들:

```
=== 셀 참조 규칙 ===

$B$2 = COGS 입력 셀
$B$3 = 목표 순마진율 입력 셀 (소수점, 예: 0.25)
$B$4 = 배송비 입력 셀
판매플랫폼!$H$2:$H$9 = 채널별 최종적용수수료율

=== 채널별 필요 공급가 ===
(열 C, 행 8~15)
=($B$2+$B$4)/(1-VLOOKUP(A8,판매플랫폼!$A:$H,8,0)-$B$3)

=== 필요 판매가(VAT포함) ===
(열 D)
=C8*1.1

=== 라운딩 판매가 ===
(열 E) - 수동 입력 또는 자동 올림
=CEILING(D8, 100)

=== 실제 순마진율 검증 ===
(열 F)
=(E8/1.1-$B$2-E8/1.1*VLOOKUP(A8,판매플랫폼!$A:$H,8,0)-$B$4)/(E8/1.1)

=== 순마진 금액 ===
(열 G)
=E8/1.1-$B$2-E8/1.1*VLOOKUP(A8,판매플랫폼!$A:$H,8,0)-$B$4

=== 마진 판정 ===
(열 H)
=IF(F8>=0.3,"EXCELLENT",IF(F8>=0.2,"HEALTHY",IF(F8>=0.15,"CAUTION",IF(F8>=0.05,"WARNING","CRITICAL"))))

=== 조건부 서식 ===
EXCELLENT: 배경 #C6EFCE (진녹)
HEALTHY:   배경 #E2EFDA (연녹)
CAUTION:   배경 #FFEB9C (노랑)
WARNING:   배경 #FFC7CE (주황)
CRITICAL:  배경 #FF6B6B (빨강)
```

### A-3. 계절별 마진 조정 가이드

```
[분기별 마진 전략]

Q1 (1~3월): 봄 준비 시즌
  - 압화 재료 선주문 물량 확보 (COGS 절감 기회)
  - 봄 신상품 출시 시 초기 마진 30% 이상 설정
  - 겨울 캔들 재료 시즌오프 → 할인 판매 (마진 15%까지 허용)

Q2 (4~6월): 압화 피크 시즌
  - 압화 재료 수요 최고 → 정가 판매 유지, 할인 자제
  - 어버이날/스승의날 시즌 키트 출시 → 프리미엄 가격
  - 광고비 적극 투입 (시즌 수요에 올라타기)

Q3 (7~9월): 비수기
  - 레진/비즈 (계절 무관) 카테고리 집중 프로모션
  - 재고 정리 세일 → 마진 10%까지 한시적 허용
  - 가을/겨울 상품 기획 및 원가 확정

Q4 (10~12월): 캔들/석고 피크 + 연말 시즌
  - 캔들/석고 재료 수요 증가 → 정가 판매
  - 크리스마스 종합 키트 출시 → 최고 마진 상품
  - 연말 재고 정리 + 신년 물량 계획
```
