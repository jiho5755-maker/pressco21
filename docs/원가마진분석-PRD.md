## 1. 개요

### 1.1 배경

PRESSCO21은 8개 판매 채널에서 꽃 공예 재료를 판매하며, 조달 경로 3가지(국내 직제조/중국 사입/국내 사입)를 운영 중이다. 체계적인 원가·마진 관리 시스템을 구축하여 각 채널별 실질 수익과 광고비 여력을 정확히 파악하고 전략적 의사결정에 활용한다.

### 1.2 목표

- 조달 유형별 COGS 정확 산출 체계 구축
- 8개 채널별 실질 순마진 자동 계산 (구글시트 VLOOKUP 연동)
- 판매가 시뮬레이터 → 채널별 목표 판매가 역산
- 광고비 여력 계산 → ROAS 목표 설정
- 채널 전략 수립 → 자사몰 비중 확대 로드맵

### 1.3 범위

에이전트 개발 → 구글시트 데이터 인프라 → 운영 프로세스 → 전략 도구까지 전체 설계

## 2. 핵심 비즈니스 규칙

### 2.1 판매가 = 부가세 포함가 (VAT 포함)

모든 채널의 판매가는 VAT 포함가로 설정됨. 마진 계산 시:

```
실질 매출(공급가) = 판매가(VAT포함) / 1.1
부가세 = 판매가 - 실질 매출
순마진 = 실질 매출 - COGS - 채널 수수료 - 배송비
순마진율 = 순마진 / 실질 매출
```

### 2.2 쿠팡 로켓배송 = 매입가 협의 방식 (특수 구조)

쿠팡이 매입가(납품가)를 제시/협의하여 매입. 수수료 차감이 아닌 별도 마진 구조:

```
로켓배송 마진 = 쿠팡 납품가(매입가) - COGS
(판매가는 쿠팡이 결정, 셀러 관여 불가)
```

### 2.3 조달 유형별 COGS 계산

```
[직제조] COGS = 원재료비 + 직접노무비 + 제조간접비 + 품질검사비 + 포장재비
[중국사입] COGS = 위안화매입가 × CNY환율 × (1 + 지사관리비율10%) + 해상운송비/개당 + 관세 + 부대비용
[국내사입] COGS = 매입단가 + 매입부대비용(운송비/검수비)
```

### 2.4 마진 등급 5단계

| 등급 | 기준 | 의미 |
|------|------|------|
| EXCELLENT | ≥ 30% | 광고/프로모션 여력 충분 |
| HEALTHY | 15%~30% | 정상 운영 가능 |
| CAUTION | 5%~15% | 가격 재검토 필요 |
| WARNING | 0%~5% | 즉시 조치 필요 |
| CRITICAL | < 0% | 판매 중단 검토 |

### 2.5 향후 고도화 예정 (이번엔 미반영)

- 프로모션/할인 비용 반영 마진
- 재고 회전율 / 체화재고 비용
- 반품/클레임 비용
- 채널별 정산 주기 관리

## 3. 판매 채널 (8개)

| # | 채널 | 유형 | 역할 | 마진 계산 방식 | 수수료(일반 기준) | 마진 최소/목표 |
|---|------|------|------|-------------|----------------|-------------|
| 1 | 메이크샵(자사몰) | 자사몰 | **수익형** | 판매가/1.1 - COGS - PG수수료 | PG ~3.3% | 최소20%/목표30% |
| 2 | 쿠팡 윙 | 셀러직배송 | **볼륨형** | 판매가/1.1 - COGS - 수수료 | ~10.8% | 최소10%/목표18% |
| 3 | 쿠팡 로켓배송 | **매입형** | **볼륨형** | **납품가 - COGS** (판매가 무관) | 납품가 협의 | 납품가 협의 |
| 4 | 쿠팡 로켓그로스 | 풀필먼트 | **볼륨형** | 판매가/1.1 - COGS - 수수료 - 풀필먼트비 | ~10.8% + 풀필먼트 | 최소8%/목표15% |
| 5 | 네이버 스마트스토어 | 오픈마켓 | **트래픽형** | 판매가/1.1 - COGS - 수수료 | ~5.7% | 최소15%/목표25% |
| 6 | 11번가 | 오픈마켓 | **볼륨형** | 판매가/1.1 - COGS - 수수료 | ~12% | 최소8%/목표15% |
| 7 | 오늘의집 | 버티컬 | **프리미엄형** | 판매가/1.1 - COGS - 수수료 | ~15~20% | 최소15%/목표25% |
| 8 | 카페24 유튜브쇼핑 | 소셜커머스 | **브랜딩형** | 판매가/1.1 - COGS - 수수료 | 카페24+유튜브 ~4~6% | 최소18%/목표28% |

> ⚠️ 위 수수료율은 일반 기준. 실제 계약 수수료로 판매플랫폼 시트 업데이트 필요.

### 채널 혼합 판매가 전략

```
메이크샵(자사몰) = 기준가 (Base Price)
쿠팡 윙       = 기준가 × 1.05~1.08 (+5~8%)
오늘의집      = 기준가 × 1.10~1.15 (+10~15%)
네이버/11번가  = 기준가 ± 2% (동일 또는 소폭 조정)
카페24 유튜브  = 기준가 × 0.95~1.00 (유튜브 구독자 할인 가능)
```

### 쿠팡 의존도 경보

- **주의**: 쿠팡 3채널 합산 매출 비중 40% 초과
- **경보**: 50% 초과 → 다채널 분산 전략 강화
- **긴급**: 60% 초과 → 즉시 채널 재배분 계획 수립

| I | 비고 | 계약일, 특이사항 |

##### ④ 플랫폼별마진 시트 (핵심) ✅

행: 제품코드 + 채널코드 조합

| 열 | 필드명 | 수식 |
|----|--------|------|
| A | 제품코드 | |
| B | 채널코드 | |
| C | 채널명 | =VLOOKUP(B, 판매플랫폼!A:B, 2, 0) |
| D | 판매가(VAT포함) | =VLOOKUP(A, 제품마스터!A:F, 6, 0) |
| E | 실질매출(공급가) | =D/1.1 |
| F | COGS | =VLOOKUP(A, 제품마스터!A:G, 7, 0) |
| G | 수수료율 | =VLOOKUP(B, 판매플랫폼!A:H, 8, 0) |
| H | 채널수수료 | =E*G |
| I | 배송비 | 수동 입력 |
| J | 순마진 | =E-F-H-I |
| K | 순마진율 | =J/E |
| L | 마진등급 | =IF(K>=0.3,"EXCELLENT",IF(K>=0.15,"HEALTHY",IF(K>=0.05,"CAUTION",IF(K>=0,"WARNING","CRITICAL")))) |

| 열 | 필드명 | 수식 |
|----|--------|------|

| 경보 단계 | 변동폭 | 대응 |

##### ⑥ 대시보드 시트 ✅

- 채널별 평균 순마진율 (AVERAGEIF 연동)
- 마진 등급별 상품 수 (COUNTIF 연동)
- 위험 마진 상품 목록 헤더 (WARNING/CRITICAL 필터링용)
- 월별 마진 트렌드 (향후 데이터 축적 후 차트 추가)

##### ⑦ 시뮬레이터 시트 ✅

```
[입력값]
COGS (원)          : 수동 입력 (기본값 10,000)
목표 순마진율 (%)   : 수동 입력 (기본값 25%)
배송비 (원)         : 수동 입력 (기본값 3,000)

[자동 계산 - 채널별]
채널명 | 수수료율 | 필요공급가 | 필요판매가(VAT) | 라운딩(100원) | 실제마진율 | 실제순마진 | 판정(5단계)

[쿠팡 로켓배송 별도 계산]
납품가 입력 → 마진/마진율 자동 계산
```

## 6. 광고비 여력 (채널별 ROAS 목표)

```
광고 여력(원) = 순마진 × 광고비 할당 비율
최대 광고비 = 광고 여력 / (1 - 목표 광고비율)
ROAS 목표 = 판매가 / 최대 광고비
```

| 채널 | ROAS 목표 | 광고비 여력 | 비고 |

| 네이버 스마트스토어 | 6.5 | 마진의 12~18% | 네이버 쇼핑 광고 |
| 쿠팡 윙 | 7.5 | 마진의 10~15% | 로켓배송 노출 경쟁 |
| 11번가 | 8.0 | 마진의 8~12% | 수익성 낮아 광고 최소화 |
