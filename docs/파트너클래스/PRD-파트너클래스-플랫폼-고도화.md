## 1. 프로젝트 핵심

**목적**: 꽃공예 강사(파트너)가 수업 운영에만 집중할 수 있도록, 일정 관리부터 재료 자동 배송, 수강생 성장 파이프라인까지 플랫폼이 모든 운영 부담을 대신 처리한다.

**사용자**: 꽃공예 자격증 보유 강사(파트너), 취미 수강생, PRESSCO21 관리자 — 총 4등급 회원 체계(인스트럭터/파트너스/마스터/엠버서더)를 중심으로 재료 구매 선순환을 구동한다.

## 3. 사용자 여정

### 3-1. 수강생 여정 (신규 일정 선택 포함)

```
1. 클래스 목록 페이지 (id=2606)
   ↓ 카테고리/지역/형태 필터 적용

2. 클래스 상세 페이지 (id=2607)
   ↓ 일정 캘린더에서 가능한 날짜 선택 (Phase 3-A 신규)

   [잔여석 있음] → 인원 선택 → 예약하기 버튼
   [잔여석 없음] → "마감" 표시, 다른 일정 선택 유도
   ↓

3. 로그인 확인
   [비로그인] → 메이크샵 로그인 페이지 → 로그인 후 복귀
   [로그인됨] → 메이크샵 결제 페이지 (shopdetail.html?branduid=XXX)
   ↓

4. 결제 완료
   ↓ WF-05 폴링(5분) 감지 → 예약 확정 처리

   → 수강생: 확인 이메일 발송 (WF-12 실가동)
   → 파트너: 새 예약 알림 이메일 발송
   → 키트 배송: 재료 주문 자동 생성 (Phase 3-A 신규)
   ↓

5. 수강 D-3 / D-1 리마인더 (WF-11 실가동)
   ↓

6. 수강 완료 후 7일
   → 수강생: 후기 작성 요청 이메일
   → 수료증 발급 (Phase 3-C 신규)

   [수료증 보유] → 파트너 신청 자격 표시 ("강사 도전하기" CTA)
```

### 3-2. 파트너 여정 (강의 등록 ~ 정산)

```
1. 파트너 신청 페이지 (id=2609)
   ↓ 자격증/사업자 서류 제출 → WF-07 접수

   [관리자 승인] → WF-08 → SILVER 등급 + 교육 이수 안내 이메일
   [거부] → 사유 안내 이메일
   ↓

2. 교육 페이지 (id=2610)
   ↓ 필수 교육 영상 시청 + 퀴즈 → WF-10 이수 처리
   ↓

3. 강의 등록 페이지 (id=8009)
   ↓ 강의 정보 + 재료 목록 입력 (Phase 3-A 신규: 재료키트 필드)
   → WF-16 → 관리자 승인 대기

   [관리자 승인] → WF-17 → 메이크샵 자동 등록 → 클래스 목록 노출
   ↓

4. 파트너 대시보드 (id=2608)
   ↓ 강의 수정 (Phase 3-A 신규: WF-20)
   ↓ 일정 관리 (Phase 3-A 신규: tbl_Schedules)
   ↓ 예약 현황 확인
   ↓

5. 정산
   → WF-05가 결제 주문 감지 → 수수료 계산 → 적립금 지급 API 호출
   → tbl_Settlements 기록 (PENDING → COMPLETED)
   → 파트너 대시보드에서 적립금 잔액 + 이달 매출 확인
```

### 3-3. 관리자 여정 (Phase 3-B 신규)

```
1. 관리자 어드민 페이지 (신규 메이크샵 페이지)
   ├── 파트너 신청 목록 → 승인/거부
   ├── 강의 승인 목록 → 승인/거부
   ├── 정산 현황 → 수동 처리
   └── 협회 회원 관리 → 회원-협회 매핑
```

## 6. 페이지별 상세 기능

### 클래스 목록 페이지 (id=2606)

> **구현 기능:** `F301`, `F304` | **메뉴 위치:** 헤더 진입점, 메인페이지 배너 클릭

| 항목 | 내용 |
|------|------|
| **역할** | 수강생이 수업을 탐색하는 카탈로그 — 필터/정렬/탭으로 원하는 강의를 발견하는 진입 허브 |
| **진입 경로** | 메인페이지 인기 클래스 배너 클릭, 헤더 메뉴 직접 접근 (`/shop/page.html?id=2606`) |
| **사용자 행동** | 카테고리/지역/형태(원데이/정기/온라인) 필터 적용, "협회 세미나/행사" 탭 전환, 클래스 카드 클릭 → 상세 이동 |
| **주요 기능** | • 클래스 카드에 잔여석 뱃지 표시 — tbl_Schedules에서 min(잔여석) 계산 (`F301`)<br>• "마감" 처리된 클래스는 목록 하단 배치 (`F301`)<br>• **협회 세미나/행사 탭** — tbl_Affiliations 데이터, 협회 로고 + 공식 배지 표시 (`F304`)<br>• 협회 탭 클릭 시 예약 → PRESSCO21 통해 예약하면 추가 적립금 안내 (`F304`)<br>• 찜(관심) + 최근 본 클래스 (Phase 2.7 기존 기능 유지) |
| **다음 이동** | 클래스 카드 클릭 → 클래스 상세 페이지 (`?id=2607&class_id=XXX`) |
