# 목록 배지 + 퀵 필터 가이드

작성일: 2026-03-10

## 목적

S1-3는 목록 페이지를 단순 카드 모음이 아니라 `빠르게 믿고 고르는 탐색 화면`으로 바꾸는 작업이다.

- 수강생은 카드만 봐도 왜 눈여겨볼 만한 클래스인지 이해해야 한다.
- `카테고리 -> 지역 -> 형태 -> 가격대 -> 난이도` 흐름으로 빠르게 좁혀갈 수 있어야 한다.
- 오프라인 클래스는 리스트 안에서 바로 지도 탐색으로 이어져야 한다.

## 반영 범위

- `파트너클래스/목록/Index.html`
- `파트너클래스/목록/css.css`
- `파트너클래스/목록/js.js`

## 신뢰 배지 6종

카드 상단 제목 아래에 최대 3개까지 노출한다.

| 배지 | 기준 |
| --- | --- |
| 신규 | `class_id`의 `CL_YYYYMM_` 기준 현재월 또는 전월 |
| 인기 | 후기수 8건 이상 또는 평점 4.7 이상 + 후기수 5건 이상 |
| 마감임박 | `total_remaining` 1~3석 |
| 키트포함 | `kit_enabled = 1` |
| 협회제휴 | `affiliation_code`, `tags`, 클래스명 등에 `협회` 또는 `제휴` 텍스트 포함 |
| 높은평점 | 평점 4.8 이상 + 후기수 3건 이상 |

현재는 WF-01 응답만으로 계산하고, 별도 badge 전용 필드는 추가하지 않았다.

## 퀵 필터 구조

목록 상단에 5개 그룹을 둔다.

- 카테고리: `압화`, `보존화`, `캔들`, `리스`
- 지역: `서울`, `경기`, `부산`, `대구`
- 형태: `오프라인`, `온라인`
- 가격대: `5만원 이하`, `10만원 이하`
- 난이도: `입문`, `중급`, `심화`

형태 필터는 `오프라인 -> 원데이 + 정기`, `온라인 -> 온라인`으로 매핑한다.

## 상태 저장

- localStorage 키: `pressco21_catalog_filters_v1`
- URL 파라미터가 있으면 URL 우선
- URL 파라미터가 없으면 localStorage 상태 복원
- 저장 항목:
  - `category`
  - `level`
  - `type`
  - `format`
  - `region`
  - `sort`
  - `maxPrice`
  - `search`

## 지도 진입점

- 오프라인 클래스만 카드 하단에 `가까운 공방 보기` CTA를 노출한다.
- 링크는 `partner_name + location/region` 기준 Google Maps 검색 URL을 생성한다.
- 온라인 클래스는 지도 CTA를 노출하지 않는다.

정식 파트너맵 통합은 S2-3에서 다루고, S1-3은 목록 탐색의 즉시성 확보를 우선한다.

## 숨은 버그 수정

S1-3 정리 중 목록 페이지에 있던 기존 충돌도 함께 수정했다.

- 협회 탭 전용 금액 포맷 함수가 카드 가격 포맷 함수를 덮어쓰던 문제를 분리했다.
- 배지 계산에서 쓰는 `normalizedContains()` 누락으로 런타임 에러가 날 수 있던 문제를 보완했다.
- 최근 본 클래스 추적에서 외부 지도 링크 클릭은 제외했다.

## 검증

정적 검증:

- `node --check 파트너클래스/목록/js.js`
- `python3 ~/.codex/skills/makeshop-d4-dev/scripts/check_makeshop_d4.py 파트너클래스/목록/Index.html 파트너클래스/목록/css.css 파트너클래스/목록/js.js`

메모:

- D4 가드의 `http://` 경고는 SVG namespace 문자열(`xmlns`) 기준 false positive다.

Playwright 모킹 검증:

- 초기 로딩: `총 4개의 클래스`, 배지 6종 전체 확인
- 필터 조합: `서울 + 오프라인 + 입문 + 5만원 이하` -> `서울 압화 원데이 클래스` 1건
- 새로고침 복원: 같은 조합 유지 확인
- 온라인 전환: `서울 온라인 플로럴 세미나` 1건, 지도 CTA 비노출 확인
