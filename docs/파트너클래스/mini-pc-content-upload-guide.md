# PRESSCO21 자료 업로드 가이드

작성일: 2026-03-18

이 문서는 직원들이 `수동 업로드`를 해야 할 때 지켜야 할 최소 규칙만 정리한다.

주의:

- 영상 편집자와 웹디자이너의 기본 경로는 `자동 동기화`다.
- 이 문서는 모바일 업로드, 단발성 파일, 예외 상황에만 쓴다.

## 핵심 원칙

1. 자료는 항상 `inbox` 에만 올린다.
2. `youtube`, `reels`, `shared` 최종 폴더는 직접 만지지 않는다.
3. 업로드 후 바로 찾지 말고 `20분~1시간` 정도 기다린다.
4. 파일이 안 보이면 먼저 `quarantine` 폴더를 확인한다.

## 접속 방법

### 사무실/집에서 같은 네트워크일 때

- `http://192.168.0.54:8090/login`

### 외부에서 접속할 때

1. `Tailscale` 앱 설치
2. 회사에서 지정한 같은 계정으로 로그인
3. 아래 주소로 접속

- `http://100.76.25.105:8090/login`
- 또는 `http://pressco21-backup.tailee581a.ts.net:8090/login`

## 로그인 정보

- 직원은 `upload-only` 계정만 사용한다.
- 관리자 계정은 관리자만 사용한다.
- 아이디와 비밀번호는 관리자에게 직접 받는다.
- 비밀번호를 메신저방에 고정하지 않는다.

## 어디에 올려야 하는지

### 유튜브 관련 예외 업로드

- 촬영 원본: `inbox/youtube/raw`
- 편집 프로젝트: `inbox/youtube/project`
- 업로드 직전 완성본: `inbox/youtube/export`

### 릴스 관련 예외 업로드

- 촬영 원본: `inbox/reels/raw`
- 편집 프로젝트: `inbox/reels/project`
- 업로드 직전 완성본: `inbox/reels/export`

### 공용 자료 예외 업로드

- 로고, 폰트, 템플릿: `inbox/shared/brand-assets`
- 자막 파일: `inbox/shared/subtitles`
- 썸네일 파일: `inbox/shared/thumbnails`

## 업로드 순서

1. 브라우저에서 로그인한다.
2. 왼쪽 폴더 목록에서 `inbox` 를 연다.
3. 맞는 폴더까지 들어간다.
4. `Upload` 버튼을 누르거나 파일을 끌어다 놓는다.
5. 업로드가 끝나면 브라우저를 닫아도 된다.

## 업로드 후 무슨 일이 생기나

시스템이 일정 시간 뒤 자동으로 파일을 정리한다.

예:

- `inbox/youtube/raw/interview-01.mp4`
- 자동 이동 후:
  - `youtube/raw/2026/2026-03/interview-01.mp4`

즉 `inbox` 는 임시 투입구라고 생각하면 된다.

## 파일명 권장 규칙

가능하면 다음처럼 올린다.

- `2026-03-브랜드명-주제-원본-01.mp4`
- `2026-03-브랜드명-릴스-최종-v1.mp4`
- `2026-03-브랜드명-썸네일-01.png`

좋은 파일명 예시:

- 날짜가 들어간다.
- 브랜드명 또는 프로젝트명이 들어간다.
- `원본`, `최종`, `자막`, `썸네일` 같은 성격이 보인다.

## 하면 안 되는 것

- 최종 폴더에 직접 업로드하기
- 같은 파일을 여러 폴더에 중복 업로드하기
- 자동 동기화 대상 작업 파일을 다시 `inbox` 로 중복 업로드하기
- 회사 자료와 개인 자료를 같은 이름 규칙 없이 섞어 올리기
- `backup` 관련 폴더를 건드리기

## 파일이 사라진 것처럼 보일 때

1. 먼저 `inbox` 에서 사라졌는지 본다.
2. 최종 폴더를 본다.
3. 없으면 `quarantine` 을 본다.
4. 그래도 없으면 관리자에게 파일명과 업로드 시간을 전달한다.

## 관리자에게 전달할 때 필요한 정보

문제가 생기면 아래 3가지를 같이 보낸다.

1. 파일명
2. 어떤 폴더에 올렸는지
3. 대략 몇 시에 올렸는지

이 3개가 있어야 빠르게 찾을 수 있다.
