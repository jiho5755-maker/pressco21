# PRESSCO21 x 꽃공예 협회 파트너십 제안서

> **PRESSCO21** | foreverlove.co.kr
> "Forever and ever and Blooming"
> 제안일: 2026년 3월

---

## 1. PRESSCO21은 이런 회사입니다

PRESSCO21은 **1982년부터 시작된 꽃공예 연구를 바탕으로, 1999년 창업 이후 현재까지 27년간** 압화(프레스드플라워) 및 꽃 공예 재료 전문 기업으로 성장해온 업계 선두 업체입니다.

| 항목 | 내용 |
|------|------|
| **사업 분야** | 압화/플로리스트/레진/캔들 등 꽃 공예 재료 제조 및 유통 |
| **자사몰** | foreverlove.co.kr (메이크샵 기반) |
| **회원 규모** | 총 약 4,000명 (그중 강사 회원 약 1,200명) |
| **업계 위치** | 꽃 공예 재료 분야 국내 선두 업체 |
| **핵심 역량** | 1982년부터 이어온 압화 연구 (40년 이상), 다수 특허 보유, 방송/전시/국제 교류 경력 |

PRESSCO21은 재료를 판매하는 데 그치지 않고, **"꽃 공예를 하는 모든 분이 더 쉽고 즐겁게 활동할 수 있는 환경"**을 만들고자 합니다. 이번 제안은 그 철학의 연장선에 있습니다.

---

## 2. 파트너클래스 플랫폼을 소개합니다

PRESSCO21은 2026년, 꽃 공예 강사와 공방을 위한 **클래스 예약 플랫폼**을 자체 개발하여 운영 중입니다.

### 플랫폼이 하는 일

```
강사가 클래스를 등록한다
    --> 수강생이 플랫폼에서 검색하고 예약/결제한다
    --> 수강료가 자동 정산된다
    --> 정산금은 적립금으로 전환되어 재료 구매에 사용할 수 있다
```

### 주요 기능

| 기능 | 설명 |
|------|------|
| **클래스 목록/상세** | 전국의 꽃 공예 클래스를 지역, 분야, 가격별로 검색 |
| **온라인 예약 결제** | 수강생이 직접 예약하고 결제까지 원스톱 완료 |
| **파트너 대시보드** | 강사 전용 페이지에서 예약 현황, 수익, 리뷰를 한눈에 확인 |
| **자동 정산** | 수강 완료 후 수수료를 제외한 금액이 자동 정산 |
| **이메일/알림 자동화** | 예약 확인, 리마인더, 후기 요청까지 자동 발송 |
| **수강 후기 시스템** | 수강생 후기로 신뢰도 축적, 강사 프로필 강화 |

### 강사(파트너)가 얻는 것

- 별도 홈페이지 없이도 **전문적인 클래스 페이지** 보유
- PRESSCO21의 기존 회원 4,000명에게 **자연스러운 홍보** 효과
- 수강료 정산금을 **적립금으로 전환**하여 재료를 할인 구매
- 강사 등급(SILVER / GOLD / PLATINUM)에 따른 **차등 혜택**

## 3. 협회와 함께하면 이렇게 좋습니다

본 제안의 핵심은 **"PRESSCO21과 협회가 함께 협회원의 활동을 지원하는 것"**입니다.
협회는 별도의 비용 부담 없이, 협회원에게 실질적인 혜택을 제공할 수 있습니다.

---

### 제안 1: 협회원 전용 할인 쿠폰 제공

PRESSCO21 자사몰에서 사용할 수 있는 **재료 구매 할인 쿠폰**을 협회원에게 제공합니다.

| 항목 | 내용 |
|------|------|
| **할인율** | 10~20% (협회별 협의) |
| **적용 범위** | PRESSCO21 자사몰 전 상품 또는 특정 카테고리 |
| **제공 방식** | 협회 전용 쿠폰 코드 발급 / 협회 채널을 통해 배포 |
| **갱신 주기** | 연간 계약, 반기별 조건 검토 |

**협회가 얻는 것:**
- 협회원에게 즉시 체감 가능한 혜택 제공
- 재료비 부담 경감으로 협회 활동 참여율 향상
- "우리 협회에 가입하면 이런 혜택이 있다"는 가입 유인 효과

---

### 제안 2: 협회 공동 적립금 풀 (연간 인센티브)

협회원들의 연간 누적 구매액이 기준을 달성하면, **협회 자체에 인센티브 적립금을 지급**합니다.

| 연간 누적 구매액 (협회원 합산) | 협회 인센티브 적립금 |
|-------------------------------|-------------------|
| 500만원 이상 | 25만원 |
| 1,000만원 이상 | 50만원 |
| 2,000만원 이상 | 120만원 |

> 위 기준은 예시이며, 협회 규모와 상황에 따라 조정 가능합니다.

**적립금 활용 방안:**
- 협회 세미나/워크숍 재료비
- 협회 행사 운영비
- 회원 대상 경품/상품
- 협회 전시회 부스 재료

**협회가 얻는 것:**
- 별도의 예산 부담 없이 행사 재료비 확보
- 협회원의 구매 활동이 곧 협회의 자산이 되는 구조
- 협회장이 협회 운영에 활용할 수 있는 실질적 재원

---

### 제안 3: 클래스 플랫폼 홍보 채널

협회의 세미나, 워크숍, 행사 일정을 PRESSCO21 파트너클래스 플랫폼에 **무료로 게시**합니다.

| 항목 | 내용 |
|------|------|
| **게시 비용** | 무료 |
| **게시 형태** | 클래스 목록에 협회 공식 배지 + 로고와 함께 노출 |
| **예약 연동** | PRESSCO21 플랫폼을 통한 예약 시 수강생에게 추가 적립금 제공 |
| **홍보 범위** | PRESSCO21 회원 4,000명 + 자사몰 방문자 대상 |

**협회가 얻는 것:**
- 협회 행사를 PRESSCO21의 기존 고객층에 노출
- 협회 로고가 담긴 공식 인증 배지로 브랜드 신뢰도 향상
- 온라인 예약/결제 인프라를 별도 구축 비용 없이 활용

---

### 제안 4: 파트너스 등급 우선 부여

해당 협회의 **수료증 또는 자격증을 보유한 강사**에게 PRESSCO21 파트너 등급을 우선 부여합니다.

| 항목 | 일반 강사 | 협회 인증 강사 |
|------|----------|--------------|
| **파트너 심사** | 개별 심사 (1~3 영업일) | 협회 인증으로 간소화 심사 |
| **초기 등급** | SILVER (기본) | SILVER + 추가 혜택 (협의) |
| **재료 할인** | 기본 등급 할인 | 협회원 전용 할인 중복 적용 가능 |
| **플랫폼 노출** | 일반 노출 | 협회 인증 배지 부착, 우선 노출 검토 |

**협회가 얻는 것:**
- 협회 자격증/수료증의 실질적 가치 강화
- "우리 협회 자격증이 있으면 PRESSCO21 파트너 혜택을 받는다"는 차별화
- 협회원의 수입원 다각화 지원

## 4. 제안 조건 요약

| 항목 | 내용 |
|------|------|
| **계약 형태** | 업무 협약 (MOU) |
| **계약 기간** | 1년 단위, 자동 갱신 (쌍방 1개월 전 해지 통보) |
| **협회 비용** | 없음 (PRESSCO21이 플랫폼 운영비 전액 부담) |
| **할인 조건** | 협회별 개별 협의 (10~20% 범위) |
| **인센티브 기준** | 협회 규모에 따라 맞춤 설계 |
| **데이터 관리** | 협회원 개인정보는 PRESSCO21 개인정보 처리방침에 따라 안전하게 관리 |
| **중도 해지 시** | 기발급 쿠폰은 유효기간까지 사용 가능, 미지급 인센티브는 협의 후 정산 |
