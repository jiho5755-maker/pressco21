# MakeShop 가격 데이터 정비 플레이북

작성일: 2026-04-21  
작업 브랜치: `work/homepage/makeshop-price-display`

## 목적

가격표시 이슈를 프론트 예외처리 확대가 아니라 **상품 데이터 정합성 정비**로 해결하기 위한 운영 기준 문서입니다.
핵심 원칙은 아래 두 가지입니다.

1. **데이터 정비가 본체**다.
2. **프론트 JS는 이행기간용 최소 안전장치만 유지**한다.

## 현재 확인된 사실

- 프론트에는 이미 회원가 fallback 정책맵이 대량 내장되어 있습니다.
  - 메인: `PC21_MAIN_MEMBER_PRICES` 42개
  - 목록/검색/상세 공용 fallback: `PC21_MEMBER_PRICE_DATA` 500개
  - 합집합 기준 현재 **515개 SKU가 프론트 보정 후보**입니다.
- 이 구조는 단기 응급처치에는 유효하지만, 페이지별 drift가 생기기 쉬워 근본 해결이 아닙니다.
- 실제 라이브 강사회원 세션에서도 동일/유사 상품군이 서로 다른 할인율로 공존하는 사례가 확인되었습니다.

## 최종 원칙

### 1. 운영 데이터 원칙
- 소비자가(`price_consumer`)와 강사가/회원가(`usergroup_price`)를 운영의 단일 기준값으로 삼습니다.
- `price_sell`은 일반 판매가 역할로 정리하되, 회원가와 소비자가 관계가 항상 설명 가능해야 합니다.
- 동일 코드/유사명 상품군은 **대표 정책 그룹**으로 묶어서 관리합니다.
- 같은 정책 그룹이면 최소한 아래 값은 동일해야 합니다.
  - 소비자가
  - 회원가
  - 의도 할인율
  - 노출 대상 그룹

### 2. 프론트 유지 원칙
- 프론트에서는 **보이는 숫자 기준 할인율 재계산**은 유지합니다.
- 기준가=판매가면 할인 배지/취소선은 숨깁니다.
- 회원가 fallback은 **회원 컨텍스트가 확실할 때만** 적용합니다.
- fallback이 개입한 SKU는 `data-*` 출처 태깅을 유지해 운영 감사가 가능해야 합니다.

## 데이터 정비 우선순위

### P0 — 즉시 정비
- 같은 상품코드/유사명이 서로 다른 회원가/할인율로 공존하는 상품
- 메인/브랜드/검색/상세 중 한 화면이라도 값이 다르게 보이는 상품
- 강사회원 문의 빈도가 높은 카네이션/보존화 계열

### P1 — 단기 정비
- 프론트 fallback 정책맵에만 존재하는 SKU 515개
- 모바일 홈/검색/목록에 자주 노출되는 베스트셀러
- 행사/옵션 제외 시 표준가격화 가능한 상품

### P2 — 중기 정비
- 옵션/세트/용량 차이로 같은 이름처럼 보이지만 실제 정책이 다른 상품군
- 장기 미판매/숨김/중복 레코드 정리

## 운영 실행 순서

1. **대표 정책 그룹 정의**
   - 예: `02380-192 모부핑크`, `PR830 미뇽카네이션`, `카네이션잎/진빨강`
2. **현재 SKU 수집**
   - branduid, 상품명, 코드, 소비자가, 판매가, 회원가, 할인율, 노출화면
3. **정책 기준 확정**
   - 어떤 값이 정답인지 MD/운영 기준으로 확정
4. **MakeShop 관리자 데이터 수정**
   - 동일 그룹 SKU를 한 번에 정비
5. **화면 교차검수**
   - 비로그인 / 강사회원 로그인
   - 메인 / 브랜드 / 검색 / 상세 / 가능하면 장바구니
6. **프론트 fallback 감축**
   - 데이터가 정상화된 SKU부터 JS 정책맵에서 제거 후보로 이동

## 관리자 수정 시 필수 체크 컬럼

- `branduid`
- `대표 상품코드`
- `현재 상품명`
- `price_sell`
- `price_consumer`
- `usergroup_price`
- `의도 할인율`
- `정책 그룹`
- `중복/통합 대상 여부`
- `비고(옵션/행사/예외)`

## 승인 게이트

### Gate A — 정책 확정
- 대표 코드 기준값 확정
- 고객에게 불리한 수정 금지

### Gate B — 데이터 수정
- 동일 그룹 SKU 일괄 정비
- 행사/옵션 예외는 별도 표시

### Gate C — 화면 검수
- 비로그인 / 강사회원 로그인 교차 확인
- 메인/브랜드/검색/상세 값 일치 확인
