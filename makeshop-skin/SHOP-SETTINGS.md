## 2. 결제 설정

### 2-1. 주문서 버전

| 항목 | 현재 설정 |
|------|----------|
| 쇼핑몰 주문서 버전 | **고급형 주문서** (선택됨) |

> **고급형 vs 일반형 주요 차이:**
>
> | 항목 | 고급형 | 일반형 |
> |------|-------|-------|
> | 주문 흐름 | 주문서에서 결제방법까지 선택 후 주문 | 주문하기 버튼 → 결제팝업 |
> | 결제방법 노출 | PG사 결제수단 (은행에스크로 제외) | PG사 결제수단 + 은행에스크로 |
> | 적립금 즉시할인 | 불가 | 가능 |
> | 혜택 초기화 | 주소/배송 변경 시 자동 초기화 | - |

### 2-2. 지원 결제수단

| 구분 | 결제수단 |
|------|---------|
| 카드결제만 가능 | 신용카드, 휴대폰 결제, 카카오페이, 페이코, 삼성페이, 쓱페이, 해외결제 페이팔, 내통장결제 |
| 무통장입금만 가능 | 무통장입금, 실시간 계좌이체, 에스크로 |

### 2-3. 결제 설정 하위 메뉴 구조

```
결제 설정
├── 전자결제(PG) 신청/관리  ← PG사 계약 현황
├── 바로 구매              ← 상품 상세에서 즉시 결제
├── 삼페이 신청
├── 삼페이 설정
├── 해외/부기결제
└── 결제 기본 설정         ← 주문서 버전 선택 (현재: 고급형)
```

### 5-2. 배송비 관련 상세

| 항목 | 현재 설정 |
|------|----------|
| 배송비 청구 고객선택 | **고객선택 불가** (주문시 배송비 포함 청구) |

> - 조건 배송비 기준: 5만원 미만 주문 시 2,500원 배송비 자동 청구

> - 지역별 추가 배송비: 제주/도서산간 추가 요금 설정되어 있음

## 6. 회원 설정

### 6-1. 회원 가입 설정

| 항목 | 현재 설정 |
|------|----------|
| 회원가입 페이지 | **개인회원 가입페이지** 사용 |
| 가입유형 선택화면 노출 | **사용안함** |
| 14세 미만 가입 | **불가** |
| 회원 수신동의 변경 안내 | **변경 안내 미노출** |
| 회원 가입 완료 화면 | **사용안함** |

> 가입 완료 화면 OFF → 상품 상세에서 바로구매 클릭 후 회원가입 시
> 완료페이지 없이 바로 주문서 작성페이지로 이동 (`link_order_join` 가상태그 활용)

### 6-2. 로그인 / 보안 설정

| 항목 | 현재 설정 |
|------|----------|
| 비밀번호 재확인 | **사용함 (권장)** |
| 로그인 정보 유지 (자동로그인) | **사용안함** |
| 로그인 실패 시 계정 잠금 | **사용안함** |

### 6-3. 배송지 / 회원정보 연동

| 항목 | 현재 설정 |
|------|----------|
| 배송지 정보를 회원정보로 교체 | **사용함** |

> 주문 시 배송 주소를 회원 기본 주소로 업데이트 가능.
> 일반형 주문서 사용 시 아래 가상태그 추가 필요 (고급형은 자동):
> ```html
> <!--/IF_UPDATE_ADDRESS/-->
> <label><!--/UPDATE_ADDRESS_CHECK/--> 배송지를 회원정보로 등록</label>
> <!--/IFEND_UPDATE_ADDRESS/-->
> ```

### 6-4. SNS 간편 가입

| SNS | 상태 | 방식 |
|-----|------|------|
| 카카오 | ✅ ON | 카카오싱크 (REST API Key + Javascript Key) |
| 네이버 | ✅ ON | 네이버 로그인 플러스 (Client ID + Client Secret) |
| 페이스북 | ❌ OFF | - |
| 애플 | ❌ OFF | - |

> **개발 연관성:**
> - `if_login` / `link_login` / `link_logout` 가상태그는 메이크샵 기본 로그인 + SNS 로그인 모두 처리
> - SNS 로그인 버튼을 커스텀 HTML로 만들 필요 없음 (메이크샵 기본 가입 페이지에서 처리)

### 6-5. 회원 설정 하위 메뉴 구조

```
회원 설정
├── 회원 설정              ← 가입, 로그인, 보안 기본값
├── 회원 가입 정보 관리     ← 가입 필수/선택 항목 설정
├── 비밀번호 갱신 설정      ← 주기적 변경 강제 여부
├── SNS 간편 가입          ← 카카오✅ 네이버✅ 설정
├── 휴면회원 기능 설정      ← 1년 미로그인 → 휴면 처리
├── 휴면회원 삭제 관리
├── 쿠폰/에치금            ← 쿠폰 사용 정책
├── 포인트                 ← 포인트 적립/사용 정책
├── 추천인
├── 본인인증 서비스 신청
├── 본인인증 서비스 설정
└── 안심번호 서비스 신청
```
