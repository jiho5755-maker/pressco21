## 메이크샵 상세

### 처리

- `3:1에폭시레진` 추가상품/옵션 중 재고가 남아 있던 항목 24개를 1차로 재고 `0` 처리했다.
- 이후 품절성 상태였지만 실제 재고값이 남아 있던 항목 5개를 추가로 재고 `0` 처리했다.
- 총 처리: 29개 stock row
- 검증: 29개 모두 `sto_state_code=SOLDOUT`, `sto_real_stock=0`, `sto_unlimit=N` 확인

### 제외

- `최수연 님 개인결제건(3:1에폭시레진 경화제 2통)`은 개인결제건이므로 자동 품절 처리에서 제외했다.

### 대표 처리 범위

- `1:1에폭시레진/3:1에폭시레진` 본상품의 `3:1 에폭시레진_1000g`
- 레진/몰드/키트류에 붙은 추가상품:
  - `3:1에폭시레진 120g`
  - `3:1에폭시레진 1kg`
  - `3:1에폭시레진 1k`
  - `3:1에폭시레진/120g`
  - `3:1에폭시레진/1kg`

## 스마트스토어 상세

- `에폭시레진 크리스탈 투명 레진 공예 재료 2액형 3:1, 1:1`
  - `3:1 에폭시레진_1000g` 옵션은 재고 `0` 확인
  - 한 상품은 `OUTOFSTOCK`, 다른 상품은 1:1 옵션 재고 때문에 상품 자체는 `SALE`이나 3:1 옵션은 재고 `0`
- `프레스코21 투명 3:1 에폭시 레진 1kg`
  - 상품 상태 `OUTOFSTOCK` / 채널 표시 `SUSPENSION` / 재고 `0` 확인
- `프레스코21 투명 3:1 에폭시 레진 120g`
  - 상품 상태 `SUSPENSION` / 표시 `SUSPENSION`으로 판매 불가 확인
  - 상세조회는 네이버 rate limit으로 일부 제한됨. 판매중 상태가 아니라 write하지 않음
- 조회된 3:1 대상 상품 상세의 `supplementProducts` 안에는 3:1 에폭시레진 추가상품이 없었다.
