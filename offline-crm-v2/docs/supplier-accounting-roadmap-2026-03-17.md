## 이번 마감 기준

이번 단계에서 끝내야 하는 기준:

- 지급 우선순위가 한눈에 보일 것
- 계좌 누락/기한 초과/오늘 지급이 분리될 것
- 공급처 목록과 지급 관리가 같은 상태 체계를 공유할 것
- 비전공자 직원이 설명 없이 상태를 읽을 수 있을 것
