# PRESSCO21 에이전트 팀 사용 가이드

> 7명 직원의 꽃 공예 전문 회사에서, 전략/재무/마케팅/행정/법무/데이터 인력 공백을 **36개 AI 에이전트 팀**으로 채웁니다.

---

## 빠른 시작

### 그냥 말만 하면 됩니다

에이전트를 직접 호출할 필요 없습니다. **평소처럼 질문하면** Claude가 자동으로 적절한 에이전트를 선택합니다.

```
"이 상품 마진이 어때?"          → CFO + sales-margin-strategist 자동 선택
"광고 예산을 어떻게 써야 해?"    → CMO + ad-operations-specialist 자동 선택
"주문 처리가 느린데 어떻게 해?"  → COO + customer-experience-specialist 자동 선택
```

### 슬래시 커맨드로 팀 회의도 가능합니다

| 커맨드 | 사용법 | 결과 |
|--------|-------|------|
| `/team-meeting` | `/team-meeting 쿠팡 로켓배송 진출` | CSO가 진행, 관련 C-Suite가 각 관점에서 분석 → 종합 권고안 |
| `/daily-briefing` | `/daily-briefing` | 오늘 할 일 + 긴급사항 + 우선순위 정리 |
| `/weekly-report` | `/weekly-report` | 주간 성과 + 이번주 과제 + 지표 변화 |
| `/price-strategy` | `/price-strategy 레진 몰드` | 원가 → 마진 → 8채널 최적가 분석 |
| `/marketing-plan` | `/marketing-plan 봄 시즌 캠페인` | 채널별 예산/소재/일정 기획 |
| `/process-design` | `/process-design 주문 출고` | 현행 분석 → 개선안 → 자동화 제안 |

## 상황별 사용 예시

### "이거 어떻게 해야 할지 모르겠어" (전략 고민)

```
사장님: "교육이랑 재료 판매 중에 뭘 더 키워야 해?"
```

→ **CSO**가 자동 선택됩니다. 생태계 관점에서 분석하고, 필요시 CFO(수익성)와 CMO(시장성)도 함께 의견을 줍니다.

### "돈 관련" (가격/마진/세금/정산)

```
사장님: "이 상품 얼마에 팔아야 해?"
사장님: "부가세 신고 준비해야 해"
사장님: "이번 달 수익이 어때?"
```

→ **CFO** 계열이 자동 선택됩니다. 금전 관련은 항상 CFO를 거칩니다.

### "마케팅/홍보" (광고/콘텐츠/SNS)

```
사장님: "네이버 광고 어떻게 해?"
사장님: "유튜브 콘텐츠 전략을 짜줘"
사장님: "인스타에 뭘 올려야 해?"
```

→ **CMO** 계열이 자동 선택됩니다. 예산 관련 결정은 CFO 검증을 거칩니다.

### "업무가 비효율적이야" (프로세스/CS/물류)

```
사장님: "주문 처리 프로세스를 정리해줘"
사장님: "고객 불만이 많아"
사장님: "재고 관리가 엉망이야"
```

→ **COO** 계열이 자동 선택됩니다. 자동화 가능 부분은 CTO와 협업합니다.

### "개발/시스템 관련"

```
사장님: "ERP를 어떻게 대체하지?"
사장님: "자동화를 어디부터 해야 해?"
```

→ **CTO**가 선택되고, 필요시 CFO(요구사항) + COO(프로세스)와 협업합니다.

### 복합 질문 (여러 영역 걸침)

```
사장님: "쿠팡 로켓배송에 진출할까?"
```

→ **CSO**가 주도하고, CFO(수익성) + CMO(시장성) + COO(물류) 관점을 종합합니다.
이런 경우 `/team-meeting 쿠팡 로켓배송 진출`을 사용하면 더 체계적입니다.

## 팀 미팅 활용법

복잡한 의사결정이 필요할 때, `/team-meeting`을 사용하면 여러 에이전트가 각자 관점에서 분석해줍니다.

```
/team-meeting 신상품 라인 확대 검토
```

**결과물 예시:**

```
## 팀 미팅: 신상품 라인 확대 검토

### [CFO 관점] 수익성
- 현재 CRITICAL 759건 해소가 선행되어야...
- 신상품 투자 예상 자금: ...

### [CMO 관점] 시장성
- 레진 공예 카테고리 성장 추세...
- 교육 콘텐츠 연계 가능성 높음...

### [COO 관점] 운영
- 재고 관리 SKU 추가 부담...
- 물류 용량 여유 있음...

### [CSO 종합] 권고안
1. 권고: 레진 키트 3종 우선 출시 (리스크 낮음)
2. 리스크: 환율 경계 단계 → 원가 변동 주시
3. 대안: 기존 상품 리뉴얼 우선
4. 다음 단계: ...
```

## 일과에서 활용하기

### 아침 (출근 후)
```
/daily-briefing
```
→ 오늘 할 일, 긴급사항, 우선순위가 정리됩니다.

### 업무 중 (궁금할 때)
그냥 물어보면 됩니다. 적절한 에이전트가 자동 선택됩니다.

### 금요일 (주간 마무리)
```
/weekly-report
```
→ 이번 주 성과와 다음 주 계획이 정리됩니다.

### 가격 결정이 필요할 때
```
/price-strategy 압화 에코백 키트
```
→ 원가부터 8채널 최적가까지 한번에 분석됩니다.

## 알아두면 좋은 것

### 에이전트가 먼저 말 걸기도 합니다

에이전트들은 자기 영역에서 이슈를 발견하면 능동적으로 제안합니다:
- **CFO**: "이 상품의 마진이 CRITICAL입니다"
- **CMO**: "이 채널의 ROAS가 목표에 미달합니다"
- **COO**: "이 프로세스에 비효율이 있습니다"
- **PM**: "마감 임박 태스크가 3건 있습니다"

### 최종 판단은 항상 대표님이

에이전트는 분석과 권고만 합니다. **결정은 항상 대표님**이 합니다.
특히 다음은 반드시 확인 후 실행합니다:
- 금전 관련 결정 (투자, 가격 변경)
- 대외 커뮤니케이션 (파트너, 고객)
- 원장님(CEO) 확인이 필요한 사항

### 법률/세무는 전문가 확인 필요

compliance-advisor(법무)와 CFO(세무)는 일반적인 가이드를 제공하지만,
**중요한 법적/세무 판단은 반드시 변호사/세무사에게 확인**하세요.
