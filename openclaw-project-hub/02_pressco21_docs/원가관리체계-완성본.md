## 1. 원가 계산 체계 완성본

### 1.1 조달 유형별 COGS 공식 (완성판)

#### A. 국내 직제조 원가

```
COGS = 직접재료비 + 직접노무비 + 제조간접비 + 포장비 + 품질검수비 + 손실충당금

[항목 분해]
직접재료비  = SUM(원재료 단가 x 소요량)
              + SUM(부재료 단가 x 소요량)      ← 접착제, 심지, 색소 등
직접노무비  = 공정시간(분) / 60 x 시간당 인건비
              + 준비/정리시간 배분액              ← 작업 전후 세팅 시간
제조간접비  = (임대료 + 설비감가상각 + 전기료 + 수도료 + 소모품비) / 월 생산량
포장비      = 포장재 단가 + 라벨/스티커 단가 + 완충재 단가
품질검수비  = 검수시간(분) / 60 x 시간당 인건비 x (검수 대상 비율)
손실충당금  = 직접재료비 x 불량률(%)             ← 일반적으로 2~5%
```

**직제조 원가 산출 시 흔히 누락되는 항목**:
- 시제품/테스트 비용 배분 (신상품 개발 시)
- 금형/도구 감가상각 (실리콘 몰드 원형, 압화 프레스 등)
- 작업장 소모품 (장갑, 클리너, 이형제 등)
- 계절성 전력 차이 (여름 냉방, 겨울 난방)

---

#### B. 중국 사입 원가

```
COGS = 매입가(KRW) + 지사관리비 + 국제운송비 + 관세 + 통관수수료 + 국내배송비 + 검품비

[항목 분해]
매입가(KRW)     = 매입가(CNY) x 적용환율(CNY/KRW)
                  또는 매입가(USD) x 적용환율(USD/KRW)
지사관리비      = 매입가(KRW) x 10%                 ← 중국 지사 운영비 배분
국제운송비      = 총 운송비(해상/항공/특송) / 해당 주문 수량
                  + 보험료(선택, CIF 조건 시 포함)
관세            = (매입가 + 국제운송비) x HS Code별 관세율
통관수수료      = 건당 고정비(약 30,000~50,000원) / 해당 주문 수량
국내배송비      = 통관 후 국내 물류센터까지 택배비 / 수량
검품비          = 검품 인건비 / 검품 수량            ← 불량 선별, 재포장

[부가세 처리]
수입 부가세     = (매입가 + 운송비 + 관세) x 10%
                → 사업자 매입세액 공제 가능 → 실질 부담 = 0원
                → COGS에 미포함 (단, 현금흐름 일시 부담 존재)
```

**중국 사입 원가 산출 시 흔히 누락되는 항목**:
- 환율 스프레드: 은행 매매기준율 vs 실제 송금 환율 차이 (약 1~2%)
- 중국 내 내륙운송비: 공장 → 항구/공항까지 운송비
- 샘플 비용: 초기 샘플 발주 비용 (수량 배분)
- 검역/인증 비용: KC 인증, 전기용품 안전확인 등 (해당 시)
- MOQ 초과 재고: 최소주문량 맞추기 위한 초과 수량의 보관 비용

---

#### C. 국내 사입 원가

```
COGS = 도매매입가 + 국내배송비 + 재포장비(선택) + 검품비(선택)

[항목 분해]
도매매입가      = 공급업체 견적가 (VAT 별도)
국내배송비      = 실제 배송비 / 주문 수량
재포장비        = 자체 포장재 + 라벨 교체 비용       ← OEM/자체 브랜딩 시
검품비          = 입고 검수 인건비 / 수량            ← 대량 입고 시
```

**국내 사입 원가 산출 시 흔히 누락되는 항목**:
- 수량 할인 반영: 대량 구매 시 단가 차이
- 반품/교환 비용: 불량 발생 시 반품 배송비
- 카드 결제 수수료: 사업자 카드 결제 시 수수료 부담 (약 1~2%)

## 2. 환율 리스크 관리 프로세스

### 2.1 환율 변동이 원가에 미치는 영향 계산

#### 기준 환율 설정

```
기준환율 (2026-03 기준 예시):
  CNY/KRW: 약 190원 (GOOGLEFINANCE 자동 조회)
  USD/KRW: 약 1,380원

적용환율 결정 규칙:
  1순위: 실제 송금 시점의 은행 매매기준율
  2순위: 발주일 기준 GOOGLEFINANCE 환율
  3순위: 구글시트 환율 시트의 수동입력 환율
```

#### 환율 변동 영향도 계산 공식

```
[단일 상품 환율 영향]
원가 변동액 = 매입가(외화) x (변동후 환율 - 기준 환율)
원가 변동률 = 원가 변동액 / 기존 COGS x 100

예시:
  매입가 = 50 CNY
  기준환율 = 190 KRW/CNY → 매입가(KRW) = 9,500원
  환율 5% 상승 = 199.5 KRW/CNY → 매입가(KRW) = 9,975원
  원가 변동액 = +475원

  기존 COGS = 13,000원 (지사관리비+운송비+관세 포함)
  원가 변동률 = 475 / 13,000 = +3.65%

  ※ 관세도 매입가에 연동되므로 실제 영향은 약 3.65% x (1 + 관세율)
```

```
[전체 포트폴리오 환율 영향]
월간 중국사입 총액 = SUM(각 상품별 월 판매량 x 매입가(CNY))
환율 1% 변동 시 총 영향 = 월간 중국사입 총액(CNY) x 기준환율 x 1%

예시:
  월간 중국사입 총액 = 10,000 CNY
  기준환율 = 190원
  1% 변동 시 = 10,000 x 190 x 0.01 = 19,000원/월
  연간 = 228,000원
```

#### 환율 스프레드 반영

```
실제 원가 = 매입가(외화) x 은행 매매기준율 x (1 + 스프레드 보정률)

스프레드 보정률:
  일반 전신환 송금: +1.0~1.5%
  외환거래 우대 적용 시: +0.3~0.8%
  달러 현찰 보유 사용 시: +0~0.3%

권장: 원가 계산 시 스프레드 보정률 +1.0% 적용 (보수적)
```

### 2.2 환율 경보 기준

#### 3단계 경보 시스템

| 단계 | 환율 변동폭 | 경보 수준 | 대응 행동 |
|------|-----------|----------|----------|
| 관심 | +/-3% (기준 대비) | 주의 관찰 | 환율 모니터링 강화, 기록 |
| 주의 | +/-5% (기준 대비) | 원가 재계산 | 중국사입 상품 COGS 재산출, 마진 영향 분석 |
| 경계 | +/-10% (기준 대비) | 가격 조정 검토 | 판매가 조정/조달전략 변경 의사결정 필요 |

```

예시:
  2026 Q2 기준환율 산정:
  2026-03-01 ~ 2026-03-31 CNY/KRW 평균 = 191.2원
```

```
    |                         |        - 자사몰/네이버 비중 확대

    |                              |

    |                              +-- 기회: 가격 경쟁력 확보 → 프로모션 활용
    |                              +-- 주의: 즉시 가격 인하는 지양 (마진 확보 우선)

```

#### 환율 변동 시 판매가 조정 계산 공식

```
[환율 연동 판매가 인상 계산]

COGS 변동액 = 매입가(CNY) x (신환율 - 구환율)
              + 지사관리비 변동 (매입가 변동의 10%)
              + 관세 변동 ((매입가변동 + 운송비) x 관세율)

필요 판매가 인상액 = COGS 변동액 / (1 - 채널수수료율) x 1.1(VAT)

예시:
  매입가 = 50 CNY
  환율 190 → 200 (5.3% 상승)
  COGS 변동 = 50 x 10 = 500원
  지사관리비 변동 = 500 x 10% = 50원
  관세 변동 (13%) = 500 x 13% = 65원
  총 COGS 변동 = 615원

  네이버(수수료 5.7%) 기준:
  필요 판매가 인상 = 615 / (1-0.057) x 1.1 = 717원 → 약 800원 인상
```

### 3.1 중국 사입: 견적서 → 원가 입력 체크리스트

#### Step 1: 견적서 접수 및 검증 (소요: 1~2일)

```
[ ] 1-1. 공급업체 견적서 수신

    - 이전 견적 대비 가격 변동 확인
    - MOQ 충족 가능 여부 확인
    - 품질 등급/사양 일치 여부 확인

    - 상품명(한글/중문)

    - 카테고리 분류

```

#### Step 2: 환율 및 비용 산출 (소요: 30분)

```
[ ] 2-1. 적용 환율 결정

    - 매입가(KRW) = 매입가(CNY) x 적용환율
    - 환율 스프레드 보정 적용 여부: Y / N (적용 시 +1%)

[ ] 2-3. 지사관리비 산출
    - 지사관리비 = 매입가(KRW) x 10%

[ ] 2-4. 국제운송비 산출
    - 운송 방법: 해상 / 항공 / 특송(EMS,순풍)

    - 개당 운송비 = 총 운송비 / 수량

[ ] 2-5. 관세 산출
    - HS Code: _____________
    - 관세율: _______%
    - 과세 가격 = 매입가(KRW) + 개당 운송비

[ ] 2-6. 통관수수료 산출
    - 건당 통관수수료: _______ KRW (일반 30,000~50,000원)
    - 개당 배분 = 통관수수료 / 수량

[ ] 2-7. 기타 비용 확인
    - 검품비: Y / N (해당 시 금액: _______ KRW)
    - 국내배송비 (통관후): _______ KRW
    - 포장/재포장비: _______ KRW
```

```
    검품비(개당)         _________
    국내배송비(개당)     _________
    포장비(개당)         _________
    ---------------------------
    COGS 합계           _________

[ ] 3-2. COGS 검증
    - COGS가 예상 판매가의 50% 미만인가? → 마진 확보 가능
    - COGS가 예상 판매가의 60% 이상인가? → 가격/조달 재검토 필요
    - 이전 유사 상품 COGS와 비교하여 괴리가 없는가?

    - 자사몰 예상 마진율: _____%

    - 마진율 20% 이상 달성 가능한가? Y / N
```

#### Step 4: 구글시트 입력 (소요: 10분)

### 5.2 원가 절감 목표 설정 방법

#### 목표 설정 프레임워크

```
[Step 1] 현재 원가 베이스라인 수립
  - 전 상품 COGS 총액 (월간)
  - 조달유형별 비중
  - 비용 항목별 비중 (재료비/노무비/간접비/운송비/관세 등)

[Step 2] 절감 기회 영역 식별

  영역 1: 재료비 절감 (기대 효과: 5~15%)
    - 대량 구매 → 단가 협상 (MOQ 활용)
    - 대체 원재료 탐색 (소이왁스 공급처 다변화 등)
    - 국내 직구매 vs 중국 직구매 비교
    - 불량률 감소 → 손실충당금 절감

  영역 2: 운송비 절감 (기대 효과: 10~30%)
    - 해상운송으로 전환 (항공 대비 50~70% 절감, 리드타임 +2주)
    - 합배송 → 건당 통관수수료 배분 최적화
    - 국내 물류 계약 단가 협상

  영역 3: 간접비 절감 (기대 효과: 5~10%)
    - 생산 효율화 → 시간당 생산량 증가
    - 공정 표준화 → 불량률 감소
    - 설비 투자 → 자동화 (장기적)

  영역 4: 관세/통관 최적화 (기대 효과: 3~5%)
    - HS Code 재분류 검토 (저세율 코드 적용 가능 여부)
    - FTA 원산지 증명 활용 가능 여부
    - 소액 면세(150달러 미만) 활용 전략

[Step 3] 분기별 절감 목표 설정
  - 보수적 목표: 전 분기 대비 COGS 2% 절감
  - 도전적 목표: 전 분기 대비 COGS 5% 절감
  - 항목별 세부 목표 분해 (어디서 얼마를 줄일지)
```

## 부록: 원가 관련 용어 정리

| 용어 | 정의 | 비고 |
|------|------|------|
| COGS | Cost of Goods Sold, 매출원가 | 상품 1개 생산/조달에 드는 직접 비용 |
| 공급가 | VAT를 제외한 실질 매출액 | = 판매가 / 1.1 |
| MOQ | Minimum Order Quantity, 최소주문수량 | 중국 사입 시 공급업체 최소 발주량 |
| FOB | Free On Board, 본선인도 조건 | 선적항까지 비용은 판매자, 이후 구매자 |
| CIF | Cost, Insurance & Freight | 목적지까지 운송+보험 포함 |
| HS Code | Harmonized System Code, 관세 분류 번호 | 6자리 국제 공통, 10자리 국가별 세분화 |
| 매입세액 공제 | 사업자가 납부한 부가세를 환급받는 제도 | 수입부가세 실질 부담 = 0 |
| PG | Payment Gateway, 전자결제 대행 | 자사몰 카드결제 수수료 |
| FTA | Free Trade Agreement, 자유무역협정 | 원산지 증명 시 관세 감면 가능 |
