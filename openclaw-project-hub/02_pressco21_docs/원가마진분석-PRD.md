## 1. 시스템 목적

### 1.1 핵심 목적 (v2.0 재정의)

PRESSCO21 구글시트 원가마진분석 시스템은 다음 3가지만 담당한다:

1. **SKU 원가 관리**: 1688 매입가 → 환율 → 부대비용 → 부가세 포함 원가 자동 계산
2. **판매가 전략**: 채널별 수수료 반영한 마진율 자동 계산 + 마진등급 판정
3. **3개 시스템 코드 연결**: 1688 소싱코드 ↔ 사방넷 SKU코드 ↔ 메이크샵 branduid

### 1.2 범위 제외 (사방넷 전담)

아래 항목은 사방넷이 전담하므로 구글시트에서 재발명하지 않는다:
- 매출/주문 데이터 집계
- 재고 현황 및 입출고 관리
- 채널별 정산 처리
- 발주/입고 관리

## 4. SKU마스터 탭 설계 (핵심)

### 4.1 컬럼 구조 (21개)

| 열 | 필드명 | 소스 | 설명 |
|----|--------|------|------|
| A | 1688 한글품명 | 연계마스터 | 중국 소싱 품목명 |
| B | 1688 규격 | 연계마스터 | 규격 상세 |
| C | 매입가(CNY) | 연계마스터 | 최근 단가 |
| D | 최근주문일 | 연계마스터 | 마지막 발주일 |
| E | 주문횟수 | 연계마스터 | 누적 발주 횟수 |
| F | 1688 URL | 연계마스터 | 소싱 링크 |
| G | 사방넷 DB코드 | 연계마스터 | 사방넷 DB코드 |
| H | 사방넷 SKU코드 | 연계마스터 | 사방넷 SKU코드 |
| I | SKU상품명 | 연계마스터 | 사방넷 상품명 |
| J | 메이크샵 branduid | 연계마스터 | 자사몰 상품 ID |
| K | 메이크샵 상품명 | 연계마스터 | 자사몰 상품명 |
| L | 메이크샵 판매가 | 연계마스터 | 자사몰 판매가(원) |
| M | 카테고리 | 연계마스터 | 메이크샵 카테고리 |
| **N** | **KRW 환산 원가** | **수식** | `=IF(C2="","",ROUND(C2*환율!$B$2,0))` |
| **O** | **부대비용포함 원가** | **수식** | `=IF(N2="","",ROUND(N2*1.2,0))` |
| **P** | **부가세포함 원가(COGS)** | **수식** | `=IF(O2="","",ROUND(O2*1.1,0))` |
| **Q** | **마진액** | **수식** | `=IF(OR(L2="",P2=""),"",L2-P2)` |
| **R** | **마진율** | **수식** | `=IF(OR(L2="",P2=""),"",ROUND((L2-P2)/L2,3))` |
| **S** | **마진등급** | **수식** | EXCELLENT/HEALTHY/CAUTION/CRITICAL 자동 판정 |
| **T** | **매핑상태** | 연계마스터 | 완전매핑/부분매핑/미매핑 |
| **U** | **사방넷 상품코드** | 품번코드매핑 LEFT JOIN | branduid 기준 VLOOKUP으로 품번코드 자동 매칭 |

### 4.2 핵심 수식

```
N열 (KRW 환산 원가): =IF(C2="","",ROUND(C2*환율!$B$2,0))
O열 (부대비용포함):  =IF(N2="","",ROUND(N2*1.2,0))     [×1.2 = 지사관리비10%+운송비+기타]
P열 (부가세포함):    =IF(O2="","",ROUND(O2*1.1,0))     [×1.1 = VAT 포함]
Q열 (마진액):        =IF(OR(L2="",P2=""),"",L2-P2)
R열 (마진율):        =IF(OR(L2="",P2=""),"",ROUND((L2-P2)/L2,3))
S열 (마진등급):      =IF(R2="","",IF(R2>=0.6,"EXCELLENT",IF(R2>=0.4,"HEALTHY",IF(R2>=0.25,"CAUTION","CRITICAL"))))
```

### 4.3 마진 등급 기준 (4단계)

| 등급 | 기준 | 의미 |
|------|------|------|
| EXCELLENT | 마진율 ≥ 60% | 광고/프로모션 여력 충분 |
| HEALTHY | 40% ~ 60% | 정상 운영 가능 |
| CAUTION | 25% ~ 40% | 가격 재검토 필요 |
| CRITICAL | < 25% | 즉시 조치 필요 |

> 참고: 판매가는 VAT 포함이므로 마진율 기준이 v1.0보다 높음. 실질 마진율 = (판매가 - COGS) / 판매가

### 4.4 원가 계산 공식 상세

```
[중국 사입 원가 계산 공식]
매입가(CNY) × CNY/KRW 환율     = KRW 환산 원가
KRW 환산 원가 × 1.2            = 부대비용포함 원가
  (지사관리비 10% + 운송비 5% + 관세/기타 5% = 합산 20%)
부대비용포함 원가 × 1.1         = 부가세포함 원가 (COGS)
```

> 주의: ×1.2와 ×1.1은 단순화된 근사값. 정밀 계산 필요 시 사입원가_워크시트 탭 사용.


## 10. 검증 방법

1. **수식 오류 0건**: 전체 탭에서 #NAME?, #DIV/0!, #ERROR! 없음
2. **환율 자동 전파**: 환율 탭 변경 → SKU마스터 N~S열 전체 자동 갱신
