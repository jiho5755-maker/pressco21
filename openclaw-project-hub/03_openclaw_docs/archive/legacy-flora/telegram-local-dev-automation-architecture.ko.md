# 텔레그램 기반 로컬/개발 자동화 아키텍처 v0.1 (초안)

_상태: 초안 / 계획 문서 / 실행 보류 가능_

## 1. 목적
이 문서는 텔레그램을 명령 인터페이스로 사용해 로컬 파일/개발/문서 작업을 보조하는 구조 초안이다.

## 2. 목표 구조
텔레그램 명령 → 플로라 해석 → 서버/맥북 실행 → 결과 회신

## 3. 적합한 작업
- 문서 찾기
- 최신 파일 위치 안내
- 프로젝트 변경 파일 요약
- 회의자료/초안 생성
- 개발 상태/로그 요약

## 4. 부적합한 작업
- 민감한 외부 발송 자동 실행
- GUI 의존 자동화
- 승인 없는 금액/정산 실행

## 5. 장점
- 카카오톡 GUI 자동화보다 안정적
- 로그/권한/승인 흐름 설계 쉬움
- 플로라 종합 OS 방향과 잘 맞음

## 6. 현재 상태
- 아이디어/방향성으로만 보관
- 실행은 회사 핵심 설계 후 검토

## 7. 최종 운영 문장 (초안)
> PRESSCO21는 텔레그램을 명령 인터페이스로 활용해 로컬 문서/개발 보조 자동화를 검토할 수 있다. 다만 현재는 계획 수준으로 보관하고, 핵심 운영 설계가 먼저다.
