# 카카오톡 열린 채팅창 직접 전송 가이드 v1

## 목적
메인 카카오톡 검색창이 아니라, 이미 열려 있는 개별 채팅창을 직접 잡아서 메시지를 보내기 위한 가이드다.

## 전제
- 대상 채팅창이 이미 열려 있어야 함
- 창 제목에 대상명(`jin`, `프레스코21 디자인 기획팀`)이 포함되어 있어야 함
- 카카오톡과 Hammerspoon 권한이 정상이어야 함

## 방식
1. KakaoTalk 앱의 열린 창 목록 수집
2. 제목에 대상 문자열이 포함된 창 찾기
3. 그 창을 포커스
4. 하단 입력 영역 좌표 클릭
5. 메시지 입력 후 Enter 전송

## 장점
- 메인 검색창 포커스 문제 회피
- 이미 열린 분리 채팅창 기준으로 동작 가능
- 개인톡/단톡방 둘 다 같은 패턴으로 확장 가능

## 주의
- 창 제목 문자열이 실제로 포함되는지 확인 필요
- 입력 영역 좌표는 창 크기/배치가 크게 달라지면 조정 필요
