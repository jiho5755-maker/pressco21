## 결론

사용자 판단은 대체로 맞다.

- 원산지증명 비용은 실무상 `250 CNY / 건`으로 보는 것이 맞다.
- 2025년 청구서 표본에서는 KRW 기준 `49,000 ~ 51,250원`, 평균 약 `49,875원` 수준이었다.
- 면장 관세율은 상당수가 `0% / 1.7% / 2.1%` 구간에 몰려 있었다.
- 다만 모든 품목이 그렇지는 않았고, `6.5%`와 `8.0%` 사례도 실제로 존재했다.
- 드라이 식물류는 특별케이스로 봐야 한다.

즉, `원산지증명 = 항상 0~2.1%`로 일반화하면 위험하고, `품목군 또는 HS코드별 기본치 + 상품별 override` 구조가 맞다.

## 점검 요약

### 1. 원산지증명 비용

2025년 청구서 PDF에서 `원산지발급비용` 항목을 확인한 결과:

- CNY 기준: `250`
- KRW 기준 표본:
  - `50,000`
  - `50,000`
  - `51,250`
  - `49,000`
  - `49,250`
  - `49,750`

정리:

- 최소: `49,000원`
- 최대: `51,250원`
- 평균: 약 `49,875원`
- 운영 기본값 추천:
  - 1차 계산: `250 CNY / 건`
  - KRW 간편값: `50,000원 / 건`

### 2. 원산지증명서 유무

면장 PDF에서 `원산지증명서 유무 Y` 표기가 다수 확인됐다.  
즉, 단발 예외가 아니라 `실제로 반복 사용 중인 제도`로 봐도 된다.

### 3. 관세율 분포

2025년 면장 텍스트를 기준으로 반복 확인된 관세율 구간:

- `0.0`
- `1.7`
- `2.1`
- `6.5`
- `8.0`

자주 보인 패턴:

- `0% / 1.7% / 2.1%`가 다수
- 일부 품목은 `6.5% / 8.0%`

의미:

- 원산지증명 적용 품목 다수는 저율 구간에 머문다.
- 그러나 품목군이 다르면 여전히 중간 세율이 남는다.

## 원가 시스템 반영 원칙

### 1. 원산지증명 비용

- 기본값: `250 CNY / 건`
- 개당 반영 방식: `배분수량으로 나눠 배부`

### 2. 관세율

- 1단계: 상품군 기본값 사용
- 2단계: 실제 면장/HS코드가 쌓이면 상품별 override

### 2-1. 현재 특별 예외군

- `DRY LEAF`
  - 2025-08-29 면장 확인
  - HS: `0604.90-9000`
  - 관세율: `3.60%`
- `DRY CUT FLOWER (SP:NARCISSUS)`
  - 2025-08-29 면장 확인
  - HS: `0603.90-0000`
  - 관세율: `25.00%`

해석:

- 사용자가 말한 `꽃류는 특별케이스`는 서류로 확인된다.
- 다만 `잎류는 8%`가 아니라, 이번 2025 확인 표본에서는 `3.6%`였다.
- 따라서 `드라이 리프=8%`로 고정하기보다, 현재는 `3.6% 특별 예외군`으로 두는 쪽이 더 정확하다.

### 3. 앞으로 필요한 데이터

- 상품 또는 SKU별 대표 HS코드
- 원산지증명 적용 여부
- 최근 3회 실제 관세율
- 특이 운송조건 여부
