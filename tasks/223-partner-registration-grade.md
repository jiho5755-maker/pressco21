# Task 223: 파트너 가입 및 인증 등급 시스템

> **상태**: ✅ 완료
> **규모**: S
> **의존성**: Task 201
> **에이전트**: `주도` ecommerce-business-expert, brand-planning-expert | `협업` gas-backend-expert

## 목표

파트너(강사/공방)가 플랫폼에 가입하고, 관리자가 심사/승인하며,
실적에 따라 자동으로 등급이 조정되는 체계를 구축한다.
GAS 엔드포인트는 Task 221에서 함께 구현한다.

## 대상

- Google Forms 파트너 신청서 (설계 + 링크 제공)
- `파트너클래스/class-platform-gas.gs` (Task 221에서 확장)
- `docs/phase2/partner-registration-guide.md` (신규 생성)

## 구현 단계

- [ ] 1단계: 파트너 신청 Google Forms 항목 정의
  - 사업자등록증, 포트폴리오 URL, 강사 소개, 연락처
  - 희망 수업 카테고리, 지역, 시설 정보
- [ ] 2단계: 등급 체계 세부 기준 확정
  - SILVER(기본), GOLD(10건+, 4.0+), PLATINUM(50건+, 4.5+)
  - 각 등급별 수수료율, 적립금 전환율, 혜택
- [ ] 3단계: 파트너맵 배지 연동 설계 (등급별 배지 표시)
- [ ] 4단계: 강의 등록 프로세스 문서화
  - Google Forms -> 관리자 검수 -> 승인 -> 플랫폼 자동 노출
- [ ] 5단계: 관리자 가이드 문서 작성

## 수락 기준

- [ ] 파트너 신청 Google Forms 항목이 설계되어 있다
- [ ] 등급 체계(SILVER/GOLD/PLATINUM) 기준이 명확히 정의되어 있다
- [ ] GAS 파트너 코드 자동 발급 로직이 구현되어 있다 (Task 221 연계)
- [ ] 관리자가 심사/승인할 수 있는 프로세스 문서가 있다

## 테스트 체크리스트

- [ ] 파트너 신청 -> 관리자 검토 알림 이메일 수신 확인
- [ ] 관리자 승인 -> 파트너 코드 발급 -> 신청자 안내 이메일 발송 확인
- [ ] 등급 자동 업데이트 (거래 건수/평점 기준) 정상 동작
- [ ] 파트너맵에 등급 배지 표시 확인
