# Task 232: 메인페이지에 클래스 플랫폼 진입점 추가

> **상태**: ✅ 완료
> **규모**: S
> **의존성**: Task 211, Task 212
> **에이전트**: `주도` makeshop-ui-ux-expert | `협업` brand-planning-expert, seo-performance-expert

## 목표

메인페이지에 "원데이 클래스" 섹션을 추가하여 클래스 플랫폼으로의 유입을 유도한다.
GAS에서 인기 클래스 데이터를 조회하여 3~4개 카드를 미리보기로 표시한다.

## 기술 제약

- **메인페이지 Index.html 수정 금지**: 원본 가상태그 손상 3곳(lines 802, 892, 1075) 존재.
  모든 섹션 삽입은 `메인페이지/js.js`에서 DOM 조작으로 처리.
- IIFE 패턴 필수
- CSS: `.main-class-entry` 스코핑 필수
- 기존 메인페이지 스타일과 조화롭게 (압화/보존화 브랜드 톤)

## 대상 파일

- `메인페이지/js.js` (확장 - 클래스 섹션 삽입 로직 추가)
- `메인페이지/css.css` (확장 - `.main-class-entry` 스타일 추가)
- GAS `파트너클래스/class-platform-gas.gs`: 인기 클래스 조회 엔드포인트 확인 (이미 구현됨)

## 현재 메인페이지 구조

- `메인페이지/js.js` - IIFE 적용됨 (기존 코드 패턴 준수 필수)
- `메인페이지/css.css` - 기존 스타일
- `메인페이지/Index.html` - 수정 금지 (가상태그 손상 방지)
- GAS 엔드포인트: `?action=getClasses&sort=popular&limit=4` 사용 가능
