## 💬 1차 의견 (병렬 수령)

### 박서연 CFO — 조건부
- 쿠팡 로켓 수수료 15~18% vs 자사몰 PG 1~2% → SKU당 순마진 격차 13~16%p
- 776 SKU 중 EXCELLENT/HEALTHY 등급만 유효 등록 가능 → 230~310 SKU (30~40%)
- 선입고 리드타임 45~60일 → 추가 재고 묶임 7천만~1억
- 리스크: 반품률 1.8~2.5배, 자사몰 소매 8.85% 카니발라이제이션
- **초기 조건**: 파일럿 50 SKU, 추가 재고 3천만, 순마진 35%+

### 정유나 CMO — 조건부 ("로켓=미끼, 자사몰=본진")
- 입문자/선물 구매자 신규 C-end 유입구로서 로켓은 "가장 얕은 물가"
- 플랫폼 톤 충돌 불가피 → **브랜딩 아닌 유입 목적으로 설계**
- 쿠팡(15.7%)과 자사몰(34.4%)은 타겟 분리됨 → 상충 적음
- 리스크: 가격 기준점 붕괴, 리뷰 생태계 분산
- **SKU 원칙**: 소모성·범용·저관여만, **시그니처·파트너클래스 키트 자사몰 봉인**
- 인서트 카드 + QR + 첫구매 쿠폰 필수

### 김도현 COO — 조건부 찬성
- 쿠팡 FBR 초기 공수 3~4주, 이재혁 과장 자동화 파일럿과 병행 불가
- 오히려 운영 부담 감소 여지: 송파 포장 병목 20~30% 해소 가능
- 리스크: 재고 이원화 지옥, 꽃 공예 재료 편차 → 평점 붕괴
- **선행 조건**: SKU 20개 이내 시작 + 재고 동기화 WF + 쿠팡 담당자 1명 + 2개월 KPI 리뷰

### 최민석 CTO — 조건부
- 쿠팡 윙 API 연동 신규 구축 3~4주 (HMAC SHA256 서명 래퍼 Code 노드 필수)
- **진짜 로켓 = 재고 원천 이원화** (사방넷 밖, 쿠팡 풀필먼트)
- F23~F25 매출 리포팅 확장은 1~2일로 간단
- 리스크: 메이크샵 API Rate Limit(500/h), 쿠팡 파트너 승인 2~4주
- **질문**: "풀필먼트 모델이 먼저, API는 그 다음"

## 🔄 상호 질의 (1라운드)

### Q1: 박서연 → 한지훈 — 전략 목표 명확화
> "외형 매출 성장 / 신규 고객 획득 / 재고 소진 중 무엇인지?"

**CSO 답변**: **신규 C-end 고객 획득 채널**로 단일화.
- Q2 로켓은 매출 숫자 아닌 "자사몰로 끌어올 신규 접점 수"로 평가
- 핵심 KPI = 로켓 구매자의 자사몰 재구매 전환율(목표 8%+) + 파트너클래스 유입률
- 허용 마진: BEP ±3%
- "쿠팡=미끼, 자사몰=본진" (정유나 프레임) 공식 채택

### Q2: 정유나 → 박서연 — LTV 포함 손익분기
> "로켓 수수료/반품 녹이고도 LTV로 손익분기? 2차 구매율 8~12% 가정."

**CFO 답변**: **보수적 하향 (5%)**.
- 강사공간 고객 LTV ≈ 87만 원 / 로켓 신규 유입 고객 LTV ≈ 4,550원(유입당 기대값)
- 허용 적자: **SKU당 -3,000원** 또는 **채널 총 월 450만**
- **수정 입장**: 조건부 확대 — 50 SKU 중 30개 순마진 -10~+15% 구간 허용, 20개는 35%+ 유지
- 3개월 후 자사몰 재구매율 실측 필수, 5% 미달 시 즉시 축소

### Q3: 최민석 → 김도현 — 풀필먼트 모델
> "진짜 로켓(FBR) vs 판매자로켓(자체 출고)?"

**COO 답변**: **하이브리드 (FBR 15 SKU + 판매자로켓 35 SKU)**.
- 성수기 송파 포장 병목 20~30% 검증 가능 최소 규모
- 묶임 재고 2~3천만으로 통제 (전량 FBR 시 7천만~1억 대비)
- 학습 비용 분산 → Q3 확장 시 실패 비용 감소
- 핵심 한 문장: "전량 FBR은 자살행위, 전량 자체출고는 병목 해소 실패"

## 🎯 한지훈 CSO 종합안

쿠팡은 '매출 채널'이 아니라 **'신규 C-end 고객 획득 퍼널'**로 재정의. 하이브리드 50 SKU · 3개월 파일럿으로 2차 구매율 실측값 확보 후 확대/축소 결정.

| A | 6개월 유예 + 자사몰·앱 먼저 | 브랜드 통제력, 마진 방어 | 경쟁사 선점 허용, 최상단 퍼널 공백 | 하 (기회비용 중) |
| **B ✅** | **하이브리드 50 SKU 파일럿 Q2** | **리스크 분산, 실측 데이터 확보, 락인 테스트** | **CTO 3~4주 부담, 초반 적자 감내** | **중** |
| C | 전량 776 SKU 확대 | 규모 효과, SEO 선점 | 의존도 경보선 돌파, CRITICAL 마진 확대, 봉인 불가 | 상 |

### CSO 권고: **B안**
1. **전략 정합성**: 생태계 선순환 최상단 공백 메우면서 자사몰 락인 장치 유지 — '퍼널' 목표 일치
2. **리스크 통제성**: CFO 허용 적자 월 450만 (연 5,400만 = 매출 11.6억의 4.7%) 내 실측, 5% 미달 시 exit 조건 명확
3. **조직 수용성**: 8명 + 3개월 병행 과제와 충돌 없는 최소 규모
