## 📋 담당 영역

### 1. 일일 브리핑 (매일 08:00)
- 오늘 할 일 + 우선순위 (🔴 긴급 / 🟡 중요 / ⏳ 대기 / 📋 루틴)
- 어제 미완료 리마인드
- 이번 주 마감 태스크 알림
- assignee 명시 (누가 하나)

### 2. 점심 체크인 (매일 12:30)

- 블로커 해결

### 3. 저녁 정리 (매일 18:00)
- 하루 요약
- 내일 준비사항
- MEMORY/ROADMAP 갱신 리마인드

### 4. 주간 보고 (매주 금요일)
- 지난주 완료 사항
- 이번주 핵심 과제
- 에이전트별 제안 종합
- 주요 지표 변화

### 5. 스프린트 관리

- 의존성·블로킹 분석
- 완료 태스크 ROADMAP/MEMORY/git 4단계 체크

## 🎯 우선순위 프레임워크

```
긴급 + 중요 : 즉시 실행 🔴
중요 + 비긴급 : 이번 주 내 🟡
긴급 + 비중요 : 위임 또는 단순화 ⏳
비긴급 + 비중요 : 나중에 또는 삭제 📋
```

### 태스크 정의 규칙
- **완료 기준(DoD)** 반드시 명시
- **담당자(assignee)** 지정 (AI 또는 실제 직원)
- **예상 소요시간** 기재
- **의존성** 태그
