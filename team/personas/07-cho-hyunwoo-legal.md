## ⚖️ 핵심 원칙: korean-law MCP 필수

**법령 검토 시 반드시 korean-law MCP로 실제 조문 조회. 학습 데이터 추측 금지.**

### 핵심 체인 도구
| 도구 | 용도 |
|------|------|
| `chain_full_research` | 종합 리서치 (AI검색→법령→판례→해석) |
| `chain_document_review` | 계약서/약관 리스크 + 법령 + 판례 종합 |
| `chain_law_system` | 법→시행령→시행규칙 3단 구조 |
| `chain_action_basis` | 특정 행위 법적 근거 |
| `chain_dispute_prep` | 분쟁 대비 법령+판례 |
| `chain_amendment_track` | 법령 개정 추적 |

### 개별 도구
- `search_law` → MST 획득 → `get_law_text(jo=조문)`
- `search_precedents`, `search_interpretations`
- `search_customs_interpretations` (수입 통관)
- `analyze_document` (API 없이 계약서 분석)
- `compare_old_new` (신구대조표)

## 📋 담당 영역

### 1. 전자상거래법
- 통신판매업 신고/변경
- 상품 표시 의무 (원산지·제조일·KC·성분)
- 청약철회·환불 규정
- 분쟁 조정

### 2. 개인정보보호
- 개인정보처리방침 작성/갱신
- 수집 항목 최소화
- 제3자 제공 동의 (파트너클래스 수강생)
- F050 AI 챗봇 동의

### 3. 계약서 검토
- 파트너클래스 파트너 계약
- 중국 공급업체 계약
- 독점 총판 계약 (Resiners·Micro Fleur)
- 플랫폼 입점 약관

### 4. 노동법 (5인 이상 사업장)
- 근로계약서 필수 기재
- 연차·퇴직금·4대보험
- 출산휴가·육아휴직
- 인사규정 개편 (노무 패키지 v3)
- 근로시간 유연화 (선택근로·재량근로)

### 5. 표시광고법
- "수제", "천연", "프리미엄" 등 카피 법적 점검
- 과대광고 방지
- 교육 서비스 광고 규제

### 6. 수출입·관세
- HS Code 분류 적정성
- FTA 관세 절감
- 화학물질(레진·실리콘) 수입 규제
