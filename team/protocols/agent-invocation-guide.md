# 🎛️ 에이전트 호출 가이드

> "누구에게 뭘 맡길지 헷갈릴 때 여는 페이지."
> 확정: 2026-04-16

---

## 🚀 일상 호출 패턴

### 혼자 생각 정리 (가장 많이 쓸 패턴)
```
지호님 → 유준호님 (페어코더)
```
- 바이브코딩 즉석 도움
- 아이디어 → MVP 변환
- 에러 해석

### 의사결정 검증
```
지호님 → 한지훈님 (CSO)
```
- "이거 해야 할지 말지"
- 선택지 A/B/C 도출
- 리스크 체크

### 숫자 확인
```
지호님 → 박서연님 (CFO)
```
- "이거 돈 되나?"
- 마진·수익성
- 비용 검증

### 마케팅 기획
```
지호님 → 정유나님 (CMO)
```
- 브랜드 일관성
- 채널 전략
- 콘텐츠 방향

### 기술 판단
```
지호님 → 최민석님 (CTO)
```
- 아키텍처 결정
- 시스템 통합
- 기술 부채

### 실행 계획
```
지호님 → 김도현님 (COO)
```
- 프로세스 설계
- SOP 문서화
- 8명 업무 분담

### 일정 관리
```
지호님 → 윤하늘님 (PM)
```
- 오늘 할 일
- 우선순위
- 마감 임박

### 법적 검토
```
지호님 → 조현우 고문님
```
- 계약 검토
- 법령 확인
- 노무 규정

### 직원 관련
```
지호님 → 강예린 코치님
```
- 업무 매뉴얼
- 교육 프로그램
- 온보딩

## 🏛️ 다관점 필요 시 (회의)

### 주간 전략회의
```
/team-meeting [주제]
```
→ 한지훈 CSO 주관, 4-5명 임원 병렬 의견

### 월간 리뷰
```
/monthly-report
```
→ 박서연 CFO + 한지훈 CSO 공동

### 단일 C-Suite 심층
```
/board-call [임원] [주제]
```
→ 특정 임원 1:1 심층 상담

## 💡 초보자용 팁

### 첫 한 달 (기준)
- 하루에 3명 이하 호출
- 유준호님 가장 많이
- 임원진 중에는 CSO/CFO/CMO/CTO 우선

### 2개월 차
- `/team-meeting` 주 1회 시도
- 2층 실무진 직접 호출 시작
- 피드백 루프 (`/feedback`) 습관화

### 3개월 차
- 월간 리뷰 루틴화
- 에이전트 레벨업 체크
- 신규 에이전트 필요성 검토
