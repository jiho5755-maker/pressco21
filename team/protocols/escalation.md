# 🚨 에스컬레이션 프로토콜

> "막히면 올리고, 위험하면 즉시 보고하고, 의심되면 한 번 더 물어본다."
> 확정: 2026-04-16

---

## 🎯 에스컬레이션 경로

```
[Level 0] 실무 에이전트 (2층)
    │ 해결 가능: 자율 실행
    │ 해결 불가: Level 1
    ▼
[Level 1] 소속 임원 (1층 C-Suite)
    │ 단일 도메인: 자체 결정
    │ 복합(2+): Level 2
    ▼
[Level 2] 한지훈 CSO (종합)
    │ 일상 업무: 자체 종합안
    │ 중대 결정: Level 3
    ▼
[Level 3] 장지호 팀장 (경영기획)
    │ 운영 결정: 자체 승인
    │ 대외·브랜드·중대 투자: Level 4
    ▼
[Level 4] 사장님·원장님 (최종 승인)
    │
    ▼
[Level 5] 경영위원회 공동 결정
    (필요 시 3인 집결: 원장님 + 사장님 + 지호님)
```

---

## 📏 에스컬레이션 기준

### 즉시 지호님으로 (Level 3 직통)
- 회사 전략 방향 전환
- 신규 프로젝트 착수
- 10만원 이상 투자·광고비 집행
- 기술 아키텍처 중대 변경
- 신규 에이전트 생성·폐지
- 외부 계약 체결 전 승인

### 사장님 확인 필요
- 물류·출고 프로세스 중대 변경
- 재고 정리 방침
- 창고 공간 재편
- 배송업체 계약 변경

### 원장님 확인 필요
- 상품 품질 기준 변경
- 교육 콘텐츠 핵심 메시지
- 파트너·협회 관계 변경
- 브랜드 정체성 수정
- 신상품 최종 승인
- 꽃공예 기술·전문성 이슈

### 경영위원회 공동 (3인)
- 공동대표 등록
- 사업 방향 대전환
- 5천만원 이상 투자
- 법인 전환
- 장기차입금 신규

## 👮 필수 경유 원칙

### 박서연 CFO 필수
**모든 금전 관련**:
- 광고비 집행
- 쿠폰·할인 설계
- 원가 책정
- 투자 판단
- 지원사업 수혜 예산

### 조현우 법무 필수
**모든 법적 리스크**:
- 계약서 작성·검토
- 개인정보 수집·처리
- 표시광고 카피 (의심 시)
- 수출입 관세
- 노무·인사 규정 변경

### 한지훈 CSO 주관
**복합 2+ 도메인**:
- 예: 쿠팡 진출 (재무+마케팅+운영)
- 예: 파트너클래스 확장 (브랜드+기술+법무)
- 예: 가업승계 관련 의사결정

## ⏱️ 응답 시간 기준

| 긴급도 | 정의 | 대응 시간 |
|--------|------|--------|
| 🔴 CRITICAL | 서비스 중단, 법적 위반, 보안 침해 | **즉시** (5분 내) |
| 🟠 HIGH | 주요 기능 장애, 매출 직접 영향 | **30분 내** |
| 🟡 MEDIUM | 기능 부분 장애, 일부 불편 | **당일 내** |
| 🟢 LOW | 개선 제안, 기능 요청 | **주간 회의에서** |

## 📞 긴급 연락 체계

### 🔴 CRITICAL 발생 시
```
1. 실무 에이전트 → 텔레그램 즉시 알림 (지호님 + 해당 임원)
2. server-ops가 최초 발견 시 → auto-heal 실행
3. 최민석 CTO 즉시 호출
4. 지호님 → 원장님·사장님 전화
```

### 🟠 HIGH 발생 시
```
1. 해당 임원 호출
2. 한지훈 CSO 브리핑
3. 지호님 보고
4. 대응 계획 30분 내 수립
```

## 🎭 에스컬레이션 예시

### Case 1: 광고비 1,000만 원 집행
```
광고 실무 → 정유나 CMO (안 작성)
정유나 CMO → 박서연 CFO (예산 검증)
박서연 CFO → 한지훈 CSO (전략 정합)
한지훈 CSO → 지호님 (최종 승인)
```

### Case 2: 파트너 계약 체결
```
community-partnership → 정유나 CMO
정유나 CMO → 조현우 법무 (계약 검토)
조현우 법무 → 한지훈 CSO (전략 가치)
한지훈 CSO → 지호님 (승인)
지호님 → 원장님 (파트너 관계 보고)
```

### Case 3: 서버 장애 (긴급)
```
server-ops auto-heal 실행
  ↓ 실패 시
최민석 CTO 긴급 호출 (Level 1 바로)
  ↓ 30분 내 미복구
지호님 + 장준혁 사장님 텔레그램 (Level 3+4)
```

### Case 4: 신상품 출시
```
product-planner → 한지훈 CSO (전략 검토)
한지훈 CSO → 박서연 CFO (원가·마진)
박서연 CFO → 정유나 CMO (마케팅 가능성)
정유나 CMO → detail-page-planner (상세페이지)
종합 → 지호님 → 원장님 (품질 최종 확인)
```

## 🤖 AI 자율 결정 허용 범위

AI 임원진이 **지호님 승인 없이** 자율 실행 가능한 범위:

### 한지훈 CSO
- 주간회의 진행
- 내부 분석 보고서 작성
- 경쟁사 모니터링

### 박서연 CFO
- 일상 회계 처리
- 월간 정산 내역 검증
- 마진 모니터링 리포트

### 정유나 CMO
- 기존 광고 A/B 테스트 (예산 100만원 이내)
- SNS 콘텐츠 발행 (브랜드 톤 준수 시)
- 기획전 기획안 작성

### 김도현 COO
- CS 매뉴얼 업데이트
- 재고 알림 리포트
- 프로세스 개선 제안

### 최민석 CTO
- 버그 수정
- 기존 WF 마이너 개선
- 정기 보안 감사

### 윤하늘 PM
- 일일 브리핑 발송
- 마감 알림
- ROADMAP 업데이트

### 조현우 법무
- 법령 개정 모니터링
- 일상 카피 법적 검토

### 강예린 HR
- 매뉴얼 업데이트
- 온보딩 문서 작성
- 교육 프로그램 초안

### 유준호 페어코더
- 지호님 요청 즉석 코딩
- 스니펫 라이브러리 업데이트
- 에러 해석·수정 제안
