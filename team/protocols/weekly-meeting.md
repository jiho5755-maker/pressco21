#### ⚠️ SendMessage 재스폰 필수 단계 (2026-04-17 추가)

현재 Claude Code `SendMessage`는 **inbox 전달만** 함. 받은 임원은 자동 응답하지 않으므로,
응답을 받으려면 **같은 `name`으로 Agent 도구를 재호출**해야 한다.

```
# 1) SendMessage로 질문 inbox에 전달
SendMessage(to: "cfo-seoyeon", summary: "LTV 질문", message: "...")

# 2) 바로 같은 name으로 Agent 재스폰 (컨텍스트 유지됨)
Agent(
  name: "cfo-seoyeon",
  subagent_type: "chief-financial-officer",
  prompt: "박서연 CFO님, inbox에 정유나 CMO가 던진 LTV 질문이 들어왔습니다. 답변 포맷..."
)
```

**라운드 관리**:
- **최소 2라운드 (권장 2~3)** — 1라운드 이견·질문이 드러났으면 반드시 2라운드 추진
- 1라운드 핵심: CFO↔CMO (숫자·브랜드) + COO↔CTO (실행·기술)
- 2라운드 필수 페어: **CMO↔COO** (브랜드 장치가 운영 가능한지) — 1회차 테스트에서 누락됨
- 임원 5명 이상 소집 시 매트릭스 질의 권장 (A→B, B→C, C→A 순환)
