# 저장본 폴더

이 폴더는 `레지너스 화이트페이퍼` 페이지의 메이크샵 저장본을 날짜별로 보관하는 용도다.

현재 저장소에는 본문 `index.html`, `style.css`, `script.js`가 없으므로, 파일이 다시 정리되면 아래 이름으로 맞춰 관리한다.

- `레지너스 화이트페이퍼/index.html`
- `레지너스 화이트페이퍼/style.css`
- `레지너스 화이트페이퍼/script.js`

보관 규칙:

- 폴더명 형식: `YYYY-MM-DD-작업명`
- 항상 `HTML/CSS/JS` 3개를 같이 저장
- 공통 절차: `docs/메이크샵-페이지-협업-표준.md`
