# 메인페이지 저장본 폴더

이 폴더는 메이크샵 편집기에서 저장된 메인페이지 코드를 날짜별로 보관하는 용도다.

## 기본 원칙

- `메인페이지/Index.html`
- `메인페이지/css.css`
- `메인페이지/js.js`

위 3개는 항상 **현재 live와 맞춘 최신 운영본**이다.

`저장본/`은 별도 보관용이다.

## 폴더명 규칙

형식:

```text
YYYY-MM-DD-작업명
```

예시:

```text
2026-03-15-디자이너수정-1차
2026-03-15-파트너클래스-연동전
2026-03-16-배포완료본
```

## 각 저장본 폴더 안에 넣을 파일

항상 아래 3개를 같이 넣는다.

```text
Index.html
css.css
js.js
```

한 파일만 따로 보관하지 않는다.

## 추천 사용 흐름

### 디자이너가 메이크샵에서 수정한 직후

1. 메이크샵 `HTML/CSS/JS` 탭 내용을 현재 파일에 덮어쓴다.
2. 같은 내용을 `저장본/YYYY-MM-DD-작업명/`에도 저장한다.

### Codex 수정 후 메이크샵에 다시 반영한 직후

1. 메이크샵 저장이 끝난 최종 탭 내용을 현재 파일에 다시 덮어쓴다.
2. 필요하면 `저장본/YYYY-MM-DD-작업명/`으로 한 번 더 보관한다.

## 금지

- 최신 운영본을 `기존 코드/`에 저장하지 않음
- HTML만 저장하고 CSS/JS를 빼먹지 않음
- 메이크샵에서 저장하지 않은 임시 코드를 최신본이라고 간주하지 않음
