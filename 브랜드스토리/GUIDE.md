# 브랜드스토리 가이드

> PRESSCO21 브랜드 헤리티지/철학 소개 페이지
> 편집 위치: 관리자 > 개별 페이지

## 파일 구조

```
브랜드스토리/
└── 브랜드페이지/
    ├── index.html              메인 HTML (873줄, 48KB)
    ├── css/
    │   ├── common.css          공통 스타일 (373줄, 8.3KB)
    │   └── heritage.css        헤리티지 전용 스타일 (3,172줄, 62.5KB)
    ├── js/
    │   ├── common.js           공통 JS (140줄, 4.5KB)
    │   └── heritage.js         헤리티지 전용 JS (722줄, 27.6KB)
    ├── images/
    │   ├── hero/               히어로 이미지
    │   ├── gallery/            갤러리 이미지 (10장)
    │   ├── books/              서적 이미지 (4장)
    │   └── sections/           섹션별 이미지 (2장)
    ├── docs/
    └── README.md
```

## 페이지 구성

### 주요 섹션
- 히어로 섹션 (브랜드 비주얼, 대형 이미지)
- 갤러리 섹션 (작품 이미지 그리드)
- 헤리티지 타임라인 (브랜드 역사)
- 브랜드 철학/비전

### 기술 특징
- 반응형 디자인
- 이미지 최적화 (로컬 images/ 폴더 사용)
- 스크롤 애니메이션
- CSS/JS 파일 분리 (common + heritage)

## 배포 방식

이 페이지는 **독립적인 개별 페이지**로 운영됩니다:
- 메이크샵 개별 페이지에 `index.html` 전체 코드를 붙여넣기
- CSS/JS/이미지는 메이크샵 FTP나 외부 CDN에 업로드 후 참조
- 또는 인라인으로 HTML에 포함

## 이미지 파일 목록

```
images/hero/     - 히어로 배경 이미지
images/gallery/  - 10장의 작품 갤러리 이미지 (PNG)
images/books/    - 4장의 서적/출판물 이미지
images/sections/ - 2장의 섹션 배경 이미지
```

## 주의사항
- 25개 파일로 구성된 가장 파일 수가 많은 프로젝트
- 이미지 경로가 로컬 상대 경로로 되어 있으므로 배포 시 CDN 경로로 변환 필요
- heritage.css가 62.5KB로 큰 편 - 섹션별 애니메이션/레이아웃 스타일 포함
