## 개요

## 구현 완료 기능

### 1. Scroll-Driven Animations

스크롤에 따라 자연스럽게 동작하는 애니메이션을 구현했습니다.

| 기능 | 설명 |
|------|------|
| 타임라인 순차 등장 | Intersection Observer로 스크롤 시 연도별 하나씩 reveal |
| 타임라인 진행선 | 스크롤에 따라 세로선이 점진적으로 채워지는 효과 |
| 노드 pop 효과 | 각 연도 노드가 scale 애니메이션으로 등장 |
| Stagger 딜레이 | 각 아이템마다 0.05초씩 지연되어 자연스러운 흐름 |

**기술 구현:**
```javascript
const timelineObserver = new IntersectionObserver((entries) => {
    // threshold: 0.3, rootMargin: '0px 0px -80px 0px'
    // 화면 하단 80px 전에 트리거
}, options);
```

### 2. 갤러리 필터링

카테고리별로 갤러리 이미지를 필터링하는 기능을 추가했습니다.

| 카테고리 | data-category |
|----------|---------------|
| 전체 | `all` |
| 압화 작품 | `pressflower` |
| 투명 표본 | `specimen` |
| 전시 현장 | `exhibition` |
| 교육 현장 | `education` |

**UI 특징:**
- 필터 버튼 가로 스크롤 (모바일)
- fade + scale 전환 애니메이션
- 활성 필터 시각적 강조

---

### 3. Timeline 인터랙티브 모달

타임라인 아이템 클릭 시 상세 정보를 모달로 표시합니다.

**모달 구성요소:**
- 연도 배지
- 제목
- 상세 설명 (확장된 내용)
- 관련 이미지

**접근성 기능:**
- ESC 키로 닫기
- 외부 클릭으로 닫기
- 포커스 트랩 (Tab 키 제한)
- ARIA 속성 적용

---

### 4. 모바일 드로워 네비게이션

기존 하단 탭바를 대체하는 슬라이드 드로워 메뉴입니다.

**구성요소:**

```
┌─────────────────────┐
│ ☰ 햄버거 버튼        │  ← 좌상단 고정
├─────────────────────┤
│ 프레스코21     ✕    │  ← 드로워 헤더
├─────────────────────┤
│ • 시작              │
│ • 철학              │
│ • 여정              │
│ • 발자취            │
│ • 혁신              │  ← 전체 섹션 링크
│ • 교육              │
│ • 국제 교류         │
│ • 저서              │
│ • 갤러리            │
│ • 레거시            │
├─────────────────────┤
│ [쇼핑몰 바로가기]    │  ← CTA 버튼
└─────────────────────┘
```

**애니메이션:**
- 왼쪽에서 슬라이드 인 (transform: translateX)
- 햄버거 → X 아이콘 변환
- 배경 오버레이 딤 처리
- 스크롤 잠금

### 5. Hero Ken Burns 효과

비디오 대신 이미지에 영화적 효과를 적용했습니다.

```css
@keyframes kenBurns {
    0% {
        background-size: 100% auto;
        background-position: center center;
    }
    100% {
        background-size: 115% auto;
        background-position: center top;
    }
}
/* 20초 주기, 무한 반복, alternate */
```

**최적화:** 모바일에서는 성능을 위해 비활성화

│              │  │ └───┴───┘ │  │
